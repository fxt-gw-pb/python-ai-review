1.1 Transformer зҡ„жҸҗеҮәиғҢжҷҜ

еңЁ Transformer еҮәзҺ°д№ӢеүҚпјҢеәҸеҲ—е»әжЁЎдё»иҰҒдҫқиө– RNN зі»еҲ—жЁЎеһӢпјҲLSTM / GRUпјүгҖӮ иҝҷзұ»жЁЎеһӢеӯҳеңЁдёӨдёӘж №жң¬й—®йўҳпјҡ

  • й•ҝи·қзҰ»дҫқиө–е»әжЁЎеӣ°йҡҫпјҢеҸҘеӯҗи¶Ҡй•ҝж•Ҳжһңи¶Ҡе·®

дёәи§ЈеҶіиҝҷдәӣй—®йўҳпјҢGoogle еңЁ 2017 е№ҙжҸҗеҮәдәҶ Transformer жһ¶жһ„пјҡ

Attention Is All You NeedпјҲVaswani et al., 2017пјү и®әж–Үең°еқҖпјҡhttps://arxiv.org/pdf/1706.03762.pdf

Transformer дёҚеҶҚдҪҝз”ЁеҫӘзҺҜз»“жһ„пјҢиҖҢжҳҜе®Ңе…ЁеҹәдәҺ Attention иҝӣиЎҢеәҸеҲ—е»әжЁЎгҖӮ

1.2 Transformer зҡ„ж ёеҝғжҖқжғі

з”ЁиҮӘжіЁж„ҸеҠӣпјҲSelf-Attentionпјүжӣҝд»ЈеҫӘзҺҜз»“жһ„пјҢзӣҙжҺҘе»әжЁЎеәҸеҲ—дёӯд»»ж„ҸдҪҚзҪ®д№Ӣй—ҙзҡ„е…ізі»

е…¶е…ій”®и®ҫи®ЎеҢ…жӢ¬пјҡ

  • Self-Attentionпјҡд»»ж„ҸдҪҚзҪ®еҸҜзӣҙжҺҘе»әз«Ӣдҫқиө–
  • е№¶иЎҢи®Ўз®—пјҡж•ҙдёӘеәҸеҲ—еҗҢж—¶еӨ„зҗҶпјҢи®ӯз»ғж—¶е®һзҺ°еәҸеҲ—ж–№еҗ‘зҡ„е№¶иЎҢ
  • дҪҚзҪ®зј–з ҒпјҡжҳҫејҸеј•е…ҘйЎәеәҸдҝЎжҒҜ

1.3 Transformer зҡ„дё»иҰҒдјҳеҠҝ

  2. дҫқиө–и·Ҝеҫ„жӣҙзҹӯпјҢй•ҝи·қзҰ»дҝЎжҒҜе»әжЁЎиғҪеҠӣжӣҙејә
    иҮӘжіЁж„ҸеҠӣи®©д»»ж„ҸдёӨдёӘдҪҚзҪ®зӣҙжҺҘдәӨдә’пјҢдҝЎжҒҜдј йҖ’дёҚеҶҚеҸ—вҖңжӯҘж•°вҖқйҷҗеҲ¶пјҢй•ҝеҸҘжҲ–и·Ёж®өдҫқиө–иғҪиў«жӣҙзЁіе®ҡең°жҚ•иҺ·гҖӮ

  3. з»“жһ„з»ҹдёҖпјҢжҳ“дәҺе ҶеҸ е’Ңжү©еұ•пјҢйҖӮеҗҲеӨ§жЁЎеһӢи®ӯз»ғ
    д»ҘеӨҡеӨҙжіЁж„ҸеҠӣ + еүҚйҰҲзҪ‘з»ңдёәеҹәжң¬жЁЎеқ—пјҢе ҶеҸ е’Ңж”ҫеӨ§еҸӮж•°и§„жЁЎйқһеёёзӣҙжҺҘпјҢйҖӮз”ЁдәҺеӨ§и§„жЁЎйў„и®ӯз»ғдёҺиҝҒз§»еӯҰд№ гҖӮ

дёӢеӣҫжҳҫзӨәпјҡж—  Attention зҡ„ RNN еңЁй•ҝеҸҘдёҠжҖ§иғҪеҝ«йҖҹдёӢйҷҚпјҢиҖҢеҹәдәҺ Attention зҡ„жЁЎеһӢдҝқжҢҒзЁіе®ҡиЎЁзҺ°гҖӮ

  • жҜҸеұӮеҢ…еҗ« 2 дёӘеӯҗеұӮиҝһжҺҘз»“жһ„пјҡ

  • жҜҸеұӮеҢ…еҗ« 3 дёӘеӯҗеұӮиҝһжҺҘз»“жһ„пјҡ
    1. еӨҡеӨҙиҮӘжіЁж„ҸеҠӣпјҲеёҰжҺ©з Ғпјү+ AddNorm(ж®Ӣе·®иҝһжҺҘ + LayerNorm)

иҫ“еҮәйғЁеҲҶ

  • зәҝжҖ§еұӮпјҡе°Ҷи§Јз ҒеҷЁиҫ“еҮәжҳ е°„еҲ°иҜҚиЎЁз»ҙеәҰ
  • Softmax еұӮпјҡз”ҹжҲҗжҰӮзҺҮеҲҶеёғ

иҫ“еҮә зәҝжҖ§еұӮ + Softmax

class MyEmbedding(nn.Module):
    """
    иҫ“е…Ҙ:
        x: иҜҚзҙўеј•, (batch_size, seq_len)
    иҫ“еҮә:
        x: иҜҚеҗ‘йҮҸ, (batch_size, seq_len, d_model)

        return x*self.d_model**0.5

class MyPositionalEncoding(nn.Module):
    """
    иҫ“е…Ҙ:
        x:иҜҚеҗ‘йҮҸ,(batch_size,seq_len,d_model)
    иҫ“еҮә:
        x:ж·»еҠ дҪҚзҪ®зј–з ҒеҗҺзҡ„иҜҚеҗ‘йҮҸ,(batch_size,seq_len,d_model)
    """
    # 1.е®ҡд№ү__init__ж–№жі•
    def __init__(self, d_model=512, dropout=0.1, max_len=1000):
        super().__init__()
        # 1.еҲқе§ӢеҢ–еҸӮж•°
        self.d_model = d_model
        if dropout is not None:
            self.dropout = nn.Dropout(dropout)
        else:
            # е®ҡд№үдёҖдёӘжҒ’зӯүеұӮ,е®һзҺ°д»Җд№ҲйғҪдёҚеҒҡ, y=x
            self.dropout = nn.Identity()
        self.max_len = max_len
        # 2.е®ҡд№үдҪҚзҪ®зј–з Ғ,(max_len, d_model)
        pe = torch.zeros(max_len, d_model)
        # 3.е®ҡд№үдҪҚзҪ®зј–з Ғзҡ„зҙўеј•еәҸеҲ—pos,0~max_len-1
        pos = torch.arange(0, max_len).unsqueeze(1) # (max_len,1)
        # 4.е®ҡд№үеҸӮж•°пјҢи®°еҪ•10000^(-2i/d_model)
        # a^n = exp(log(a))^n = exp(log(a)*n)
        # 10000^(-2i/d_model) = exp(log(10000)*(-2i/d_model))
        div_term = torch.exp(np.log(10000.0)*(-torch.arange(0, d_model, 2) / d_model))
        # 5.и®Ўз®—дҪҚзҪ®зј–з Ғpe, (max_len, d_model)
        pe[:, ::2] = torch.sin(pos * div_term)
        pe[:, 1::2] = torch.cos(pos * div_term)
        # 6.жҠҠ2DиҪ¬дёә3D, (1,max_len,d_model),еҗҺйқўеҒҡеј йҮҸеҠ жі•пјҢж¶үеҸҠеҲ°е№ҝж’ӯжңәеҲ¶
        pe = pe.unsqueeze(0)
        # 7.е°ҶдҪҚзҪ®зј–з Ғж·»еҠ еҲ°зј“еҶІеҢәbuffer: дёҚдјҡи®ӯз»ғпјҢдҪҶеҝ…йЎ»и·ҹзқҖжЁЎеһӢдёҖиө·дҝқеӯҳе’Ңз§»еҠЁзҡ„еҸҳйҮҸ
        self.register_buffer("pe", pe)  # е°ҶpeеҲӣе»әдёәдёҖдёӘself.peзҡ„еҸҳйҮҸ
    # 2.forwardж–№жі•
    def forward(self, x):
        # иҫ“е…Ҙx: иҜҚеҗ‘йҮҸ,(batch_size,seq_len,d_model)
        # 1.иҜҚеҗ‘йҮҸ(batch_size,seq_len,d_model) + дҪҚзҪ®зј–з Ғ(batch_size,max_len,d_model)
        x = x + self.pe[:, :x.size(1)]
        return self.dropout(x)

4.1 Mask жҳҜд»Җд№ҲпјҹпјҲдёҖеҸҘиҜқпјү

Mask з”ЁжқҘеңЁжіЁж„ҸеҠӣи®Ўз®—дёӯвҖңйҒ®жҺүвҖқдёҚиҜҘиў«зңӢзҡ„дҪҚзҪ®дҝЎжҒҜвҖ”вҖ”е®ғеҶіе®ҡжҹҗдёӘ Query жҳҜеҗҰеҸҜд»Ҙе…іжіЁжҹҗдёӘ KeyпјҲдҪңз”ЁеңЁжіЁж„ҸеҠӣеҲҶж•°дёҠпјҢиҖҢдёҚжҳҜзӣҙжҺҘеҺ»жҺүиҫ“е…Ҙ tokenпјүгҖӮ

4.2 жң¬иҙЁдёҺе®һзҺ°иҰҒзӮ№

  • жң¬иҙЁпјҡеұҸи”ҪжіЁж„ҸеҠӣеҲҶж•°пјҲQK^TпјүпјҢйҖҡеёёз”Ё 0/1 жҲ–еёғе°”зҹ©йҳөиЎЁзӨәгҖӮ
  • еҰӮдҪ•е®һзҺ°пјҡеңЁ softmax еүҚжҠҠиў«еұҸи”Ҫзҡ„дҪҚзҪ®еЎ«е…ҘеӨ§иҙҹеҖјпјҲеҰӮ -1e9 жҲ– -infпјүпјҢдҪҝе…¶жҰӮзҺҮжҺҘиҝ‘ 0гҖӮ $$ Attention = \operatorname{softmax}!\left(\frac{Q\cdot K^T}{\sqrt{d_k}} + \text{mask}\right)\cdot V $$

4.3 дёӨзұ»еёёз”Ё Mask

Mask зұ»еһӢ Padding MaskеЎ«е……жҺ©з Ғ Causal Maskеӣ жһңжҺ©з Ғ
еҪўзҠ¶ (batch_size, seq_len) (tgt_seq_len, tgt_seq_len)
зӣ®зҡ„ еҝҪз•ҘеәҸеҲ—дёӯзҡ„ <PAD> еЎ«е……дҪҚзҪ® зҰҒжӯўзңӢеҲ°жңӘжқҘ tokenпјҢдҝқиҜҒиҮӘеӣһеҪ’з”ҹжҲҗйЎәеәҸ
дҪ•ж—¶з”Ё жүҖжңүжіЁж„ҸеҠӣпјҲEncoder/Decoder йҖҡз”Ёпјү Decoder зҡ„ Self-AttentionпјҲиҮӘеӣһеҪ’и§Јз Ғпјү

5 зј–з ҒеҷЁйғЁеҲҶе®һзҺ°

еӯҰд№ зӣ®ж Ү

  • дәҶи§Јзј–з ҒеҷЁдёӯеҗ„дёӘз»„жҲҗйғЁеҲҶзҡ„дҪңз”Ё.
  • жҺҢжҸЎзј–з ҒеҷЁдёӯеҗ„дёӘз»„жҲҗйғЁеҲҶзҡ„е®һзҺ°иҝҮзЁӢ.

5.1 зј–з ҒеҷЁд»Ӣз»Қ

зј–з ҒеҷЁйғЁеҲҶ: * з”ұNдёӘзј–з ҒеҷЁеұӮе ҶеҸ иҖҢжҲҗ * жҜҸдёӘзј–з ҒеҷЁеұӮз”ұдёӨдёӘеӯҗеұӮиҝһжҺҘз»“жһ„з»„жҲҗ * з¬¬дёҖдёӘеӯҗеұӮиҝһжҺҘз»“жһ„еҢ…жӢ¬дёҖдёӘеӨҡеӨҙиҮӘжіЁж„ҸеҠӣеұӮе’ҢеұӮеҪ’дёҖеҢ–д»ҘеҸҠдёҖдёӘж®Ӣе·®иҝһжҺҘ * з¬¬дәҢдёӘеӯҗеұӮиҝһжҺҘз»“жһ„еҢ…жӢ¬дёҖдёӘеүҚйҰҲзҪ‘з»ңеұӮе’ҢеұӮеҪ’дёҖеҢ–д»ҘеҸҠдёҖдёӘж®Ӣе·®иҝһжҺҘ

5.2 жіЁж„ҸеҠӣжңәеҲ¶

иҝҷйҮҢдҪҝз”Ёзҡ„жіЁж„ҸеҠӣзҡ„и®Ўз®—и§„еҲҷ: $$ Attention(Q,K,V)=Softmax(\frac{Q\cdot K^T}{\sqrt{d_{k}}})\cdot V $$

  • Q: QueryпјҢжҹҘиҜўпјҢеҪ“еүҚвҖңжҲ‘жғіе…іжіЁд»Җд№ҲвҖқзҡ„иЎЁзӨәпјҲеҪ“еүҚдҪҚзҪ® / еҪ“еүҚиҜҚпјүгҖӮ
  • K: KeyпјҢй”®пјҢжүҖжңүеҖҷйҖүдҝЎжҒҜзҡ„вҖңзҙўеј•вҖқжҲ–вҖңж ҮзӯҫвҖқгҖӮ
  • V: ValueпјҢеҖјпјҢиҰҒиў«еҸ–иө°гҖҒеҠ жқғиһҚеҗҲзҡ„дҝЎжҒҜеҶ…е®№гҖӮ

"""
жј”зӨә
    transformer зҡ„ Encoder, е®һзҺ°Encoderзј–з ҒеҷЁеҗ„дёӘйғЁеҲҶзҡ„еҠҹиғҪ
з»„жҲҗпјҡ
    NеұӮе ҶеҸ 
        Multi-Head Attention еӨҡеӨҙиҮӘжіЁж„ҸеҠӣжңәеҲ¶
            Add & Norm ж®Ӣе·®иҝһжҺҘ & еұӮеҪ’дёҖеҢ–
        Feed Forward еүҚйҰҲзҪ‘з»ң
            Add & Norm ж®Ӣе·®иҝһжҺҘ & еұӮеҪ’дёҖеҢ–
"""

# еҜјеҢ…
import torch
import torch.nn as nn
import torch.nn.functional as F

# from transformer_input import MyEmbedding, MyPositionalEncoding
import matplotlib.pyplot as plt

# и®ҫзҪ®дёӯж–Үеӯ—дҪ“
plt.rcParams['font.sans-serif'] = ['SimHei']  # еҫ®иҪҜйӣ…й»‘
plt.rcParams['axes.unicode_minus'] = False    # и§ЈеҶіиҙҹеҸ·жҳҫзӨәй—®йўҳ

# 1.е®ҡд№үAttentionзұ»пјҢе®һзҺ°зј©ж”ҫзӮ№з§ҜжіЁж„ҸеҠӣжңәеҲ¶
class MyAttention(nn.Module):
    """
    Transformerзҡ„ зј©ж”ҫзӮ№з§ҜжіЁж„ҸеҠӣжңәеҲ¶
    иҫ“е…Ҙ:
        query/key/value: (batch_size, seq_len, d_model)
        padding_mask(еҸҜйҖү): (batch_size, seq_len),еЎ«е……жҺ©з ҒпјҢTrue=иў«йҒ®и”Ҫ/жҺ©з ҒпјҢFalse=жІЎжңүиў«йҒ®и”Ҫ
        tgt_mask(еҸҜйҖү,Decoderдё“з”Ё): (tgt_seq_len, tgt_seq_len),зӣ®ж ҮеәҸеҲ—зҡ„еЎ«е……жҺ©з ҒпјҢйҳІжӯўзңӢеҲ°жңӘжқҘ/йҒ®и”ҪжңӘжқҘtokenпјҢTrue=иў«йҒ®и”Ҫ/жҺ©з ҒпјҢFalse=жІЎжңүиў«йҒ®и”Ҫ
    иҝ”еӣһ:
        output: (batch_size, seq_len, d_model)
        attention_weights: (batch_size, seq_len, seq_len)
    """
    # 1.е®ҡд№ү__init__ж–№жі•
    def __init__(self, dropout=0.1):
        super().__init__()
        # 1.еҲқе§ӢеҢ–dropoutеұӮ
        if dropout is not None:
            self.dropout = nn.Dropout(dropout)
        else:
            # е®ҡд№үдёҖдёӘжҒ’зӯүеұӮ,е®һзҺ°д»Җд№ҲйғҪдёҚеҒҡ, y=x
            self.dropout = nn.Identity()
    # 2.forwardж–№жі•
    def forward(self, query, key, value, padding_mask=None, tgt_mask=None):
        # query/key/value: (batch_size, seq_len, d_model),(batch_size,num_heads,seq_len,d_head)
        # 1.иҺ·еҸ–еҸӮж•°batch_size, seq_len, d_model
        batch_size = query.shape[0]
        d_model = query.shape[-1]
        q_len = query.shape[-2]
        k_len = key.shape[-2]
        # 2.и®Ўз®—зј©ж”ҫзӮ№з§ҜжіЁж„ҸеҠӣпјҢжү“еҲҶ-еҪ’дёҖеҢ–-еҠ жқғжұӮе’Ң
        scores = query @ key.transpose(-2, -1) / d_model**0.5
        # 3.еӨ„зҗҶжҺ©з ҒпјҢеЎ«е……жҺ©з ҒпјҢеӣ жһңжҺ©з Ғ
        # 3.1 еӨ„зҗҶеЎ«е……жҺ©з Ғ, padding_mask(еҸҜйҖү): (batch_size, k_len)
        if padding_mask is not None:
            # 1.з»ҙеәҰж ЎйӘҢ
            if padding_mask.dim() == 2:
                # 2.ж ЎйӘҢеҪўзҠ¶(batch_size, k_len)
                assert padding_mask.shape == (batch_size, k_len), f"еЎ«е……жҺ©з Ғзҡ„еҪўзҠ¶{padding_mask.shape}й”ҷиҜҜ, йңҖдёә{(batch_size, k_len)}"
                # 3.жү©еұ•еЎ«е……жҺ©з Ғдёә4D,(batch_size, k_len)-> (batch_size, 1, 1, k_len)
                padding_mask = padding_mask.unsqueeze(1).unsqueeze(2)
            elif padding_mask.dim() == 3:
                # 4.жү©еұ•еЎ«е……жҺ©з Ғдёә4D,(batch_size,*, k_len)-> (batch_size, 1, *, k_len)
                padding_mask = padding_mask.unsqueeze(1)
            scores = torch.masked_fill(scores, padding_mask, -1e9)
        # 3.2 еӨ„зҗҶеӣ жһңжҺ©з ҒпјҢtgt_mask(еҸҜйҖү,Decoderдё“з”Ё): (q_len, k_len)
        if tgt_mask is not None:
            assert tgt_mask.shape == (q_len, k_len), f"еӣ жһңжҺ©з Ғзҡ„еҪўзҠ¶{tgt_mask.shape}й”ҷиҜҜ, йңҖдёә{(q_len, k_len)}"
            # жү©еұ•еӣ жһңжҺ©з Ғдёә4D,(q_len, k_len)-> (1, 1, q_len, k_len)
            tgt_mask = tgt_mask.unsqueeze(0).unsqueeze(0)
            scores = torch.masked_fill(scores, tgt_mask, -1e9)
        # 4.еҪ’дёҖеҢ–
        attention_weights = F.softmax(scores, dim=-1)
        # 5.еҠ жқғжұӮе’Ң
        output = attention_weights @ value
        return output, attention_weights

"""

        Feed Forward еүҚйҰҲзҪ‘з»ң
            Add & Norm ж®Ӣе·®иҝһжҺҘ & еұӮеҪ’дёҖеҢ–

import torch

# и®ҫзҪ®дёӯж–Үеӯ—дҪ“

# 1.е®ҡд№үAttentionзұ»пјҢе®һзҺ°зј©ж”ҫзӮ№з§ҜжіЁж„ҸеҠӣжңәеҲ¶
class MyAttention(nn.Module):
    """
    Transformerзҡ„ зј©ж”ҫзӮ№з§ҜжіЁж„ҸеҠӣжңәеҲ¶
    иҫ“е…Ҙ:
        query/key/value: (batch_size, seq_len, d_model)
        padding_mask(еҸҜйҖү): (batch_size, seq_len),еЎ«е……жҺ©з ҒпјҢTrue=иў«йҒ®и”Ҫ/жҺ©з ҒпјҢFalse=жІЎжңүиў«йҒ®и”Ҫ
        tgt_mask(еҸҜйҖү,Decoderдё“з”Ё): (tgt_seq_len, tgt_seq_len),зӣ®ж ҮеәҸеҲ—зҡ„еЎ«е……жҺ©з ҒпјҢйҳІжӯўзңӢеҲ°жңӘжқҘ/йҒ®и”ҪжңӘжқҘtokenпјҢTrue=иў«йҒ®и”Ҫ/жҺ©з ҒпјҢFalse=жІЎжңүиў«йҒ®и”Ҫ
    иҝ”еӣһ:
        output: (batch_size, seq_len, d_model)
        attention_weights: (batch_size, seq_len, seq_len)
    """
    # 1.е®ҡд№ү__init__ж–№жі•
    def __init__(self, dropout=0.1):
        super().__init__()
        # 1.еҲқе§ӢеҢ–dropoutеұӮ
        if dropout is not None:
            self.dropout = nn.Dropout(dropout)
        else:
            # е®ҡд№үдёҖдёӘжҒ’зӯүеұӮ,е®һзҺ°д»Җд№ҲйғҪдёҚеҒҡ, y=x
            self.dropout = nn.Identity()
    # 2.forwardж–№жі•
    def forward(self, query, key, value, padding_mask=None, tgt_mask=None):
        # query/key/value: (batch_size, seq_len, d_model),(batch_size,num_heads,seq_len,d_head)
        # 1.иҺ·еҸ–еҸӮж•°batch_size, seq_len, d_model
        batch_size = query.shape[0]
        d_model = query.shape[-1]
        q_len = query.shape[-2]
        k_len = key.shape[-2]
        # 2.и®Ўз®—зј©ж”ҫзӮ№з§ҜжіЁж„ҸеҠӣпјҢжү“еҲҶ-еҪ’дёҖеҢ–-еҠ жқғжұӮе’Ң
        scores = query @ key.transpose(-2, -1) / d_model**0.5
        # 3.еӨ„зҗҶжҺ©з ҒпјҢеЎ«е……жҺ©з ҒпјҢеӣ жһңжҺ©з Ғ
        # 3.1 еӨ„зҗҶеЎ«е……жҺ©з Ғ, padding_mask(еҸҜйҖү): (batch_size, k_len)
        if padding_mask is not None:
            # 1.з»ҙеәҰж ЎйӘҢ
            if padding_mask.dim() == 2:
                # 2.ж ЎйӘҢеҪўзҠ¶(batch_size, k_len)
                assert padding_mask.shape == (batch_size, k_len), f"еЎ«е……жҺ©з Ғзҡ„еҪўзҠ¶{padding_mask.shape}й”ҷиҜҜ, йңҖдёә{(batch_size, k_len)}"
                # 3.жү©еұ•еЎ«е……жҺ©з Ғдёә4D,(batch_size, k_len)-> (batch_size, 1, 1, k_len)
                padding_mask = padding_mask.unsqueeze(1).unsqueeze(2)
            elif padding_mask.dim() == 3:
                # 4.жү©еұ•еЎ«е……жҺ©з Ғдёә4D,(batch_size,*, k_len)-> (batch_size, 1, *, k_len)
                padding_mask = padding_mask.unsqueeze(1)
            scores = torch.masked_fill(scores, padding_mask, -1e9)
        # 3.2 еӨ„зҗҶеӣ жһңжҺ©з ҒпјҢtgt_mask(еҸҜйҖү,Decoderдё“з”Ё): (q_len, k_len)
        if tgt_mask is not None:
            assert tgt_mask.shape == (q_len, k_len), f"еӣ жһңжҺ©з Ғзҡ„еҪўзҠ¶{tgt_mask.shape}й”ҷиҜҜ, йңҖдёә{(q_len, k_len)}"
            # жү©еұ•еӣ жһңжҺ©з Ғдёә4D,(q_len, k_len)-> (1, 1, q_len, k_len)
            tgt_mask = tgt_mask.unsqueeze(0).unsqueeze(0)
            scores = torch.masked_fill(scores, tgt_mask, -1e9)
        # 4.еҪ’дёҖеҢ–
        attention_weights = F.softmax(scores, dim=-1)
        # 5.еҠ жқғжұӮе’Ң
        output = attention_weights @ value
        return output, attention_weights

# 3.е®ҡд№үеӨҡеӨҙжіЁж„ҸеҠӣжңәеҲ¶зұ»
class MyMultiHeadAttention(nn.Module):
    """
    еҸӮж•°пјҡ
        d_model: QKVзҡ„зү№еҫҒз»ҙеәҰпјҢйҖҡеёёдёә512,1024,2048
        num_heads: еӨҡеӨҙжіЁж„ҸеҠӣзҡ„еӨҙж•°пјҢйҖҡеёёдёә8,16, иҰҒж»Ўи¶іж•ҙйҷӨе…ізі»пјҢd_model % num_heads == 0
        dropout: еҸҜйҖүпјҢй»ҳи®Ө0.1
    иҫ“е…Ҙ:
        query/key/value: (batch_size, seq_len, d_model)
        padding_mask(еҸҜйҖү): (batch_size, seq_len),еЎ«е……жҺ©з ҒпјҢTrue=иў«йҒ®и”Ҫ/жҺ©з ҒпјҢFalse=жІЎжңүиў«йҒ®и”Ҫ
        tgt_mask(еҸҜйҖү,Decoderдё“з”Ё): (tgt_seq_len, tgt_seq_len),зӣ®ж ҮеәҸеҲ—зҡ„еЎ«е……жҺ©з ҒпјҢйҳІжӯўзңӢеҲ°жңӘжқҘ/йҒ®и”ҪжңӘжқҘtokenпјҢTrue=иў«йҒ®и”Ҫ/жҺ©з ҒпјҢFalse=жІЎжңүиў«йҒ®и”Ҫ
    иҝ”еӣһ:
        output: (batch_size, seq_len, d_model)
        attention_weights: (batch_size, seq_len, seq_len)
    """
    # 1.е®ҡд№ү__init_ж–№жі•
    def __init__(self, d_model=512, num_heads=8, dropout=0.1):
        super().__init__()
        # 1.еҲқе§ӢеҢ–еҸӮж•°
        self.d_model = d_model
        self.num_heads = num_heads
        # 2.ж ЎйӘҢж•ҙж•°е…ізі»пјҡd_model % num_heads == 0
        assert d_model % num_heads == 0, f"d_model={d_model}дёҚиғҪиў«num_heads={num_heads}ж•ҙйҷӨ!"
        self.d_head = d_model // num_heads
        # 3.е®ҡд№үзәҝжҖ§еұӮпјҢ4дёӘзәҝжҖ§еұӮпјҢquery/key/value/output
        self.linear_q = nn.Linear(d_model, d_model)
        self.linear_k = nn.Linear(d_model, d_model)
        self.linear_v = nn.Linear(d_model, d_model)
        self.linear_out = nn.Linear(d_model, d_model)
        # 4.е®ҡд№үзј©ж”ҫзӮ№з§ҜжіЁж„ҸеҠӣжңәеҲ¶
        self.attention = MyAttention(dropout=dropout)
        self.attention_weights = None   # з”ЁдәҺеҸҜи§ҶеҢ–жіЁж„ҸеҠӣжқғйҮҚзҹ©йҳө
    # 2.forwardж–№жі•
    def forward(self, query, key, value, padding_mask=None, tgt_mask=None):
        # query/key/value: (batch_size, seq_len, d_model)
        # 0.иҺ·еҸ–batch_size
        batch_size = query.shape[0]
        # 1.з»ҸиҝҮзәҝжҖ§еұӮеҫ—еҲ°ж–°зҡ„QKV
        query = self.linear_q(query)    # (batch_size, seq_len, d_model)
        key = self.linear_k(key)    # (batch_size, seq_len, d_model)
        value = self.linear_v(value)    # (batch_size, seq_len, d_model)
        # 2.жӢҶеҲҶQKVдёәеӨҡдёӘеӨҙ
        # BSD(batch_size,seq_len,d_model) -> BSNH -> BNSH(batch_size,num_heads,seq_len,d_head)
        query = query.view(batch_size, -1, self.num_heads, self.d_head).transpose(1, 2) # BNSH
        key = key.view(batch_size, -1, self.num_heads, self.d_head).transpose(1, 2) # BNSH
        value = value.view(batch_size, -1, self.num_heads, self.d_head).transpose(1, 2) # BNSH
        # 3.и®Ўз®—жҜҸдёӘеӨҙзҡ„жіЁж„ҸеҠӣ

            query, key, value,
            padding_mask=padding_mask,
            tgt_mask=tgt_mask
        )
        # output: BNSH(batch_size,num_heads,seq_len,d_head)
        # attention_weights: BNSS(batch_size,num_heads,seq_len,seq_len)

        output = output.transpose(1, 2).reshape(batch_size, -1, self.d_model)   # BSD(batch_size,seq_len,d_model)
        # 5.з»ҸиҝҮзәҝжҖ§еұӮ
        output = self.linear_out(output)
        return output, self.attention_weights

# еҜјеҢ…
import torch
import torch.nn as nn
import torch.nn.functional as F

class MyLayerNorm(nn.Module):
    # 1.еҲқе§ӢеҢ–ж–№жі•
    def __init__(self, normalized_shape, eps=1e-6):
        super().__init__()
        # еҲқе§ӢеҢ–еҸӮж•°
        self.eps = eps
        self.normalized_shape = normalized_shape
        # еҲқе§ӢеҢ–еҸҜеӯҰд№ еҸӮж•°
        # зј©ж”ҫзі»ж•°еҲқе§ӢеҢ–дёә1
        self.gamma = nn.Parameter(torch.ones(normalized_shape))
        # еҒҸзҪ®зі»ж•°еҲқе§ӢеҢ–дёә0
        self.beta = nn.Parameter(torch.zeros(normalized_shape))
    # 2.forwardж–№жі•
    def forward(self, x):
        # иҫ“е…Ҙx: (batch_size, seq_len, d_model)
        # 1.и®Ўз®—жңҖеҗҺдёҖдёӘиҪҙзҡ„еқҮеҖје’Ңж ҮеҮҶе·®
        mean = torch.mean(x,-1, keepdim=True) # (batch_size, seq_len, 1)
        std = torch.std(x,-1, keepdim=True) # (batch_size, seq_len, 1)
        # 2.иҝӣиЎҢж ҮеҮҶеҢ–еӨ„зҗҶ
        x = (x - mean) / (std**2 + self.eps)**0.5
        # 3.иҝӣиЎҢзј©ж”ҫе’ҢеҒҸз§»
        x = self.gamma * x + self.beta
        return x

"""

# 1.е®ҡд№үжЁЎеһӢзұ»пјҢе®һзҺ°е®Ңж•ҙзҡ„transformer encoder
class MyTransformerEncoder(nn.Module):

class MyTransformerDecoder(nn.Module):

        self.decoder_layers = nn.ModuleList([
            MyTransformerDecoderLayer(self.d_model, self.num_heads, self.d_ff, self.dropout) for _ in range(self.num_layers)
        ])
        # 3.еҲӣе»әеұӮеҪ’дёҖеҢ–
        self.layer_norm = MyLayerNorm(self.d_model)
    # 2.forwardж–№жі•

        """
        еҸӮж•°пјҡ
            x,еҸіз§»дёҖдҪҚзҡ„зӣ®ж ҮеәҸеҲ—зҡ„иЎЁзӨәпјҢ(batch_size, tgt_seq_len, d_model)

"""
жј”зӨә
    дҪҝз”ЁиҮӘе®ҡд№үзҡ„зұ»пјҢе®һзҺ°е®Ңж•ҙзҡ„transformerжЁЎеһӢ

    и§Јз ҒеҷЁdecoder
    иҫ“еҮәйғЁеҲҶ
ж•°жҚ®пјҡ
    иҫ“е…Ҙж•°жҚ®:

        tgt_input: (batch_size, tgt_seq_len)
        src_mask: (batch_size, src_seq_len), жәҗеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
        tgt_mask: (tgt_seq_len, tgt_seq_len), зӣ®ж ҮеәҸеҲ—зҡ„еӣ жһңжҺ©з Ғ
    иҫ“еҮәж•°жҚ®пјҡ
        logits: (batch_size, tgt_seq_len, tgt_vocab_size), йў„жөӢеҲҶж•°
        encoder_output: (batch_size, src_seq_len, d_model), зј–з ҒеҷЁиҫ“еҮә
        decoder_output: (batch_size, tgt_seq_len, d_model), и§Јз ҒеҷЁиҫ“еҮә
"""

# еҜјеҢ…
import torch
import torch.nn as nn
import torch.nn.functional as F

# from transformer_decoder import *
# from transformer_output import *

# 1.е®ҡд№үжЁЎеһӢзұ»пјҢе®һзҺ°е®Ңж•ҙзҡ„transformerжЁЎеһӢ
class MyTransformer(nn.Module):
    """
    еҸӮж•°:
        d_model: зү№еҫҒз»ҙеәҰ
        num_layers:зј–з ҒеҷЁ/и§Јз ҒеҷЁеұӮж•°
        num_heads: жіЁж„ҸеҠӣеӨҙж•°
        d_ff: еүҚйҰҲзҪ‘з»ңзҡ„йҡҗи—ҸеұӮз»ҙеәҰ
        dropout: йҡҸжңәеӨұжҙ»жҰӮзҺҮ
        src_vocab_size: жәҗеәҸеҲ—зҡ„иҜҚиЎЁеӨ§е°Ҹ
        tgt_vocab_size: зӣ®ж ҮеәҸеҲ—зҡ„иҜҚиЎЁеӨ§е°Ҹ
        max_len: дҪҚзҪ®зј–з Ғзҡ„жңҖеӨ§еәҸеҲ—й•ҝеәҰ
    иҫ“е…Ҙ:
        src: (batch_size, src_seq_len)
        tgt_input: (batch_size, tgt_seq_len)
        src_mask: (batch_size, src_seq_len), жәҗеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
        tgt_mask: (tgt_seq_len, tgt_seq_len), зӣ®ж ҮеәҸеҲ—зҡ„еӣ жһңжҺ©з Ғ
    иҫ“еҮә:
        logits: (batch_size, tgt_seq_len, tgt_vocab_size), йў„жөӢеҲҶж•°
        encoder_output: (batch_size, src_seq_len, d_model), зј–з ҒеҷЁиҫ“еҮә
        decoder_output: (batch_size, tgt_seq_len, d_model), и§Јз ҒеҷЁиҫ“еҮә
    """
    # 1.еҲқе§ӢеҢ–__init__ж–№жі•
    def __init__(
        self, d_model=512, num_heads=8, d_ff=2048, dropout=0.1, num_layers=6, src_vocab_size=1000, tgt_vocab_size=2000, max_len=10000):
        super().__init__()
        # 1.еҲқе§ӢеҢ–еҸӮж•°
        self.d_model = d_model
        self.num_heads = num_heads
        # ж·»еҠ ж•ҙйҷӨж ЎйӘҢйҖ»иҫ‘пјҢd_model % num_heads == 0
        assert d_model % num_heads == 0, f"d_model={d_model} % num_heads={num_heads} != 0"
        self.d_ff = d_ff
        if dropout is not None:
            self.dropout = nn.Dropout(dropout)
        else:
            self.dropout = nn.Identity()
        self.num_layers = num_layers
        self.src_vocab_size = src_vocab_size
        self.tgt_vocab_size = tgt_vocab_size
        self.max_len = max_len
        # 2.е®ҡд№үжЁЎеқ—
        # иҫ“е…ҘйғЁеҲҶ
        # srcиҫ“е…Ҙ
        self.src_embed = nn.Sequential(
            MyEmbedding(vocab_size=src_vocab_size,d_model=d_model),
            MyPositionalEncoding(d_model=d_model,dropout=dropout,max_len=max_len)
        )
        # tgtиҫ“е…Ҙ
        self.tgt_embed = nn.Sequential(
            MyEmbedding(vocab_size=tgt_vocab_size,d_model=d_model),
            MyPositionalEncoding(d_model=d_model,dropout=dropout,max_len=max_len)
        )
        # зј–з ҒеҷЁ
        self.encoder = MyTransformerEncoder(
            d_model=d_model,
            num_heads=num_heads,
            d_ff=d_ff,
            dropout=dropout,
            num_layers=num_layers
        )
        # и§Јз ҒеҷЁ
        self.decoder = MyTransformerDecoder(
            d_model=d_model,
            num_heads=num_heads,
            d_ff=d_ff,
            dropout=dropout,
            num_layers=num_layers
        )
        # иҫ“еҮәйғЁеҲҶ
        self.output = MyTransformerOutput(
            d_model=d_model,
            vocab_size=tgt_vocab_size
        )

    # 2.forwardж–№жі•
    def forward(self, src, tgt_input, src_mask=None, tgt_mask=None):
        # src: (batch_size, src_seq_len)
        # tgt_input: (batch_size, tgt_seq_len)
        # src_mask: (batch_size, src_seq_len), жәҗеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
        # tgt_mask: (tgt_seq_len, tgt_seq_len), зӣ®ж ҮеәҸеҲ—зҡ„еӣ жһңжҺ©з Ғ
        # 1.з»ҸиҝҮиҫ“е…ҘйғЁеҲҶ
        src_embed = self.src_embed(src) # (batch_size, src_seq_len, d_model)
        tgt_embed = self.tgt_embed(tgt_input)   # (batch_size, tgt_seq_len, d_model)
        # 2.з»ҸиҝҮзј–з ҒеҷЁ
        encoder_output = self.encoder(x=src_embed, padding_mask=src_mask)
        # 3.з»ҸиҝҮи§Јз ҒеҷЁx, encoder_output, src_mask=None, tgt_mask=None
        decoder_output = self.decoder(
            x=tgt_embed,
            encoder_output=encoder_output,
            src_mask=src_mask,
            tgt_mask=tgt_mask
        )
        # 4.з»ҸиҝҮиҫ“еҮәйғЁеҲҶ
        logits = self.output(decoder_output)
        return logits
    # 3.encodeж–№жі•
    def encode(self, src, src_mask=None):
        # src: (batch_size, src_seq_len)
        # src_mask: (batch_size, src_seq_len), жәҗеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
        # 1.з»ҸиҝҮиҫ“е…ҘйғЁеҲҶ
        src_embed = self.src_embed(src)  # (batch_size, src_seq_len, d_model)
        # 2.з»ҸиҝҮзј–з ҒеҷЁ
        encoder_output = self.encoder(x=src_embed, padding_mask=src_mask)
        return encoder_output
    # 4.decodeж–№жі•
    def decode(self, tgt_input, encoder_output, src_mask=None, tgt_mask=None):
        # tgt_input: (batch_size, tgt_seq_len)
        # encoder_output: (batch_size, src_seq_len, d_model), зј–з ҒеҷЁзҡ„иҫ“еҮә
        # src_mask: (batch_size, src_seq_len), жәҗеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
        # tgt_mask: (tgt_seq_len, tgt_seq_len), зӣ®ж ҮеәҸеҲ—зҡ„еӣ жһңжҺ©з Ғ
        # 1.з»ҸиҝҮиҫ“е…ҘйғЁеҲҶ
        tgt_embed = self.tgt_embed(tgt_input)  # (batch_size, tgt_seq_len, d_model)
        # 2.з»ҸиҝҮи§Јз ҒеҷЁ
        decoder_output = self.decoder(
            x=tgt_embed,
            encoder_output=encoder_output,
            src_mask=src_mask,
            tgt_mask=tgt_mask
        )
        return decoder_output

# 2.е®ҡд№үеҮҪж•°пјҢжөӢиҜ•MyTransformerжЁЎеһӢ

"""
жј”зӨә
    дҪҝз”Ёnn.Transformerе®һзҺ°дёҖдёӘ е®Ңж•ҙзҡ„transformerжЁЎеһӢ

    иҫ“е…Ҙе’Ңиҫ“еҮәйңҖиҰҒжүӢеҠЁе®һзҺ°пјҡ
        - иҜҚеөҢе…ҘеұӮnn.Embedding
        - дҪҚзҪ®зј–з Ғnn.Embedding
            е·ҘзЁӢдёҠпјҢйҮҮз”ЁеҸҜеӯҰд№ зҡ„дҪҚзҪ®зј–з Ғnn.Embedding(max_len,d_model)
            дҪҚзҪ®зј–з Ғзҡ„жңҖеӨ§еәҸеҲ—й•ҝеәҰйҮҮз”Ёmax_len,жҳҜеӣ дёәпјҡ

        - з»ҹдёҖдёә2Dеј йҮҸ
        - tgt_mask: (tgt_seq_len,tgt_seq_len)
            дёҠдёүи§’зҹ©йҳөпјҢдёҠдёүи§’йғҪдёәTrue, еҜ№и§’зәҝеҸҠдёӢйқўдёәFalse
            Decoderзҡ„жҺ©з ҒиҮӘжіЁж„ҸеҠӣжңәеҲ¶зҡ„еӣ жһңжҺ©з ҒпјҢдҝқиҜҒDecoderеҸӘиғҪзңӢеҲ°еҪ“еүҚдҪҚзҪ®д»ҘеҸҠд№ӢеүҚдҪҚзҪ®пјҢйҳІжӯўзңӢеҲ°жңӘжқҘдҝЎжҒҜ
        - src_key_padding_mask: (batch_size,src_seq_len)
            з”ЁдәҺEncoderзҡ„иҮӘжіЁж„ҸеҠӣжңәеҲ¶

        - tgt_key_padding_mask: (batch_size,tgt_seq_len)
            з”ЁдәҺDecoderзҡ„жҺ©з ҒиҮӘжіЁж„ҸеҠӣжңәеҲ¶
            targetеәҸеҲ—зҡ„еЎ«е……жҺ©з ҒпјҢиЎЁзӨәе“ӘдәӣдҪҚзҪ®иў«PADеЎ«е……пјҢйңҖиҰҒиҝӣиЎҢйҒ®и”Ҫ/mask
        - memory_key_padding_mask: (batch_size,src_seq_len) = src_key_padding_mask
            з”ЁдәҺDecoderзҡ„дәӨеҸүжіЁж„ҸеҠӣжңәеҲ¶
            sourceеәҸеҲ—зҡ„еЎ«е……жҺ©з ҒпјҢиЎЁзӨәе“ӘдәӣдҪҚзҪ®иў«PADеЎ«е……пјҢйңҖиҰҒиҝӣиЎҢйҒ®и”Ҫ/mask
    3.еҸӮж•°batch_first = True
        дҪҝеҫ—жүҖжңүиҫ“е…Ҙиҫ“еҮәзҡ„еҪўзҠ¶дёә(batch_size, seq_len, *)

"""

# еҜјеҢ…
import torch
import torch.nn as nn
import torch.nn.functional as F

# 1.е®ҡд№үжЁЎеһӢзұ»пјҢе®һзҺ°дёҖдёӘе®Ңж•ҙзҡ„TransformerжЁЎеһӢ
# 1.1 е…Ҳе®ҡд№үдёҖдёӘеҸҜеӯҰд№ зҡ„дҪҚзҪ®зј–з Ғзұ»
class LearnablePositionalEncoding(nn.Module):
    # 1.е®ҡд№ү__init__ж–№жі•
    def __init__(self,d_model=512,max_len=1000):
        super().__init__()
        # 1.еҲқе§ӢеҢ–еҸҜеӯҰд№ зҡ„дҪҚзҪ®зј–з ҒпјҢдҪҝз”Ёnn.Embedding
        self.position_encoding = nn.Embedding(max_len,d_model)
    # 2.forwardж–№жі•
    def forward(self,x):
        # x: (batch_size, seq_len, d_model), иҜҚеҗ‘йҮҸ
        # 1.е…ҲиҺ·еҸ–seq_len
        seq_len = x.shape[1]
        # 2.жһ„йҖ seq_lenзҡ„зҙўеј•еј йҮҸпјҡ0~seq_len-1,torch.arange(0,seq_len,)
        position_ids = torch.arange(0,seq_len,dtype=torch.long, device=x.device)
        # (seq_len,) -> (1, seq_len)
        position_ids = position_ids.unsqueeze(0)
        # 3.иҝӣиЎҢдҪҚзҪ®зј–з Ғ,(1, seq_len) -> (1, seq_len, d_model)
        pos_encoding = self.position_encoding(position_ids)
        # 4.иҝ”еӣһдҪҚзҪ®зј–з Ғ+иҜҚеҗ‘йҮҸx,(batch_size, seq_len, d_model)
        return x + pos_encoding

# 1.2 е®ҡд№үжЁЎеһӢзұ»пјҢе®һзҺ°дёҖдёӘе®Ңж•ҙзҡ„transformerжЁЎеһӢ
class MyTransformer(nn.Module):
    """
    дҪҝз”Ёnn.Transformerе®һзҺ°дёҖдёӘе®Ңж•ҙзҡ„transformerжЁЎеһӢ
    еҸӮж•°:
        d_model: зү№еҫҒз»ҙеәҰ
        n_heads: жіЁж„ҸеҠӣеӨҙж•°
        num_layers: зј–з ҒеҷЁ/и§Јз ҒеҷЁеұӮж•°
        d_ff: еүҚйҰҲзҪ‘з»ңеұӮзҡ„йҡҗи—ҸеұӮз»ҙеәҰ
        dropout: дёўејғжҰӮзҺҮ
        batch_first=True: и®©иҫ“е…Ҙиҫ“еҮәзҡ„еҪўзҠ¶еӣәе®ҡдёә(batch_size,seq_len,*)
        src_vocab_size: жәҗеәҸеҲ—зҡ„иҜҚиЎЁеӨ§е°Ҹ
        tgt_vocab_size: зӣ®ж ҮеәҸеҲ—зҡ„иҜҚиЎЁеӨ§е°Ҹ
        max_src_len: жәҗеәҸеҲ—дҪҚзҪ®зј–з Ғзҡ„жңҖеӨ§й•ҝеәҰпјҢйҖҡеёёжҳҜsrc_seq_lenзҡ„2~4еҖҚ
        max_tgt_len: зӣ®ж ҮеәҸеҲ—дҪҚзҪ®зј–з Ғзҡ„жңҖеӨ§й•ҝеәҰпјҢйҖҡеёёжҳҜtgt_seq_lenзҡ„2~4еҖҚ

    иҫ“е…Ҙ:
        src: (batch_size, src_seq_len)
        tgt: (batch_size, tgt_seq_len)
        src_key_padding_mask: (batch_size, src_seq_len), жәҗеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
        tgt_key_padding_mask: (batch_size, tgt_seq_len), зӣ®ж ҮеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
        tgt_mask: (tgt_seq_len, tgt_seq_len), зӣ®ж ҮеәҸеҲ—зҡ„еӣ жһңжҺ©з Ғ
    иҝ”еӣһ:
        logits: (batch_size, tgt_seq_len, tgt_vocab_size),йў„жөӢеҲҶж•°
        encoder_output: (batch_size, src_seq_len, d_model), зј–з ҒеҷЁзҡ„иҫ“еҮә
        decoder_output: (batch_size, tgt_seq_len, d_model), и§Јз ҒеҷЁзҡ„иҫ“еҮә
    """
    # 1.е®ҡд№ү__init__ж–№жі•
    def __init__(
        self,d_model=512,n_heads=8,num_layers=6,d_ff=2048,dropout=0.1,
        batch_first=True,src_vocab_size=1000,tgt_vocab_size=2000,max_src_len=1000,max_tgt_len=1000
                 ):
        super().__init__()
        # 1.еҲқе§ӢеҢ–еҸӮж•°
        self.d_model = d_model
        self.n_heads = n_heads
        self.num_layers = num_layers
        self.d_ff = d_ff
        self.dropout = dropout
        self.batch_first = batch_first
        self.src_vocab_size = src_vocab_size
        self.tgt_vocab_size = tgt_vocab_size
        self.max_src_len = max_src_len
        self.max_tgt_len = max_tgt_len
        # 2.е®ҡд№үжЁЎеқ—
        # 2.1 иҜҚеөҢе…ҘеұӮ
        self.src_embedding = nn.Embedding(src_vocab_size,d_model)
        self.tgt_embedding = nn.Embedding(tgt_vocab_size,d_model)
        # 2.2 еҸҜеӯҰд№ зҡ„дҪҚзҪ®зј–з Ғ
        self.src_position_encoding = LearnablePositionalEncoding(d_model=d_model,max_len=max_src_len)
        self.tgt_position_encoding = LearnablePositionalEncoding(d_model=d_model,max_len=max_tgt_len)
        # 2.3 е®ҡд№үTransformerзҡ„зј–з ҒеҷЁ-и§Јз ҒеҷЁжЁЎеқ—
        self.transformer = nn.Transformer(
            d_model=d_model,    # зү№еҫҒз»ҙеәҰ
            nhead=n_heads,      # жіЁж„ҸеҠӣеӨҙж•°
            num_encoder_layers=num_layers,  # зј–з ҒеҷЁеұӮж•°
            num_decoder_layers=num_layers,  # и§Јз ҒеҷЁеұӮж•°
            dim_feedforward=d_ff,   # еүҚйҰҲзҪ‘з»ңеұӮзҡ„йҡҗи—ҸеұӮз»ҙеәҰ
            dropout=dropout,    # дёўејғжҰӮзҺҮ
            batch_first=batch_first,    # и®©иҫ“е…Ҙиҫ“иҫ“еҮәзҡ„еҪўзҠ¶еӣәе®ҡдёә(batch_size,seq_len,*)
            norm_first=True, # еӯҗеұӮиҝһжҺҘйЎәеәҸ, TrueиЎЁзӨәpre-LN: LSDA, LayerNorm -> SubLayer -> Add -> Norm; FalseиЎЁзӨәpost-LN: SDAL
        )
        # 2.4 зәҝжҖ§еұӮ, з”ЁдәҺе°Ҷзј–з ҒеҷЁзҡ„иҫ“еҮәжҳ е°„еҲ°зӣ®ж ҮиҜҚиЎЁеӨ§е°Ҹ
        self.fc_out = nn.Linear(d_model,tgt_vocab_size)
    # 2.forwardж–№жі•
    def forward(
        self,src,tgt,src_key_padding_mask=None,tgt_key_padding_mask=None,tgt_mask=None):
        # src: (batch_size, src_seq_len),иҫ“е…Ҙзҡ„жәҗеәҸеҲ—
        # tgt: (batch_size, tgt_seq_len),иҫ“е…Ҙзҡ„зӣ®ж ҮеәҸеҲ—
        # src_key_padding_mask: (batch_size, src_seq_len), жәҗеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
        # tgt_key_padding_mask: (batch_size, tgt_seq_len), зӣ®ж ҮеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
        # tgt_mask: (tgt_seq_len, tgt_seq_len), зӣ®ж ҮеәҸеҲ—зҡ„еӣ жһңжҺ©з Ғ
        # 1.з»ҸиҝҮиҜҚеөҢе…ҘеұӮ
        src_embedding = self.src_embedding(src) # (batch_size, src_seq_len, d_model)
        tgt_embedding = self.tgt_embedding(tgt) # (batch_size, tgt_seq_len, d_model)
        # 2.ж·»еҠ дҪҚзҪ®зј–з Ғзҡ„иҜҚеҗ‘йҮҸ
        src_embedding = self.src_position_encoding(src_embedding)   # (batch_size, src_seq_len, d_model)
        tgt_embedding = self.tgt_position_encoding(tgt_embedding)   # (batch_size, tgt_seq_len, d_model)
        # 3.з»ҸиҝҮTransformerжЁЎеқ—,д№ҹе°ұжҳҜз»ҸиҝҮзј–з ҒеҷЁ-и§Јз ҒеҷЁжЁЎеқ—
        output = self.transformer(
            src_embedding,
            tgt_embedding,
            src_key_padding_mask=src_key_padding_mask,  # жәҗеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
            tgt_key_padding_mask=tgt_key_padding_mask,  # зӣ®ж ҮеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
            memory_key_padding_mask=src_key_padding_mask,   # жәҗеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ, з”ЁдәҺDecoderзҡ„дәӨеҸүжіЁж„ҸеҠӣжңәеҲ¶
            tgt_mask=tgt_mask,  # зӣ®ж ҮеәҸеҲ—зҡ„еӣ жһңжҺ©з Ғ
        )   # (batch_size, tgt_seq_len, d_model)
        # 4.з»ҸиҝҮзәҝжҖ§еұӮ
        logits = self.fc_out(output)    # (batch_size, tgt_seq_len, tgt_vocab_size)
        return logits
    # 3.encodeж–№жі•,е·ҘзЁӢдёҠйҖҡеёёд№ҹиҰҒе®һзҺ°transformerзҡ„зј–з ҒеҷЁж–№жі•
    def encode(self,src,src_key_padding_mask=None):
        # 1.з»ҸиҝҮиҜҚеөҢе…ҘеұӮ
        src_embedding = self.src_embedding(src)  # (batch_size, src_seq_len, d_model)
        # 2.ж·»еҠ дҪҚзҪ®зј–з Ғ
        src_embedding = self.src_position_encoding(src_embedding)  # (batch_size, src_seq_len, d_model)
        # 3.з»ҸиҝҮTransformer.encoderжЁЎеқ—,иҝӣиЎҢзј–з Ғ
        encoder_output = self.transformer.encoder(
            src_embedding,
            src_key_padding_mask=src_key_padding_mask,  # жәҗеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
        )   # (batch_size, src_seq_len, d_model)
        return encoder_output
    # 4.decodeж–№жі•
    def decode(
        self,tgt,encoder_output,src_key_padding_mask=None,tgt_key_padding_mask=None,tgt_mask=None):
        # tgt: (batch_size, tgt_seq_len),иҫ“е…Ҙзҡ„зӣ®ж ҮеәҸеҲ—
        # encoder_output: (batch_size, src_seq_len, d_model), зј–з ҒеҷЁзҡ„иҫ“еҮә
        # src_key_padding_mask: (batch_size, src_seq_len), жәҗеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
        # tgt_key_padding_mask: (batch_size, tgt_seq_len), зӣ®ж ҮеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
        # tgt_mask: (tgt_seq_len, tgt_seq_len), зӣ®ж ҮеәҸеҲ—зҡ„еӣ жһңжҺ©з Ғ
        # 1.з»ҸиҝҮиҜҚеөҢе…ҘеұӮ
        tgt_embedding = self.tgt_embedding(tgt)
        # 2.ж·»еҠ дҪҚзҪ®зј–з Ғ
        tgt_embedding = self.tgt_position_encoding(tgt_embedding)
        # 3.з»ҸиҝҮTransformer.decoderжЁЎеқ—,иҝӣиЎҢи§Јз Ғ
        decoder_output = self.transformer.decoder(
            tgt_embedding,
            encoder_output,
            tgt_key_padding_mask=tgt_key_padding_mask,  # зӣ®ж ҮеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ
            memory_key_padding_mask=src_key_padding_mask,   # жәҗеәҸеҲ—зҡ„еЎ«е……жҺ©з Ғ, з”ЁдәҺDecoderзҡ„дәӨеҸүжіЁж„ҸеҠӣжңәеҲ¶
            tgt_mask=tgt_mask,  # зӣ®ж ҮеәҸеҲ—зҡ„еӣ жһңжҺ©з Ғ
        )
        return decoder_output

# е®ҡд№үеҮҪж•°,жөӢиҜ•MyTransformer
def test_MyTransformer():

    # жЁЎеһӢзӣёе…і
    d_model = 512
    num_layers = 6

    tgt_seq_len = 20
    # и®ӯз»ғзӣёе…і

    # 2.еҲӣе»әж•°жҚ®
    src = torch.randint(0, src_vocab_size, (batch_size, src_seq_len))

    print(f"targetзӣ®ж ҮеәҸеҲ—: {tgt.shape}")
    # 3.еҲӣе»әжҺ©з Ғ
    # еЎ«е……жҺ©з Ғ(batch_size, seq_len),иҝҷйҮҢз®ҖеҢ–дёәе…ЁжҳҜFalse,д№ҹе°ұжҳҜжІЎжңүPADеЎ«е……
    src_key_padding_mask = torch.zeros(batch_size, src_seq_len, dtype=torch.bool)
    tgt_key_padding_mask = torch.zeros(batch_size, tgt_seq_len, dtype=torch.bool)

10 жү©еұ•-Transformer иҜҚеөҢе…Ҙзј©ж”ҫеҺҹеӣ 

10.1 Embedding еҲқе§ӢеҢ–-pytorchй»ҳи®ӨеҲқе§ӢеҢ–ж–№е·®дёә1/d_model

  • PyTorch зҡ„ nn.Embedding(vocab_size, d_model) й»ҳи®ӨеҲқе§ӢеҢ–жҜҸдёӘе…ғзҙ пјҡ $$x_i \sim \mathcal{N}(0, \sigma^2)$$

  • дёәдәҶи®©ж•ҙдҪ“еҗ‘йҮҸжҜҸз»ҙж–№е·® = 1/d_modelпјҢйҖҡеёёпјҡ $$\sigma = \frac{1}{\sqrt{d_\text{model}}}$$

  • еҲҷжҜҸдёӘеҗ‘йҮҸ $\mathbf{x} = (x_1, \dots, x_{d_\text{model}})$ зҡ„жҖ»ж–№е·®пјҡ $$\text{Var}[\mathbf{x}] = \sum_{i=1}^{d_\text{model}} \text{Var}[x_i] = d_\text{model} \cdot \frac{1}{d_\text{model}} = 1$$

вң… иҝҷж ·еҒҡзҡ„еҘҪеӨ„пјҡдҝқиҜҒиҜҚеҗ‘йҮҸж•ҙдҪ“е°әеәҰдёҺдҪҚзҪ®зј–з Ғзӣёиҝ‘гҖӮ

10.2 пёҸдҪҚзҪ®зј–з Ғзҡ„е°әеәҰ

  • жӯЈејҰ-дҪҷејҰдҪҚзҪ®зј–з Ғпјҡ $$\text{PE}(pos, 2i) = \sin\left(\frac{pos}{10000^{2i/d}}\right), \quad \text{PE}(pos, 2i+1) = \cos\left(\frac{pos}{10000^{2i/d}}\right)$$

  • иҫ“еҮәиҢғеӣҙпјҡ$[-1, 1]$

  • еҜ№жүҖжңүз»ҙеәҰз»ҹи®Ўпјҡ $$\mathbb{E}[\text{PE}] \approx 0, \quad \text{Var}[\text{PE}] \approx 0.5$$

  • дёҺ embedding иҫ“еҮәеҜ№йҪҗпјҢйңҖиҰҒж•ҙдҪ“ж–№е·®жҺҘиҝ‘ 1гҖӮ

10.3 пёҸEmbedding иҫ“еҮәзј©ж”ҫ

  • з»ҸиҝҮзј©ж”ҫпјҡ $$\mathbf{x}{\text{scaled}} = \mathbf{x} \cdot \sqrt{d\text{model}}$$

  • еҚ•з»ҙж–№е·®пјҡ $$\text{Var}[x_i \cdot \sqrt{d_\text{model}}] = \frac{1}{d_\text{model}} \cdot d_\text{model} = 1$$

  • ж•Ҳжһңпјҡ

  • дҝқжҢҒ embedding дёҺдҪҚзҪ®зј–з ҒгҖҒжіЁж„ҸеҠӣи®Ўз®—иҫ“е…Ҙзҡ„е°әеәҰдёҖиҮҙ
  • йҒҝе…ҚжіЁж„ҸеҠӣзӮ№з§ҜиҝҮе°ҸпјҢжўҜеәҰж¶ҲеӨұ

10.4 пёҸжҖ»з»“

жЁЎеқ— еҚ•з»ҙж–№е·® жҖ»ж–№е·® зј©ж”ҫ
nn.Embedding еҲқе§ӢеҢ– $\frac{1}{d_\text{model}}$ 1 д№ҳ $\sqrt{d_\text{model}}$
PE дҪҚзҪ®зј–з Ғ $\approx 0.5$ $\approx \frac{d_\text{model} \cdot 0.5}{d_\text{model}}$ дҝқжҢҒдёҚеҸҳ
Attention зӮ№з§Ҝ вҖ” вҖ” йҷӨ $\sqrt{d_\text{model}}$ дҝқжҢҒ softmax иҫ“е…ҘзЁіе®ҡ

def get_positional_encoding(max_len, d_model):
    pe = torch.zeros(max_len, d_model)
    position = torch.arange(0, max_len, dtype=torch.float).unsqueeze(1)  # (max_len, 1)
    div_term = torch.exp(torch.arange(0, d_model, 2).float() * (-math.log(10000.0) / d_model))
    pe[:, 0::2] = torch.sin(position * div_term)   # еҒ¶ж•°з»ҙеәҰ: sin
    pe[:, 1::2] = torch.cos(position * div_term)   # еҘҮж•°з»ҙеәҰ: cos
    return pe
