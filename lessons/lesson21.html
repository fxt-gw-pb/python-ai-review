📊 Matplotlib 与数据可视化

🏷️ 样式 / 子图布局🏷️ 统计图 / 3D🏷️ EDA / 性能优化

可视化是数据挖掘的关键辅助工具：把数据变成图形，更直观、更客观、更有说服力。本讲从 Matplotlib 的"三步画图法"开始，掌握刻度、图例、网格、多图等全部辅助功能，认识五种常见图形的适用场景，最后学习两个更高层的画图姿势——Pandas 自带绘图和 Seaborn。

Matplotlib 是 Python 使用最多的 2D（含 3D）绘图库，官方案例库 https://matplotlib.org/stable/gallery/index.html 值得收藏——先找长得像的图，再抄代码改数据。

一、快速入门：三步画图法

matplotlib.pyplot 包含一系列类似 MATLAB 的画图函数，约定导入方式：

Python
import matplotlib.pyplot as plt

任何图都是三步：

  1. 创建画布：plt.figure(figsize=(长, 宽), dpi=清晰度)；
  2. 绘制图像：plt.plot(x, y)（以折线图为例）；
  3. 显示图像：plt.show()。
Python
import matplotlib.pyplot as plt

# 1. 创建画布
plt.figure(figsize=(10, 10), dpi=100)
# 2. 绘制折线图: 展现一周7天的天气温度
plt.plot([1, 2, 3, 4, 5, 6, 7], [17, 17, 18, 15, 11, 11, 13])
# 3. 显示图像
plt.show()

Matplotlib 图像的组成结构（了解）：画布 Figure > 绘图区 Axes > 坐标轴、刻度、标题、图例、网格、折线/散点等元素。

二、完善折线图：全部辅助功能

需求：画出某城市 11 点到 12 点每分钟的温度变化折线图（温度在 15~18 度随机波动）。借这个例子把常用辅助功能一次配齐。

Python
import matplotlib.pyplot as plt
import random
from pylab import mpl

# 中文显示(方案二: 脚本中动态设置; 运行一次即可)
mpl.rcParams["font.sans-serif"] = ["SimHei"]   # 设置显示中文字体
mpl.rcParams["axes.unicode_minus"] = False     # 字体更改后负号可能异常, 需要一并设置

# 0. 准备数据: random.uniform(a, b) 返回 [a, b] 间均匀分布的随机浮点数
x = range(60)
y_shanghai = [random.uniform(15, 18) for i in x]
y_beijing = [random.uniform(1, 3) for i in x]

# 1. 创建画布
plt.figure(figsize=(20, 8), dpi=100)

# 2. 绘制图像: 多次 plot 即可在同一坐标系中画多条线; label 配合图例使用
plt.plot(x, y_shanghai, label="上海")
plt.plot(x, y_beijing, color="r", linestyle="--", label="北京")

# 2.1 自定义x,y轴刻度: plt.xticks(刻度值, 刻度标签), [::5] 每5个显示一个
x_ticks_label = ["11点{}分".format(i) for i in x]
y_ticks = range(40)
plt.xticks(x[::5], x_ticks_label[::5])
plt.yticks(y_ticks[::5])

# 2.2 添加网格: linestyle 网格线风格, alpha 透明度
plt.grid(True, linestyle="--", alpha=0.5)

# 2.3 添加描述信息: 轴标签与标题(fontsize 控制字号)
plt.xlabel("时间")
plt.ylabel("温度")
plt.title("中午11点--12点某城市温度变化图", fontsize=20)

# 2.4 保存图片: 必须在 show() 之前! show()会释放画布资源, 之后再存只能存空图
plt.savefig("./test.png")

# 2.5 显示图例: 只在 plot 里写 label 还不够, 必须调用 legend()
plt.legend(loc="best")

# 3. 显示图像
plt.show()

几个重点展开：

中文显示问题有两种解法：方案一是下载 SimHei 字体安装到系统、删除 ~/.matplotlib 缓存并修改 matplotlibrc 配置文件；方案二（推荐）就是上面代码里的两行 rcParams 动态设置。

线条风格——plt.plot 的 color 与 linestyle 参数：

颜色字符 风格字符
r 红、g 绿、b 蓝、w 白 - 实线
c 青、m 洋红、y 黄、k 黑 -- 虚线、-. 点划线、: 点虚线、' ' 留空

图例位置——plt.legend(loc=) 可用字符串或代码：best(0)、upper right(1)、upper left(2)、lower left(3)、lower right(4)、right(5)、center left(6)、center right(7)、lower center(8)、upper center(9)、center(10)。

三、多个坐标系：plt.subplots

想把上海和北京分别画在同一张图的两个坐标系里，用 plt.subplots（面向对象的画图方法，推荐替代旧版 subplot）：

  • fig, axes = plt.subplots(nrows=1, ncols=2, figsize=(20, 8), dpi=100)——nrows/ncols 设置几行几列坐标系，返回图对象 fig 和坐标系数组 axes；
  • 注意 API 变化：plt.函数名() 是面向过程写法；换成 axes 后要用 axes[i].set_方法名() 的面向对象写法。
Python
fig, axes = plt.subplots(nrows=1, ncols=2, figsize=(20, 8), dpi=100)

axes[0].plot(x, y_shanghai, label="上海")
axes[1].plot(x, y_beijing, color="r", linestyle="--", label="北京")

# 刻度: set_xticks + set_xticklabels(分开设置)
axes[0].set_xticks(x[::5])
axes[0].set_xticklabels(x_ticks_label[::5])
axes[0].set_yticks(y_ticks[::5])

# 网格 / 标签 / 标题 / 图例
axes[0].grid(True, linestyle="--", alpha=0.5)
axes[0].set_xlabel("时间")
axes[0].set_ylabel("温度")
axes[0].set_title("上海温度变化", fontsize=20)
axes[0].legend(loc=0)
# axes[1] 同理...

plt.show()

对照记忆：plt.xticks() → axes.set_xticks() + axes.set_xticklabels()；plt.title() → axes.set_title()。

四、五种常见图形与适用场景

图形 API 特点与场景
折线图 plt.plot(x, y) 显示数据变化趋势：活跃用户数、下载量随时间变化
柱状图 plt.bar(x, height, width, align='center', color=) 离散数据的大小对比/统计
直方图 plt.hist(x, bins=组数) 连续数据的分布状况
饼图 plt.pie(x, labels=, autopct='%1.2f%%', colors=) 分类数据的占比
散点图 plt.scatter(x, y) 两变量间的关联/分布规律、离群点
Python
# 柱状图
categories = ['A', 'B', 'C', 'D']
values = [3, 7, 5, 4]
plt.bar(categories, values, color='blue')
plt.title("Simple Bar Chart")
plt.show()

# 直方图: 500个标准正态随机数的分布
data = np.random.randn(500)
plt.hist(data, bins=30, color='blue', alpha=0.7, rwidth=0.85)
# bins=30 分成30个等宽区间; alpha 透明度; rwidth 条形宽度占区间的85%
plt.show()

# 饼图: autopct 显示占比格式
sizes = [25, 35, 25, 15]
labels = ['Category A', 'Category B', 'Category C', 'Category D']
plt.pie(sizes, labels=labels, autopct='%1.1f%%')
plt.show()

# 散点图
plt.scatter([1, 2, 3, 4, 5], [2, 3, 5, 7, 11], color='red')
plt.show()

五、Pandas 自带绘图

Series 和 DataFrame 都自带 .plot() 系列方法（底层就是 matplotlib），做探索性分析时"统计 + 出图"一行连写非常顺手。以 15 万条葡萄酒评价数据为例：

Python
import pandas as pd
df = pd.read_csv('./data/winemag-data_first150k.csv', index_col=0)

# 需求1: 产葡萄酒最多的10个产地
# value_counts() 统计每个值出现的次数, 等价于 groupby + count + 降序排序
df['province'].value_counts().head(10)
# 等价写法: df.groupby('province')['province'].count().sort_values(ascending=False).head(10)

# 可视化: 统计结果直接 .plot.bar() 出柱状图
text_kwargs = dict(fontsize=15, figsize=(10, 6),
                   color=['r', 'g', 'b', 'y', 'c', 'm', 'k', 'pink', 'gray', 'orange'])
df['province'].value_counts().head(10).plot.bar(**text_kwargs)

# 需求2: 前10产地的"占比"(除以总数)
(df['province'].value_counts().head(10) / len(df)).plot.bar(**text_kwargs)

# 需求3: 每个评分的葡萄酒数量
df['points'].value_counts().sort_index().plot.bar(**text_kwargs)   # 柱状图
df['points'].value_counts().sort_index().plot.line(**text_kwargs)  # 折线图
df['points'].value_counts().sort_index().plot.area(**text_kwargs)  # 面积图 = 折线图下方填充颜色

选图小结论：分类型数据用柱状图；取值固定、有序的数据可以直接用折线图/面积图。

六、Seaborn 绘图入门

Seaborn 底层依赖 Matplotlib、集成 Pandas 数据结构，API 更简洁（pip install seaborn）。命名规律：图形名 + plot——直方图 histplot、散点图 scatterplot、箱线图 boxplot……统一的用法是 sns.xxxplot(data=df, x=列名, y=列名, hue=分组列)。

以内置的小费数据集 tips 为例：

Python
import seaborn as sns
import matplotlib.pyplot as plt

df = sns.load_dataset('tips')     # 加载内置数据集(需联网); 也可 pd.read_csv('./data/tips.csv')

# 1. 计数柱状图: 每天男女的账单笔数
sns.countplot(data=df, x='day', hue='sex')

# 2. 直方图: 男女总账单的分布
fig, ax = plt.subplots(figsize=(10, 6))
sns.histplot(data=df, x='total_bill', hue='sex')
ax.set_title('男女总账单分布情况', fontsize=20)
plt.show()

# 3. 散点图: 总账单与小费的关系
sns.scatterplot(data=df, x='total_bill', y='tip', hue='sex')

# 4. regplot: 散点 + 一条拟合回归线(默认线性回归)
sns.regplot(data=df, x='total_bill', y='tip')

# 5. jointplot: 联合分布图, kind='reg' 回归 / kind='hex' 蜂巢图
sns.jointplot(data=df, x='total_bill', y='tip', kind='hex')

# 6. 箱线图: 午餐/晚餐的账单金额分布(中位数、四分位、离群点一图看全)
sns.boxplot(data=df, x='time', y='total_bill')

# 7. 小提琴图: 箱线图 + 分布形状; split=True 时按 hue 把提琴左右分半对比
sns.violinplot(data=df, x='time', y='total_bill', hue='sex', split=True)

3. 图例的核心参数

2. 堆叠柱状图

3. 性能优化

  • 无界面服务器先 matplotlib.use('Agg')，且必须在导入 pyplot 之前设置；
  • 不要在循环里反复 plot 数十万次，先把数据整理为数组再一次绘制；
  • 大量点可下采样、聚合、使用 hexbin 或栅格化 rasterized=True；
  • 临时采用 with plt.style.context('fast'): 可减少部分渲染开销；
  • 批量出图后调用 plt.close(fig) 释放画布，避免内存持续增长。
Python
import matplotlib
matplotlib.use('Agg')
import matplotlib.pyplot as plt

for name, values in datasets.items():
    fig, ax = plt.subplots()
    ax.plot(values)
    fig.savefig(f'{name}.png', dpi=150, bbox_inches='tight')
    plt.close(fig)

十四、本讲小结

主题 关键点
三步画图 figure（figsize/dpi）→ plot → show
辅助功能 xticks/yticks 刻度、grid 网格、xlabel/title 描述、savefig（必须在 show 前）、legend 图例
中文显示 rcParams["font.sans-serif"]=["SimHei"] + axes.unicode_minus=False
多坐标系 fig, axes = plt.subplots(nrows, ncols)；axes 用 set_xxx() 面向对象写法
选图 趋势→折线；对比→柱状；分布→直方；占比→饼；相关→散点
Pandas 绘图 value_counts() + .plot.bar()/.line()/.area() 一行出图
Seaborn 图形名+plot 命名；data/x/y/hue 四参数走天下；regplot/jointplot/boxplot/violinplot
样式与编码 线型/Marker/alpha；散点颜色、大小与 colorbar；图例外置和多图例管理
子图布局 subplots 为主；GridSpec/subplot2grid 做不规则布局；sharex/sharey 统一比较尺度
进阶图形 分组/堆叠/横向柱状图；嵌套饼图；箱线图；热力图；3D 散点/曲线/曲面
EDA 时间序列 + 分布/KDE + 箱线 + 滞后散点互相验证，避免单图下结论
性能与导出 Agg/fast、批量绘制、下采样/栅格化、close；savefig 的 dpi/bbox/矢量格式

画图的顺序永远是：先想清楚"要表达什么"（趋势/对比/分布/占比/关系），再选图形，最后才是调样式。
