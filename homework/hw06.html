🗂️ 数据容器综合练习

🏷️ 实操🏷️ 17 题📖 相关讲义：数据容器：列表、元组、字典与集合

列表、元组、字典、集合四大容器 + 推导式热身 + 一道函数题收尾。每种容器的「增删改查」手感不同，动手敲一遍印象最深。

一、列表操作

1. 基础练习

创建一个包含 5 个水果名称的列表，然后：① 在列表末尾添加 "西瓜"；② 在第二个位置插入 "香蕉"；③ 删除第三个元素；④ 打印最终列表。

✅ 参考答案
Python
fruits = ['苹果', '葡萄', '橙子', '梨', '桃子']

fruits.append('西瓜')       # ① 末尾添加
fruits.insert(1, '香蕉')    # ② 第二个位置 = 下标 1
fruits.pop(2)               # ③ 第三个元素 = 下标 2（此时是'葡萄'）

print(fruits)   # ['苹果', '香蕉', '橙子', '梨', '桃子', '西瓜']

解析：注意"第二个位置"对应下标 1——人数的第 N 个 = 下标 N−1。插入香蕉后列表变了，第三个元素也跟着变，删除时要以当时的列表为准。

2. 嵌套列表操作

给定二维列表 matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]：① 提取所有偶数到一个新列表；② 计算所有元素的和。

✅ 参考答案
Python
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

# 1- 提取所有偶数
evens = []
for row in matrix:
    for num in row:
        if num % 2 == 0:
            evens.append(num)
print(evens)    # [2, 4, 6, 8]

# 2- 求所有元素的和
total = 0
for row in matrix:
    for num in row:
        total += num
print(total)    # 45

解析：二维列表要用两层 for——外层取行、内层取数。熟练后可用推导式一行完成：[n for row in matrix for n in row if n % 2 == 0]。

3. 列表排序和去重

给定 numbers = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]：① 升序排序；② 去除重复元素；③ 找出最大值和最小值。

✅ 参考答案
Python
numbers = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]

# 1- 升序排序
numbers.sort()
print(numbers)             # [1, 1, 2, 3, 3, 4, 5, 5, 5, 6, 9]

# 2- 去重：转集合再转回列表
unique = list(set(numbers))
unique.sort()
print(unique)              # [1, 2, 3, 4, 5, 6, 9]

# 3- 最大值和最小值
print(max(numbers), min(numbers))   # 9 1

解析：set() 去重是最常用技巧，但集合是无序的，转回列表后一般要重新排一下序。

4. 学生成绩列表

给定 scores = [85, 92, 78, 90, 88, 76, 95, 89, 84, 91]：① 计算平均分；② 找出高于平均分的成绩；③ 按从高到低排序；④ 统计优秀（90 分及以上）人数。

✅ 参考答案
Python
scores = [85, 92, 78, 90, 88, 76, 95, 89, 84, 91]

# 1- 平均分
avg = sum(scores) / len(scores)
print(f"平均分: {avg}")               # 86.8

# 2- 高于平均分的成绩
above = []
for s in scores:
    if s > avg:
        above.append(s)
print(above)                         # [92, 90, 88, 95, 89, 91]

# 3- 从高到低排序
scores.sort(reverse=True)
print(scores)   # [95, 92, 91, 90, 89, 88, 85, 84, 78, 76]

# 4- 优秀人数（90 分及以上）
count = 0
for s in scores:
    if s >= 90:
        count += 1
print(f"优秀人数: {count}")            # 4

二、元组操作

1. 元组基础操作

创建一个包含 5 个三国人物姓名的元组，然后：① 打印元组长度；② 打印第三个元素；③ 访问最后一个元素；④ 统计"诸葛亮"出现的次数。

✅ 参考答案
Python
heroes = ('刘备', '关羽', '张飞', '诸葛亮', '赵云')

print(len(heroes))              # 5
print(heroes[2])                # 张飞（第三个 = 下标 2）
print(heroes[-1])               # 赵云
print(heroes.count('诸葛亮'))    # 1

解析：元组的查询操作（下标、切片、count、index、len）和列表完全一样，区别只在于元组不可修改。

2. 元组解包

给定 student = ("张三", 18, "男", 90.5)，使用元组解包把姓名、年龄、性别、成绩分别赋给不同变量并打印。

✅ 参考答案
Python
student = ("张三", 18, "男", 90.5)

name, age, gender, score = student

print(name)     # 张三
print(age)      # 18
print(gender)   # 男
print(score)    # 90.5

解析：解包时左边变量个数必须与元组元素个数一致，否则报 ValueError。

3. 元组不可变验证

创建元组 numbers = (1, 2, 3, 4, 5)，尝试执行以下操作，观察结果并解释原因：

Python
numbers[0] = 10
numbers.append(6)
✅ 参考答案

两行都会报错：

Python
numbers = (1, 2, 3, 4, 5)

numbers[0] = 10
# TypeError: 'tuple' object does not support item assignment

numbers.append(6)
# AttributeError: 'tuple' object has no attribute 'append'

解析：元组是不可变类型——既不能修改已有元素（不支持下标赋值），也压根没有 append、remove 这类修改方法。需要"可修改的元组"时，请改用列表。

4. 元组方法应用

给定 fruits = ("苹果", "香蕉", "橙子", "苹果", "葡萄", "香蕉", "苹果")：① 统计"苹果"出现的次数；② 找出"香蕉"第一次出现的位置；③ 获取元组长度。

✅ 参考答案
Python
fruits = ("苹果", "香蕉", "橙子", "苹果", "葡萄", "香蕉", "苹果")

print(fruits.count('苹果'))    # 3
print(fruits.index('香蕉'))    # 1
print(len(fruits))            # 7

5. 元组和列表转换

给定 colors = ["红色", "绿色", "蓝色", "黄色", "紫色"]：① 列表转元组；② 对新元组切片取前 3 个；③ 元组转回列表；④ 添加"橙色"；⑤ 再转成元组。

✅ 参考答案
Python
colors = ["红色", "绿色", "蓝色", "黄色", "紫色"]

colors_tuple = tuple(colors)          # ① 列表 → 元组
print(colors_tuple[:3])               # ② ('红色', '绿色', '蓝色')

colors_list = list(colors_tuple)      # ③ 元组 → 列表
colors_list.append("橙色")            # ④ 列表才能添加
final_tuple = tuple(colors_list)      # ⑤ 再转回元组

print(final_tuple)
# ('红色', '绿色', '蓝色', '黄色', '紫色', '橙色')

解析：tuple() 与 list() 互转是绕过"元组不可变"限制的标准做法：转列表 → 修改 → 转回元组。

三、字典操作

1. 字典基础操作

创建学生信息字典 student = {"name": "张三", "age": 18, "gender": "男", "score": 90}：① 添加键值对 "class": "一班"；② 修改年龄为 19；③ 删除性别信息；④ 打印所有键和所有值。

✅ 参考答案
Python
student = {"name": "张三", "age": 18, "gender": "男", "score": 90}

student["class"] = "一班"    # ① 键不存在 → 添加
student["age"] = 19          # ② 键已存在 → 修改
del student["gender"]        # ③ 删除键值对

print(student.keys())        # dict_keys(['name', 'age', 'score', 'class'])
print(student.values())      # dict_values(['张三', 19, 90, '一班'])

解析：字典[键] = 值 一个语法两种效果——键不存在就是新增，存在就是修改。

2. 字典遍历

给定 fruit_prices = {"苹果": 5.5, "香蕉": 3.0, "橙子": 4.2, "葡萄": 6.8}：① 遍历并打印所有水果名称；② 计算所有水果价格的平均值。

✅ 参考答案
Python
fruit_prices = {"苹果": 5.5, "香蕉": 3.0, "橙子": 4.2, "葡萄": 6.8}

# 1- 遍历所有键
for name in fruit_prices.keys():
    print(name)

# 2- 价格平均值
total = 0
for price in fruit_prices.values():
    total += price
avg = total / len(fruit_prices)
print(f"平均价格: {avg:.2f}")    # 4.88

解析：keys() 取键、values() 取值、items() 键值一起取，三大遍历方式要熟。

3. 嵌套字典

Python
students = {
    "001": {"name": "李华", "scores": {"math": 85, "english": 92, "science": 78}},
    "002": {"name": "王明", "scores": {"math": 90, "english": 88, "science": 95}}
}

① 获取学号 "001" 的学生的英语成绩；② 为学号 "002" 的学生添加新科目成绩 "history": 87。

✅ 参考答案
Python
# 1- 一层层用键往里取
print(students["001"]["scores"]["english"])   # 92

# 2- 定位到 scores 字典后直接新增键值对
students["002"]["scores"]["history"] = 87
print(students["002"]["scores"])
# {'math': 90, 'english': 88, 'science': 95, 'history': 87}

解析：嵌套字典就像套娃，students["001"] 取出的还是字典，可以继续用键访问，一路点到目标为止。

4. 综合应用：员工信息管理

Python
employees = [
    {"id": "E1001", "name": "张三", "department": "技术部", "salary": 8000},
    {"id": "E1002", "name": "李四", "department": "市场部", "salary": 7500},
    {"id": "E1003", "name": "王五", "department": "技术部", "salary": 8500}
]

① 添加新员工 {"id": "E1004", "name": "赵六", "department": "财务部", "salary": 7000}；② 求所有人的平均工资。

✅ 参考答案
Python
# 1- 列表 append 添加新员工字典
employees.append({"id": "E1004", "name": "赵六",
                  "department": "财务部", "salary": 7000})

# 2- 遍历取每个字典的 salary 求平均
total = 0
for emp in employees:
    total += emp["salary"]
avg = total / len(employees)
print(f"平均工资: {avg}")    # 7750.0

解析：「列表套字典」是最贴近真实开发的数据结构（相当于数据库的多行记录），遍历列表、按键取值是基本功。

四、集合操作

1. 综合练习

Python
numbers = [1, 2, 2, 3, 4, 4, 4, 5, 5, 6, 7, 7, 8, 9, 9, 9]

① 使用集合去除重复元素；② 统计原列表中有多少种不同的数字；③ 找出所有出现次数超过 2 次的数字（提示：可以先统计次数）。

✅ 参考答案
Python
numbers = [1, 2, 2, 3, 4, 4, 4, 5, 5, 6, 7, 7, 8, 9, 9, 9]

# 1- 集合去重
unique = set(numbers)
print(unique)              # {1, 2, 3, 4, 5, 6, 7, 8, 9}

# 2- 不同数字的种数
print(len(unique))         # 9

# 3- 出现超过 2 次的数字
result = set()
for n in unique:
    if numbers.count(n) > 2:
        result.add(n)
print(result)              # {9, 4}

解析：先去重再对每个不同数字 count，避免重复统计。4 和 9 各出现 3 次，超过 2 次。

五、列表推导式

1. 生成 1~10 的立方列表

✅ 参考答案
Python
cubes = [i ** 3 for i in range(1, 11)]
print(cubes)
# [1, 8, 27, 64, 125, 216, 343, 512, 729, 1000]

2. 生成 1~20 中所有 3 的倍数的平方列表

✅ 参考答案
Python
squares = [i ** 2 for i in range(1, 21) if i % 3 == 0]
print(squares)
# [9, 36, 81, 144, 225, 324]

解析：带条件的推导式结构是 [表达式 for 变量 in 序列 if 条件]。3 的倍数是 3、6、9、12、15、18，平方后得到结果。也可以用 range(3, 21, 3) 直接按步长生成。

六、函数练习

1. 模拟计算器

使用函数方式定义一个计算器：函数接收两个整数，返回这两个整数的加、减、乘、除结果。

✅ 参考答案
Python
def calculator(a, b):
    he = a + b
    cha = a - b
    ji = a * b
    shang = a / b
    return he, cha, ji, shang   # 返回多个值，自动打包成元组


result = calculator(10, 5)
print(result)          # (15, 5, 50, 2.0)

# 也可以用四个变量直接拆包接收
he, cha, ji, shang = calculator(10, 5)
print(f"和:{he} 差:{cha} 积:{ji} 商:{shang}")

解析：return a, b, c, d 实际返回的是一个元组；接收端既可以整体接收，也可以按个数拆包。除法要留意 b 为 0 的情况。
