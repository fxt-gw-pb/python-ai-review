🧮 运算符、分支与循环

🏷️ 代码阅读 + 实操🏷️ 20 题📖 相关讲义：循环与字符串

本套练习分四个部分：运算符求值、if 分支应用、循环双解法（while 与 for 各写一遍）、综合编程。循环题请务必两种写法都练。

一、运算符

1. 整除与取余

Python
a = 15
b = 4
result = a // b + a % b
print(result)

result 的值是多少？

✅ 参考答案
Text
6

解析：15 // 4 = 3（整除取商），15 % 4 = 3（取余数），3 + 3 = 6。

2. 幂运算混合

Python
a, b = 10, 3
result = a ** b + a // b - a % b
print(result)

输出结果是什么？

✅ 参考答案
Text
1002

解析：10 ** 3 = 1000，10 // 3 = 3，10 % 3 = 1，即 1000 + 3 - 1 = 1002。

3. 复合赋值（一）

Python
x = 5
x **= 2
x += 1
print(x)

打印出的 x 值是多少？

✅ 参考答案
Text
26

解析：x **= 2 等价于 x = x ** 2，得 25；再 +1 得 26。

4. 复合赋值（二）

Python
x = 5
x += 3
x *= 2
x -= 4
print(x)

输出结果是什么？

✅ 参考答案
Text
12

解析：5 → +3 → 8 → ×2 → 16 → −4 → 12。复合赋值按顺序逐行执行。

5. 比较与逻辑运算

Python
m = 10
n = 20
print(m < 15 and n > 15)

打印出的结果是 True 还是 False？

✅ 参考答案
Text
True

解析：10 < 15 为 True，20 > 15 为 True，True and True → True。

6. and 与 or

Python
a, b, c = 7, 5, 7
print(a == c and a > b)
print(a != b or b > c)

两行分别输出什么？

✅ 参考答案
Text
True
True

解析：第一行 7 == 7 且 7 > 5，True。第二行 7 != 5 已经为 True，or 短路，直接得 True。

7. not 的优先级

Python
x, y, z = True, False, True
result = not (x and y) or z
print(result)

输出结果是什么？

✅ 参考答案
Text
True

解析：先算括号 x and y = False；not False = True；True or z → True。

8. 综合求值

Python
a = 7
b = 3
c = a > b and a % b == 1
print(c)

打印出的 c 值是多少？

✅ 参考答案
Text
True

解析：7 > 3 为 True；7 % 3 = 1，1 == 1 为 True；True and True → True。比较/逻辑表达式的结果也可以赋值给变量。

二、if 分支判断

1. 年龄验证

如果年龄大于等于 18，打印"已成年"，否则打印"未成年"。（初始值 age = 17）

✅ 参考答案
Python
age = 17
if age >= 18:
    print("已成年")
else:
    print("未成年")

输出：未成年。

2. 数字正负判断

判断 number = -5 是正数、负数还是零，并打印相应结果。

✅ 参考答案
Python
number = -5
if number > 0:
    print(f"{number} 是正数")
elif number < 0:
    print(f"{number} 是负数")
else:
    print("这个数是 0")

解析：三种互斥情况用 if / elif / else 一次说清，输出 -5 是负数。

3. 成绩等级判断

根据分数打印对应等级（score = 85）：90 分以上优秀；80–89 良好；70–79 中等；60–69 及格；60 以下不及格。

✅ 参考答案
Python
score = 85
if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 70:
    print("中等")
elif score >= 60:
    print("及格")
else:
    print("不及格")

输出：良好。

解析：elif 从上往下判断，走到 score >= 80 时前面的 >= 90 一定不成立，所以不需要再写 score < 90。

4. 用户名密码验证

输入的用户名为 "admin" 且密码为 "123456" 时打印"登录成功"，否则打印"用户名或密码错误"。

✅ 参考答案
Python
username = input("请输入用户名：")
password = input("请输入密码：")

if username == 'admin' and password == '123456':
    print("登录成功")
else:
    print("用户名或密码错误")

解析：两个条件必须同时成立，用 and 连接。

5. 购物折扣计算

输入购物金额：金额 ≥ 1000 打 8 折；≥ 500 打 9 折；≥ 200 打 95 折；其他原价。计算折后价格并打印。

✅ 参考答案
Python
money = float(input("请输入购物金额："))

if money >= 1000:
    pay = money * 0.8
elif money >= 500:
    pay = money * 0.9
elif money >= 200:
    pay = money * 0.95
else:
    pay = money

print(f"折后金额为：{pay:.2f}")

解析：条件从大到小排列，命中即止；先算出 pay 再统一打印，比在每个分支里各写一个 print 更清爽。

三、while / for 循环

1. 求 50–100 的和

分别用 while 和 for 两种写法求 50 到 100（含两端）所有整数的和。

✅ 参考答案
Python
# while 写法
total = 0
i = 50
while i <= 100:
    total += i
    i += 1
print(f"50-100 的和: {total}")

# for 写法
total = 0
for i in range(50, 101):
    total += i
print(f"50-100 的和: {total}")

输出：3825。

⚠️ 易错点：range(50, 101) 右端是开区间，想包含 100 就要写 101；while 循环别忘了 i += 1，否则死循环。

2. 跳过 5 的倍数求和

求 1–100 的和，但跳过所有能被 5 整除的数字。（提示：continue）

✅ 参考答案
Python
# while 写法
total = 0
i = 1
while i <= 100:
    if i % 5 == 0:
        i += 1        # 注意：continue 前必须先让 i 前进
        continue
    total += i
    i += 1
print(f"跳过 5 的倍数后的和: {total}")

# for 写法
total = 0
for i in range(1, 101):
    if i % 5 == 0:
        continue
    total += i
print(f"跳过 5 的倍数后的和: {total}")

输出：4000。

解析：continue 跳过本轮剩余代码。while 版本里如果忘了在 continue 前 i += 1，i 会永远停在 5，造成死循环——这是 while + continue 的经典大坑。

3. 遇到 7 的倍数就停

求 1–100 的和，遇到第一个 7 的倍数就停止计算。（提示：break）

✅ 参考答案
Python
# while 写法
total = 0
num = 1
while num <= 100:
    if num % 7 == 0:
        break
    total += num
    num += 1
print(f"和为: {total}")

# for 写法
total = 0
for num in range(1, 101):
    if num % 7 == 0:
        break
    total += num
print(f"和为: {total}")

输出：21（只加了 1+2+…+6，遇到 7 就 break 了）。

4. 遇到第三个 7 的倍数才停

求 1–100 的和，遇到第三个 7 的倍数（即 21）时停止计算。

✅ 参考答案
Python
# while 写法
total = 0
count7 = 0
num = 1
while num <= 100:
    if num % 7 == 0:
        count7 += 1
        if count7 == 3:
            break
    total += num
    num += 1
print(f"和为: {total}")

# for 写法
total = 0
count7 = 0
for num in range(1, 101):
    if num % 7 == 0:
        count7 += 1
        if count7 == 3:
            break
    total += num
print(f"和为: {total}")

输出：210。

解析：需要一个计数器 count7 记录已经遇到几个 7 的倍数。注意 7 和 14 本身仍会被累加（没有 continue），到 21 时先判断到 count7 == 3 就 break，21 不会被加进去，所以结果是 1+2+…+20 = 210。

5. 密码尝试限制 ⭐ 难题

正确密码为 "admin888"，用户有 3 次尝试机会，用 break 在密码正确或次数用完时退出。

✅ 参考答案
Python
# for 写法
password = "admin888"
for i in range(3):
    pwd = input("请输入密码:")
    if pwd == password:
        print("密码正确")
        break
    elif 2 - i == 0:
        print("次数已用完，退出")
    else:
        print(f"录入有误，请重新输入，还有{2 - i}次机会")

# while 写法
password = "admin888"
i = 0
while i < 3:
    pwd = input("请输入密码:")
    if pwd == password:
        print("密码正确")
        break
    elif 2 - i == 0:
        print("次数已用完，退出")
    else:
        print(f"录入有误，请重新输入，还有{2 - i}次机会")
    i += 1

解析：剩余次数 = 总次数 3 − 已用次数 (i+1) = 2 - i。当 2 - i == 0 说明这是最后一次机会且输错了。

四、综合编程

1. 购物折扣计算器

让用户输入商品原价和购买数量，计算总金额并应用折扣规则：1–2 件无折扣；3–5 件 9 折；超过 5 件 8 折。要求使用 f-string 输出，保留 2 位小数。

✅ 参考答案
Python
# 1- 用户输入商品原价和数量
price = float(input("商品的原价:"))
num = int(input("商品的数量:"))

# 2- 计算总金额
total_price = price * num
print(f"总金额:{total_price:.2f}")

# 3- 按数量判断折扣
if num <= 2:
    print(f"优惠后金额:{total_price:.2f}（无折扣）")
elif num <= 5:
    total_price *= 0.9
    print(f"优惠后金额:{total_price:.2f}（已享 9 折优惠）")
else:
    total_price *= 0.8
    print(f"优惠后金额:{total_price:.2f}（已享 8 折优惠）")

解析：判断条件从小到大写时，elif num <= 5 隐含了 num > 2，不必重复书写区间下限。

2. 乘法表生成器

让用户输入一个 1–9 之间的整数 n，用 for 循环打印 n 的 1~9 乘法结果，格式为 n x i = result；若输入不在 1–9 范围内则提示重新输入。

例如输入 3：

Text
3 x 1 = 3
3 x 2 = 6
3 x 3 = 9
...
3 x 9 = 27
✅ 参考答案
Python
# 1- 获取合法输入（不在 1~9 范围就反复提示）
while True:
    n = int(input("请输入一个正整数(1~9):"))
    if 1 <= n <= 9:
        break
    print("输入不合法，请输入 1~9 之间的整数！")

# 2- for 循环打印乘法结果
for i in range(1, 10):
    print(f"{n} x {i} = {n * i}")

解析：用 while True + break 实现"验证输入直到合法"是非常常用的套路；range(1, 10) 生成 1~9。
