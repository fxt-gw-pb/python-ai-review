🛡️ 解包、lambda、文件与异常

🏷️ 实操🏷️ 12 题📖 相关讲义：文件操作与异常处理

最后一套练习：解包预测、lambda 改写、文件读写四连、异常捕获、自定义模块，压轴一道完整的学生管理系统。做完这套，第一阶段的 Python 基础就闭环了。

一、解包相关

1. 基础解包

预测以下代码的输出结果：

Python
coordinates = (30, 40)
x, y = coordinates
print(f"x = {x}, y = {y}")
✅ 参考答案
Text
x = 30, y = 40

解析：元组按位置解包，x 接第一个元素、y 接第二个。

2. 多重交换

预测以下代码执行后各变量的值：

Python
a, b, c = 1, 2, 3
a, b, c = b, c, a
print(a, b, c)
✅ 参考答案
Text
2 3 1

解析：右边的 b, c, a 会先用旧值打包成 (2, 3, 1)，再整体解包给左边——所以不需要中间变量就能完成轮换交换。

3. 嵌套解包

预测以下代码的输出结果：

Python
data = [(1, 2), (3, 4)]
(a, b), (c, d) = data
print(a + b + c + d)
✅ 参考答案
Text
10

解析：外层解包把两个元组分给 (a, b) 和 (c, d)，内层再各自拆开，得 a=1, b=2, c=3, d=4，和为 10。

二、lambda 函数

1. 基础转换

将以下普通函数转换为 lambda 表达式：

Python
def multiply(a, b):
    return a * b
✅ 参考答案
Python
multiply = lambda a, b: a * b

print(multiply(3, 4))   # 12

解析：lambda 的结构是 lambda 参数: 返回值表达式，适合"一行就能写完"的小函数。

2. 排序处理

按照年龄对学生列表进行降序排序，补全代码：

Python
students = [
    {"name": "德华", "age": 19, "score": 100},
    {"name": "成龙", "age": 15, "score": 90},
    {"name": "李连杰", "age": 18, "score": 95}
]
✅ 参考答案
Python
students = [
    {"name": "德华", "age": 19, "score": 100},
    {"name": "成龙", "age": 15, "score": 90},
    {"name": "李连杰", "age": 18, "score": 95}
]

students.sort(key=lambda x: x["age"], reverse=True)
print(students)
# 德华(19) → 李连杰(18) → 成龙(15)

解析：key 指定"按什么排"——lambda 从每个字典里取出 age；reverse=True 表示降序。这是 lambda 最高频的实战用法。

三、文件读写

1. 将字符串写入文件 test.txt

Python
content = "Hello, Python File IO!"
✅ 参考答案
Python
content = "Hello, Python File IO!"

f = open('test.txt', 'w', encoding='utf-8')
f.write(content)
f.close()

解析：'w' 模式：文件不存在则创建，存在则清空重写。打开时指定 encoding='utf-8' 可以避免中文乱码；用完记得 close()。

2. 读取文件 test.txt 的内容并打印

✅ 参考答案
Python
f = open('test.txt', 'r', encoding='utf-8')
content = f.read()
print(content)   # Hello, Python File IO!
f.close()

解析：'r' 是只读模式，read() 一次性读出全部内容。更推荐 with 写法（见下题），可以自动关闭文件。

3. 读取配置文件的第一行和第二行

config.txt 内容如下，读取第一行和第二行并打印：

Text
username=jepson
password=123456
✅ 参考答案
Python
with open('config.txt', 'r', encoding='utf-8') as f:
    print("第一行：", f.readline().strip())
    print("第二行：", f.readline().strip())

解析：readline() 每次读一行（含末尾换行符），strip() 把换行符去掉；with 语句块结束后自动关闭文件，不怕忘写 close。

4. 拷贝图片：a.png → b.png

拷贝 a.png 图片（自己准备一张），生成一个新的 b.png。

✅ 参考答案
Python
with open('a.png', 'rb') as f1, open('b.png', 'wb') as f2:
    f2.write(f1.read())

解析：图片是二进制文件，必须用 rb / wb 模式，并且不能指定 encoding（编码只对文本有意义）。一个 with 可以同时管理两个文件。

四、异常处理

1. 捕获文件不存在的错误

读取一个不存在的文件目录 aa/bb/cc，捕获错误并给出提示。

✅ 参考答案
Python
try:
    with open('aa/bb/cc', 'r', encoding='utf-8') as f:
        content = f.read()
except FileNotFoundError:
    print("错误提示：文件或目录 'aa/bb/cc' 不存在！")
except Exception as e:
    print(f"意外错误：{e}")

解析：把可能出错的代码放进 try；except FileNotFoundError 精确捕获"文件不存在"，最后再兜底一个 Exception 处理意料之外的错误。先精确、后兜底，顺序不能反。

五、自定义模块

1. 新建自定义模块 my_tools

在模块里写入一个求和函数、一个打招呼函数，然后在其他文件里导入调用。

✅ 参考答案

第一步：新建文件 my_tools.py（模块本体）：

Python
# my_tools.py
def greeting(name):
    print(f"hello，你好，{name}")


def my_sum(a, b):
    return a + b

第二步：在同目录的另一个文件（如 main.py）中导入并调用：

Python
# main.py
from my_tools import greeting, my_sum

if __name__ == '__main__':
    greeting("张三")               # hello，你好，张三
    result = my_sum(10, 20)
    print("求和结果：", result)     # 求和结果： 30

解析：模块就是一个 .py 文件，文件名就是模块名。from 模块 import 函数 后可直接用函数名调用。求和函数命名为 my_sum 而不是 sum，避免覆盖内置函数。if __name__ == '__main__': 保证这段代码只在直接运行时执行、被导入时不执行。

六、综合项目：学生管理系统

1. 学生管理系统 ⭐ 压轴题

实现一个命令行学生管理系统，支持以下功能：

  • 添加学生信息
  • 删除学生信息（根据编号删除）
  • 修改单个学生信息
  • 查询单个学生信息
  • 遍历查询所有学生信息
  • 退出系统
✅ 参考答案
Python
# 初始化学生列表，存储字典格式的学生信息
students = []


def show_menu():
    """显示功能菜单"""
    print("\n===== 学生管理系统 =====")
    print("1. 添加学生信息")
    print("2. 删除学生信息（按编号）")
    print("3. 修改学生信息")
    print("4. 查询单个学生信息")
    print("5. 遍历所有学生信息")
    print("6. 退出系统")
    print("========================")


def add_student():
    """添加学生信息"""
    student_id = input("请输入学生编号（唯一）：")
    # 检查编号是否重复
    for s in students:
        if s["id"] == student_id:
            print("错误：该编号已存在！")
            return
    name = input("请输入学生姓名：")
    age = input("请输入学生年龄：")
    score = input("请输入学生成绩：")
    students.append({
        "id": student_id,
        "name": name,
        "age": age,
        "score": score
    })
    print("学生信息添加成功！")


def delete_student():
    """删除学生信息（按编号）"""
    student_id = input("请输入要删除的学生编号：")
    for i, s in enumerate(students):
        if s["id"] == student_id:
            del students[i]
            print("学生信息删除成功！")
            return
    print("未找到该编号的学生！")


def modify_student():
    """修改单个学生信息"""
    student_id = input("请输入要修改的学生编号：")
    for s in students:
        if s["id"] == student_id:
            print("当前信息：", s)
            s["name"] = input("新姓名（回车不修改）：") or s["name"]
            s["age"] = input("新年龄（回车不修改）：") or s["age"]
            s["score"] = input("新成绩（回车不修改）：") or s["score"]
            print("学生信息修改成功！")
            return
    print("未找到该编号的学生！")


def query_student():
    """查询单个学生信息"""
    student_id = input("请输入要查询的学生编号：")
    for s in students:
        if s["id"] == student_id:
            print("\n===== 学生信息 =====")
            print(f"编号：{s['id']}")
            print(f"姓名：{s['name']}")
            print(f"年龄：{s['age']}")
            print(f"成绩：{s['score']}")
            print("====================")
            return
    print("未找到该编号的学生！")


def query_all_students():
    """遍历查询所有学生信息"""
    if not students:
        print("暂无学生信息！")
        return
    print("\n===== 所有学生信息 =====")
    for i, s in enumerate(students, 1):
        print(f"第{i}条：编号={s['id']}，姓名={s['name']}，"
              f"年龄={s['age']}，成绩={s['score']}")
    print("========================")


# 主程序
if __name__ == '__main__':
    while True:
        show_menu()
        choice = input("请输入功能编号（1-6）：")
        if choice == "1":
            add_student()
        elif choice == "2":
            delete_student()
        elif choice == "3":
            modify_student()
        elif choice == "4":
            query_student()
        elif choice == "5":
            query_all_students()
        elif choice == "6":
            print("退出系统，再见！")
            break
        else:
            print("输入错误，请输入1-6的数字！")

解析要点：

  1. 数据结构：students 是"列表套字典"，每个字典是一条学生记录——这正是前面容器练习的实战应用。
  2. 函数拆分：每个功能一个函数，主程序只负责菜单调度，结构清晰、易维护。
  3. enumerate(students, 1)：遍历时同时拿到序号（从 1 开始）和元素。
  4. input(...) or 旧值：直接回车时 input 返回空字符串（假值），or 就会取旧值——实现"回车跳过不修改"。
  5. 提前 return：找到目标处理完就返回，函数末尾的"未找到"提示只有循环走完才执行。
