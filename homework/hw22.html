🧭 KNN 与模型调优

🏷️ 选择 + 简答 + 代码🏷️ 35 题📖 相关讲义：K 近邻算法与模型调优

35 道练习，覆盖 KNN 分类与回归、距离度量、特征缩放、K 值选择、交叉验证、网格搜索及手写实现。

一、选择题（12 题）

1. KNN 分类预测新样本时，核心依据是什么？

A. 随机选 K 个样本
B. K 个最近邻居的类别投票
C. 只看训练集均值
D. 拟合一条直线

✅ 参考答案

答案：B。 KNN 先按距离找出最近的 K 个训练样本，再由多数投票或距离加权投票决定类别。

2. 当 K=1 时，KNN 通常具有怎样的特点？

A. 决策边界更平滑，偏差高
B. 对噪声敏感，方差较高
C. 永远欠拟合
D. 与数据无关

✅ 参考答案

答案：B。 1-NN 几乎记住训练样本，局部噪声会强烈影响预测，容易过拟合。

3. K 值过大更可能导致什么？

A. 模型过于复杂
B. 决策过度平滑并欠拟合
C. 距离无法计算
D. 测试数据泄漏

✅ 参考答案

答案：B。 邻域过大时局部结构被全局多数类淹没，偏差上升，容易欠拟合。

4. 为什么 KNN 对特征尺度敏感？

A. 它只接受整数
B. 距离会被数值范围大的特征主导
C. 它不能处理两列以上数据
D. 缩放会改变样本数量

✅ 参考答案

答案：B。 例如收入以万元计、年龄以岁计时，收入列可能在欧氏距离中占据绝大部分。

5. weights="distance" 的含义是什么？

A. 所有邻居权重相同
B. 距离越近的邻居影响越大
C. 自动删除远处样本
D. 只使用一个邻居

✅ 参考答案

答案：B。 sklearn 使用距离的倒数作为主要权重思想，近邻对预测贡献更大。

6. 二维空间中点 (1, 2) 与 (4, 6) 的欧氏距离是多少？

A. 4
B. 5
C. 7
D. 25

✅ 参考答案

答案：B。 距离为 sqrt((4-1)^2 + (6-2)^2) = sqrt(9+16) = 5。

7. 曼哈顿距离更接近哪种移动方式？

A. 两点间直线移动
B. 沿坐标轴横向和纵向移动
C. 只比较角度
D. 忽略所有特征

✅ 参考答案

答案：B。 曼哈顿距离是各维绝对差之和，像城市街区中沿网格行走。

8. KNN 的“训练阶段”通常做什么？

A. 主要保存训练数据
B. 迭代求解大量模型系数
C. 训练神经网络
D. 生成决策树

✅ 参考答案

答案：A。 KNN 是惰性学习，拟合开销小；主要计算发生在预测时的近邻搜索。

9. 在二分类任务中选奇数 K 的常见原因是什么？

A. 保证准确率为奇数
B. 减少投票平局的机会
C. 让距离变小
D. 省略标准化

✅ 参考答案

答案：B。 类别数为 2 时，奇数个邻居更不容易出现票数相同，但距离加权等机制也可处理平局。

10. metric="minkowski", p=1 等价于哪种距离？

A. 欧氏距离
B. 曼哈顿距离
C. 余弦距离
D. 汉明距离

✅ 参考答案

答案：B。 Minkowski 距离在 p=1 时为曼哈顿距离，p=2 时为欧氏距离。

11. KNN 回归的典型预测方式是什么？

A. 邻居目标值的均值或加权均值
B. 邻居类别多数投票
C. 只返回最大值
D. 输出随机数

✅ 参考答案

答案：A。 回归目标是连续值，通常对邻居目标值求平均；距离加权时近邻占比更大。

12. 高维数据中 KNN 性能可能下降的主要原因是什么？

A. 维度灾难使距离区分度降低
B. 高维一定没有标签
C. KNN 只支持二维
D. 标准化函数失效

✅ 参考答案

答案：A。 维度升高后样本变稀疏，最近与最远距离趋于接近，需要更多数据或先做特征选择/降维。

二、简答题（11 题）

13. 说明 KNN 的三个关键要素

✅ 参考答案

三个关键要素是：距离度量（如何定义“近”）、K 值（参考多少邻居）和决策规则（多数投票、距离加权或回归平均）。

14. 为什么标准化器只能在训练集上 fit？

✅ 参考答案

fit 会计算均值、标准差等统计量。如果使用测试集参与计算，就把测试分布信息泄漏到训练流程。测试集只能调用训练阶段已经拟合好的 transform。

15. KNN 的主要优点和缺点各列举两点

✅ 参考答案

优点：原理直观、几乎无需参数训练、能形成非线性边界。缺点：预测时计算和存储开销大、对尺度/噪声/高维敏感，并且解释大量样本时较慢。

16. 如何通过训练分数和验证分数判断 K 值太小？

✅ 参考答案

若 K 很小时训练分数接近 1，但验证分数明显较低且波动大，说明模型过于贴合训练样本，K 可能太小。

17. KNN 遇到类别不平衡时会出现什么问题？如何缓解？

✅ 参考答案

大类样本更容易占据邻域，少数类会被淹没。可采用分层划分、重采样、距离加权、调整决策阈值/邻居规则，并使用 Recall、F1、PR-AUC 等指标调参。

18. 比较欧氏距离与曼哈顿距离

✅ 参考答案

欧氏距离是各维平方差之和再开方，强调较大的单维差异；曼哈顿距离是绝对差之和，对极端单维差异相对不那么敏感。具体选择应通过业务含义和交叉验证确定。

19. 为什么 KNN 预测通常比训练慢？

✅ 参考答案

训练阶段主要保存数据；预测每个新样本时需要计算它与大量训练样本的距离并搜索最近邻。可用 KD-Tree、Ball Tree、近似近邻或减少样本/维度加速。

20. 为什么不能根据测试集分数选择 K？

✅ 参考答案

这样会让测试集参与模型选择，使最终分数偏乐观。应在训练集内部用验证集或交叉验证选 K，最后只用测试集评估一次。

21. 对“年龄、年收入、点击次数”三个特征，预处理时应注意什么？

✅ 参考答案

三列量纲和分布差异很大，应检查缺失、异常和偏态，并在训练折内做标准化或稳健缩放。若收入极度偏态，可先做对数变换，再比较交叉验证效果。

22. 为什么 KNN 边界可以是非线性的？

✅ 参考答案

KNN 不假设固定函数形式，而是依据查询点附近的局部样本决定类别。不同区域的邻域组成不同，因此能形成弯曲甚至碎片化的决策边界。

23. 手写数字识别中，直接使用所有像素做 KNN 可能遇到哪些问题？

✅ 参考答案

像素维度高、距离计算慢，背景像素带来冗余，轻微平移也会产生较大距离。可做归一化、降维（如 PCA）、去除低方差像素、数据增强，或使用更适合图像的模型。

31. 手写一个最简单的 1-NN 分类预测函数

✅ 参考答案

import numpy as np

def predict_one_nn(X_train, y_train, x_new):
    distances = np.sqrt(((X_train - x_new) ** 2).sum(axis=1))
    nearest_index = np.argmin(distances)
    return np.asarray(y_train)[nearest_index]

prediction = predict_one_nn(X_train, y_train, X_test[0])
print(prediction)

32. 手写 KNN 多数投票函数

✅ 参考答案
Python
from collections import Counter
import numpy as np

def predict_knn(X_train, y_train, x_new, k=5):
    distances = np.linalg.norm(X_train - x_new, axis=1)
    neighbor_ids = np.argsort(distances)[:k]
    neighbor_labels = np.asarray(y_train)[neighbor_ids]
    return Counter(neighbor_labels).most_common(1)[0][0]

33. 用距离加权方式手写 KNN 回归

✅ 参考答案
Python
import numpy as np

def predict_weighted_knn_regression(X_train, y_train, x_new, k=5):
    distances = np.linalg.norm(X_train - x_new, axis=1)
    neighbor_ids = np.argsort(distances)[:k]
    d = distances[neighbor_ids]
    weights = 1.0 / (d + 1e-12)
    targets = np.asarray(y_train)[neighbor_ids]
    return np.average(targets, weights=weights)

加上很小的 1e-12 可避免新样本与训练样本重合时除以零。
