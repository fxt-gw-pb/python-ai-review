🔤 变量与数据类型

🏷️ 选择 + 代码阅读🏷️ 22 题📖 相关讲义：Python 基础语法与条件判断

本套练习覆盖变量命名、数据类型、输入输出与类型转换。选择题先在心里作答，代码阅读题先在纸上写出预测结果，再点开答案对照。

一、变量与数据类型

1. 合法的变量名

以下哪个变量名是合法的？

  • A. 2name
  • B. name@
  • C. _age
  • D. class
✅ 参考答案

C. _age

解析：变量名只能由字母、数字、下划线组成，且不能以数字开头，也不能使用 Python 关键字。A 以数字开头；B 含非法字符 @；D 的 class 是关键字；C 以下划线开头是合法的。

2. 变量赋值

执行以下代码后，输出结果是什么？

Python
a = 10
b = 20
a = b
print(a, b)
✅ 参考答案
Text
20 20

解析：a = b 是把 b 的值赋给 a，此后 a 和 b 都是 20。赋值只影响左边的变量，b 不变。

3. 蛇形命名法

以下哪种命名方式符合蛇形命名法（snake_case）？

  • A. studentName
  • B. StudentName
  • C. student_name
  • D. student-name
✅ 参考答案

C. student_name

解析：蛇形命名法用小写字母 + 下划线连接单词。A 是小驼峰、B 是大驼峰，D 中的连字符 - 在变量名中是非法字符（会被当成减号）。Python 变量推荐使用蛇形命名。

4. 链式赋值

执行以下代码后，输出结果是什么？

Python
x = y = z = "Python"
y = "Java"
print(x, y, z)
✅ 参考答案
Text
Python Java Python

解析：第一行让 x、y、z 都指向 "Python"；第二行只把 y 重新指向 "Java"，x 和 z 不受影响。

5. 找错误

以下代码有什么错误？

Python
first name = "张三"
print(first name)
✅ 参考答案

变量名不能包含空格，first name 是非法命名，运行会直接报 SyntaxError。

正确写法：

Python
first_name = "张三"
print(first_name)

6. type() 查看类型

执行以下代码后，输出结果是什么？

Python
a = 10
b = 3.14
c = "Python"
print(type(a), type(b), type(c))
✅ 参考答案
Text
<class 'int'> <class 'float'> <class 'str'>

解析：整数是 int，小数是 float，引号包起来的是字符串 str。

7. 布尔型变量

以下哪个变量的数据类型是布尔型？

  • A. var1 = "True"
  • B. var2 = True
  • C. var3 = 1
  • D. var4 = "False"
✅ 参考答案

B. var2 = True

解析：布尔型只有 True 和 False 两个值，且不带引号。A 和 D 带引号，是字符串；C 是整数 1。

8. 变量可以换类型

执行以下代码后，输出结果是什么？

Python
x = 10
print(type(x))
x = "Hello"
print(type(x))
✅ 参考答案
Text
<class 'int'>
<class 'str'>

解析：Python 是动态类型语言，变量的类型由它当前指向的值决定，同一个变量可以先存整数、再存字符串。

9. 浮点型变量

以下哪个变量的数据类型是浮点型？

  • A. a = 10
  • B. b = "3.14"
  • C. c = 3.14
  • D. d = True
✅ 参考答案

C. c = 3.14

解析：带小数点且不带引号的数值是 float。B 虽然长得像小数，但加了引号就是字符串。

10. 类型大点兵

执行以下代码后，输出结果是什么？

Python
age = 25
height = 1.75
is_student = False
name = "张三"
print(type(age), type(height), type(is_student), type(name))
✅ 参考答案
Text
<class 'int'> <class 'float'> <class 'bool'> <class 'str'>

解析：四种最常用的基本类型一次认全——整数、浮点数、布尔、字符串。

二、输出和输入

1. % 格式化输出

执行以下代码后，输出结果是什么？

Python
name = "李四"
age = 25
print('姓名:%s, 年龄:%d' % (name, age))
✅ 参考答案
Text
姓名:李四, 年龄:25

解析：%s 是字符串占位符，%d 是整数占位符，% 后面的元组按顺序填入。

2. 保留两位小数

以下哪个格式化字符串可以正确输出"身高: 1.75"（保留 2 位小数）？

  • A. '身高: %d' % 1.75
  • B. '身高: %.2f' % 1.75
  • C. '身高: %s' % 1.75
  • D. '身高: %2d' % 1.75
✅ 参考答案

B. '身高: %.2f' % 1.75

解析：%.2f 表示浮点数保留 2 位小数。A 和 D 的 %d 会把 1.75 截断成 1；C 的 %s 能输出 1.75 但无法控制小数位数。

3. f-string 补零

执行以下代码后，输出结果是什么？

Python
score = 95
print(f'分数: {score:05d}')
✅ 参考答案
Text
分数: 00095

解析：{score:05d} 表示按整数输出，总宽度 5 位，不足的位置用 0 补齐。

4. 两种格式化写法

使用格式化输出，如何得到下面的文本？

Python
product = "苹果"
price = 8.5
quantity = 3
# 需要输出: 商品:苹果, 单价:8.50, 数量:3
✅ 参考答案

两种主流写法都要会：

Python
# 写法1：f-string（推荐）
print(f'商品:{product}, 单价:{price:.2f}, 数量:{quantity}')

# 写法2：format() 方法
print('商品:{}, 单价:{:.2f}, 数量:{}'.format(product, price, quantity))

解析：8.5 要显示成 8.50，关键是 :.2f 保留两位小数。

5. 转义字符

执行以下代码后，输出结果是什么？

Python
print('Python\t编程')
print('Java\n语言')
✅ 参考答案
Text
Python	编程
Java
语言

解析：\t 是制表符（一段水平空白），\n 是换行符，所以 "Java" 和 "语言" 被拆成了两行。

6. 编程题：问候程序

编写代码实现：要求用户输入姓名和年龄，然后输出"你好，[姓名]！你今年[年龄]岁了。"，并确保年龄是整数类型。

✅ 参考答案
Python
name = input("请输入姓名：")
age = int(input("请输入年龄："))
print(f"你好，{name}！你今年{age}岁了。")

⚠️ 易错点：f-string 前面的 f 千万别漏。写成 print("你好，{name}！...") 时，{name} 不会被替换，会原样打印出来。

7. input 的返回类型

执行以下代码，如果用户输入 25，输出结果是什么？

Python
age = input("请输入您的年龄: ")
print(f"您的年龄是: {age}, 类型是: {type(age)}")
✅ 参考答案
Text
您的年龄是: 25, 类型是: <class 'str'>

解析：input() 的返回值永远是字符串，哪怕你输入的是数字。要做数学运算必须先转换类型。

8. 找问题并修正

以下代码有什么问题？如何修正？

Python
num1 = input("请输入第一个数字: ")
num2 = input("请输入第二个数字: ")
result = num1 + num2
print(f"两数之和为: {result}")
✅ 参考答案

问题：input() 返回的是字符串，num1 + num2 变成了字符串拼接。输入 3 和 5，会输出 35 而不是 8。

修正——把输入转换成数值类型再相加：

Python
num1 = int(input("请输入第一个数字: "))
num2 = int(input("请输入第二个数字: "))
result = num1 + num2
print(f"两数之和为: {result}")

三、数据类型转换

1. 四个转换函数

执行以下代码，输出结果是什么？

Python
print(int(3.14))
print(float("5.5"))
print(bool(""))
print(eval("3 + 2"))
✅ 参考答案
Text
3
5.5
False
5

解析：int() 转整数会直接砍掉小数部分（不是四舍五入）；float() 能把数字字符串转成小数；空字符串转布尔是 False；eval() 会把字符串当作 Python 表达式来计算。

2. 哪个转换会报错

以下哪个转换会报错？

Python
A. int("10")
B. float("3.14")
C. int("10.5")
D. bool(0)
✅ 参考答案

C. int("10.5")

解析：int() 只能转换"长得像整数"的字符串。"10.5" 带小数点，直接 int() 会抛出 ValueError。如果确实要转，得先 float("10.5") 再 int()。

3. eval 的智能转换

分析以下代码，如果用户输入 25，输出结果是什么？如果输入 25.5 呢？

Python
age = eval(input("请输入您的年龄: "))
print(f"类型: {type(age)}, 值: {age}")
✅ 参考答案
Text
输入 25   → 类型: <class 'int'>, 值: 25
输入 25.5 → 类型: <class 'float'>, 值: 25.5

解析：eval() 按表达式解析输入，25 被解析成整数，25.5 被解析成浮点数。它很灵活，但对不可信的输入有安全风险，实际项目慎用。

四、综合应用

1. 个人信息登记程序

编写一个程序，要求用户输入：姓名（字符串）、年龄（整数）、身高（浮点数）、是否在校学生（布尔值），然后输出格式化信息：

Text
姓名：[姓名]，年龄：[年龄]岁，身高：[身高]米，10年后年龄：[未来年龄]，学生状态：[是否学生]
✅ 参考答案
Python
name = input("请输入用户姓名:")
age = int(input("请输入用户年龄:"))
height = float(input("请输入身高:"))
is_student = bool(int(input("是否为在校学生(是:1 / 否:0):")))

future_age = age + 10
status = "在校学生" if is_student else "非在校学生"

print(f"姓名：{name}，年龄：{age}岁，身高：{height}米，"
      f"10年后年龄：{future_age}岁，学生状态：{status}")

解析：三个关键点——① 年龄要 int()、身高要 float()；② 布尔值不能直接 bool(input())（非空字符串都是 True），先转 int 再转 bool 才可靠；③ 未来年龄 = 年龄 + 10。
