🎬 Pandas 综合：电影 · GDP · 销售

🏷️ 三数据集🏷️ 23 题📖 相关讲义：Pandas 进阶：读写、清洗与分组透视

三份数据集、23 道综合题，覆盖条件筛选、分组过滤（filter）、组内最值（idxmax）、transform 组内对齐、多级分组与保存结果，是 Pandas 两讲的收官练习。配套讲义：《Pandas 进阶》。

原题面向中文字段数据编写，个别题目按实际数据做了等价改编（改编处已标注）：movie 数据集是英文片名，关键词题用 "Love" 代替"爱情"；uniqlo 数据集没有日期和单价字段，相关题改用城市维度与成本/实付单价，考察的技能不变。

三份数据预览：

movie.csv（IMDB 电影，1000 行 × 12 列）：

1 Guardians of the Galaxy Action,Adventure,Sci-Fi James Gunn 2014 121 8.1 333.13
2 Prometheus Adventure,Mystery,Sci-Fi Ridley Scott 2012 124 7.0 126.46
3 Split Horror,Thriller M. Night Shyamalan 2016 117 7.3 138.12

1960-2019全球GDP数据.csv（9930 行 × 3 列，注意用 gbk 编码读取）：

year country GDP
1960 美国 543300000000
1960 英国 73233967692
1960 法国 62225478000

uniqlo.csv（门店销售明细，22293 行 × 12 列）：

store_id city channel gender_group age_group product customer revenue order quant unit_cost

146 杭州 线下 Female 25-29 运动 1 149.0 1 1 49

Python
import pandas as pd

mv = pd.read_csv('./movie.csv')
gdp = pd.read_csv('./1960-2019全球GDP数据.csv', encoding='gbk')
uq = pd.read_csv('./uniqlo.csv')

一、movie 电影数据（13 题）

1. 筛选出评分大于 8 且时长超过 120 分钟的电影，按评分降序排序后保存为新的 CSV 文件

✅ 参考答案
Python
result = mv[(mv['Rating'] > 8) & (mv['Runtime (Minutes)'] > 120)] \
           .sort_values('Rating', ascending=False)
result.to_csv('./high_rating_long_movies.csv', index=False)

共 38 部，榜首是 The Dark Knight（9.0 分 / 152 分钟）。多条件筛选每个条件都要加小括号，用 & 连接。

2. 按电影类型分组，过滤出电影数量少于 50 部的类型，展示类型名称和电影数量

✅ 参考答案
Python
cnt = mv.groupby('Genre')['Title'].count()
cnt[cnt < 50]

Genre 是 "Action,Adventure,Sci-Fi" 这样的组合串，共 207 种组合，只有 "Action,Adventure,Sci-Fi"（恰好 50 部）不满足 <50，其余 206 种都入选。

进阶做法——把组合类型拆成单个类型再统计：

Python
mv_exploded = mv.assign(Genre=mv['Genre'].str.split(',')).explode('Genre')
single_cnt = mv_exploded.groupby('Genre')['Title'].count()
single_cnt[single_cnt < 50]

3. 找出每年票房最高的电影及其票房金额，将结果保存为新的 DataFrame

✅ 参考答案
Python
mv_rev = mv.dropna(subset=['Revenue (Millions)'])   # 票房有缺失值, 先剔除
top_by_year = mv_rev.loc[
    mv_rev.groupby('Year')['Revenue (Millions)'].idxmax(),
    ['Year', 'Title', 'Revenue (Millions)']
].sort_values('Year').reset_index(drop=True)

groupby().idxmax() 返回每组最大值所在的行索引，再用 loc 取整行——组内最值的标准套路。2006~2016 的年度冠军依次是：加勒比海盗2、蜘蛛侠3、蝙蝠侠：黑暗骑士、阿凡达（760.51）、玩具总动员3、哈利波特7下、复仇者联盟、饥饿游戏2、美国狙击手、星球大战7（936.63，全榜最高）、侠盗一号。

4. 计算每种电影类型的评分标准差，按标准差从大到小排序，取前 3 种类型

✅ 参考答案
Python
mv.groupby('Genre')['Rating'].std().dropna().sort_values(ascending=False).head(3)

前三：Action,Horror,Thriller（2.095）、Action（1.768）、Comedy,Fantasy,Horror（1.656）。只有 1 部电影的类型算不出标准差（NaN），用 dropna 去掉。

5. 筛选出评分在其类型平均分上下 0.5 分范围内的电影，展示电影名称和类型

✅ 参考答案
Python
mv2 = mv.copy()
mv2['genre_avg'] = mv2.groupby('Genre')['Rating'].transform('mean')
result = mv2[(mv2['Rating'] - mv2['genre_avg']).abs() <= 0.5][['Title', 'Genre']]

共 559 部。关键是 transform('mean')：它把每组的均值"广播"回原来的每一行，行数不变，从而能和原列直接比较——这是"和本组统计值对比"类题目的通用解法。

6. 把电影名称中包含关键词 "Love" 的电影提取出来，按票房升序排序，保存为新的 CSV 文件（改编：原题关键词为"爱情"，本数据集为英文片名）

✅ 参考答案
Python
love_movies = mv[mv['Title'].str.contains('Love', case=False)] \
                .sort_values('Revenue (Millions)')
love_movies.to_csv('./love_movies.csv', index=False)

共 14 部（Lovesong、Love, Rosie、The Love Witch……）。str.contains 做子串匹配，case=False 忽略大小写。

7. 计算每部电影的评分与同类型电影评分中位数的差值，筛选出差值绝对值大于 1 的电影

✅ 参考答案
Python
mv2 = mv.copy()
mv2['genre_median'] = mv2.groupby('Genre')['Rating'].transform('median')
mv2['diff'] = mv2['Rating'] - mv2['genre_median']
result = mv2[mv2['diff'].abs() > 1][['Title', 'Genre', 'Rating', 'genre_median']]

共 169 部"显著偏离本类型水平"的电影，例如银河护卫队（8.1，类型中位数 6.7）。和第 5 题同套路，把 mean 换成 median。

8. 按导演和电影类型分组，找出每个组内评分最高的电影名称、评分以及票房

✅ 参考答案
Python
result = mv.sort_values('Rating', ascending=False) \
           .groupby(['Director', 'Genre'], as_index=False) \
           .first()[['Director', 'Genre', 'Title', 'Rating', 'Revenue (Millions)']]

先整体按评分降序，再分组取 first()——每组的第一条就是组内最高分。共 950 个"导演×类型"组合。

9. 按电影类型和导演分组，计算每组的平均评分、最高票房和电影数量，筛选出电影数量超过 5 部且平均评分高于 7 分的组合

✅ 参考答案
Python
g = mv.groupby(['Genre', 'Director']).agg(
    avg_rating=('Rating', 'mean'),
    max_revenue=('Revenue (Millions)', 'max'),
    cnt=('Title', 'count')
)
g[(g['cnt'] > 5) & (g['avg_rating'] > 7)]

方法如上，但结果为空——本数据集里同一"类型×导演"组合最多只有 4 部电影。把条件放宽为超过 2 部，可得 4 个组合：Bryan Singer（动作冒险科幻）、David Yates（冒险家庭奇幻，最高票房 301.96）、Peter Jackson（冒险奇幻）、James Wan（恐怖悬疑惊悚）。

这是数据分析的常态：先保证方法正确，再核对结果是否合理——空结果不一定是代码错了，可能是条件对这份数据太苛刻。

10. 将每年的电影按评分降序排列，取每年评分最高的前 3 部，汇总计算这些电影的总票房

✅ 参考答案
Python
top3 = mv.sort_values('Rating', ascending=False).groupby('Year').head(3)
top3['Revenue (Millions)'].sum()    # 4156.91

11 年 × 每年 3 部 = 33 部，总票房 4156.91（百万美元）。groupby().head(n) 取每组前 n 行（配合预先排序即"组内 Top-N"）。

11. 按 Director 分组，筛选出平均评分高于 7 分的导演，列出名字和平均评分

✅ 参考答案
Python
avg = mv.groupby('Director')['Rating'].mean()
avg[avg > 7].sort_values(ascending=False)

共 231 位，前三：Nitesh Tiwari（8.8）、Christopher Nolan（8.68）、Olivier Nakache（8.6）。

12. 按 Director 分组，找出执导电影数量最多的前 5 位导演，并统计他们的电影总票房

✅ 参考答案
Python
top5 = mv['Director'].value_counts().head(5)
mv[mv['Director'].isin(top5.index)] \
    .groupby('Director')['Revenue (Millions)'].sum()

最高产：Ridley Scott 8 部（总票房 719.1），David Yates / M. Night Shyamalan / Paul W.S. Anderson / Michael Bay 各 6 部；其中 David Yates 总票房最高（1630.5）。isin() 用"名单"做筛选。

13. 按 Director 分组，统计每位导演拍摄不同类型电影的种类数量

✅ 参考答案
Python
mv.groupby('Director')['Genre'].nunique().sort_values(ascending=False)

nunique() 统计组内不同取值的个数。最"多面手"：Ridley Scott（7 种类型组合）、M. Night Shyamalan（6 种）、Paul W.S. Anderson（5 种）。

二、全球 GDP 数据（5 题）

14. 筛选出 2000 年之后（含 2000 年）的数据，计算各国家 GDP 的总和

✅ 参考答案
Python
gdp[gdp['year'] >= 2000].groupby('country')['GDP'].sum().sort_values(ascending=False)

210 个国家/地区，前三：美国（约 3.05×10¹⁴）、中国（约 1.31×10¹⁴）、日本（约 9.89×10¹³）。

15. 按国家分组，过滤出 GDP 数据标准差大于 1000000 的国家，显示国家名称和 GDP 标准差

✅ 参考答案
Python
std = gdp.groupby('country')['GDP'].std()
std[std > 1000000]

# 用 filter 的等价写法(保留整组明细数据):
gdp.groupby('country').filter(lambda g: g['GDP'].std() > 1000000)

GDP 以"美元"为单位动辄上亿，标准差大于一百万这个阈值太宽松——210 个国家全部满足。这题的价值在于练两种"组级过滤"写法：对聚合结果布尔筛选（返回统计值）vs filter（返回原始明细）。

16. 找出每年 GDP 最高的国家及其对应的 GDP 值，保存为新的 DataFrame

✅ 参考答案
Python
result = gdp.loc[gdp.groupby('year')['GDP'].idxmax(), ['year', 'country', 'GDP']] \
            .reset_index(drop=True)

60 行（1960~2019），每年的 GDP 冠军都是美国。

17. 计算每个国家 GDP 占全球 GDP 总和的比例，按年份分组，找出每年占比最高的前三个国家

✅ 参考答案
Python
gdp2 = gdp.copy()
gdp2['year_total'] = gdp2.groupby('year')['GDP'].transform('sum')
gdp2['share'] = gdp2['GDP'] / gdp2['year_total']

top3 = gdp2.sort_values('share', ascending=False) \
           .groupby('year').head(3) \
           .sort_values(['year', 'share'], ascending=[True, False])

transform 算出每年的全球总量再对齐相除。2019 年前三：美国 25.0%、中国 16.65%、日本 5.93%。

18. 把国家名称中包含"国"字的数据提取出来，按 GDP 降序排序，保存为新的 CSV 文件

✅ 参考答案
Python
result = gdp[gdp['country'].str.contains('国')].sort_values('GDP', ascending=False)
result.to_csv('./gdp_guo.csv', index=False, encoding='utf-8-sig')

命中 7 个国家（美国、中国、德国、英国、法国、韩国、泰国）共 410 行，最高一行是 2019 年美国（21.43 万亿美元）。保存含中文的 CSV 建议用 utf-8-sig 编码，Excel 打开不乱码。

三、uniqlo 销售数据（5 题）

改编说明：原题使用"价格/销量/月份"字段，本数据集没有日期列和商品单价列，故用 unit_cost（成本单价）、quant（购买件数）、revenue（实付金额）和 city（城市）等实际字段等价改编。

19. 筛选出成本单价在 50~100 元之间且购买件数大于 2 的记录，按销售金额降序排序后保存为新的 CSV 文件

✅ 参考答案
Python
result = uq[(uq['unit_cost'] >= 50) & (uq['unit_cost'] <= 100) & (uq['quant'] > 2)] \
