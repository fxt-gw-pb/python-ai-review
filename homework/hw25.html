🌳 决策树与 CART

🏷️ 选择 + 简答 + 代码🏷️ 35 题📖 相关讲义：决策树：ID3、C4.5 与 CART

35 道练习，覆盖 ID3、C4.5、CART、信息熵、信息增益、增益率、基尼指数、回归树、剪枝和特征重要性。

一、选择题（12 题）

1. 决策树内部节点通常表示什么？

A. 对某个特征的判断
B. 最终预测值
C. 数据文件路径
D. 随机种子

✅ 参考答案

答案：A。 内部节点执行特征切分，分支对应切分结果，叶节点给出类别、概率或回归值。

2. 信息熵越小通常说明什么？

A. 节点内类别越纯
B. 类别越混乱
C. 样本越多
D. 特征越多

✅ 参考答案

答案：A。 当节点内样本几乎属于同一类别时不确定性低，熵接近 0。

3. ID3 选择划分特征的主要依据是什么？

A. 信息增益
B. 欧氏距离
C. 均方误差固定为 0
D. 学习率

✅ 参考答案

答案：A。 ID3 倾向选择划分后熵下降最多，即信息增益最大的特征。

4. C4.5 使用增益率主要想缓解什么问题？

A. 信息增益偏好多取值特征
B. KNN 预测太慢
C. 线性回归共线性
D. 标签无法读取

✅ 参考答案

答案：A。 像“用户编号”这类取值很多的特征可获得很高信息增益，增益率用切分信息进行归一化。

5. CART 分类树通常使用什么指标选择切分？

A. 基尼不纯度
B. R²
C. 余弦相似度
D. AUC

✅ 参考答案

答案：A。 sklearn 分类树常用 Gini，也支持 entropy/log_loss。

6. 二分类节点中正负样本各占一半时，基尼不纯度是多少？

A. 0
B. 0.25
C. 0.5
D. 1

✅ 参考答案

答案：C。 Gini=1-(0.5²+0.5²)=0.5，这是二分类最不纯的情况。

7. 分类树叶节点的 predict_proba 通常如何得到？

A. 叶节点中各类别样本占比
B. 全部固定为 0.5
C. 距离倒数
D. 回归系数

✅ 参考答案

答案：A。 经过样本权重修正后，叶节点训练样本的类别分布形成预测概率。

8. 决策树通常是否必须标准化数值特征？

A. 通常不需要
B. 必须，否则无法训练
C. 只需标准化标签
D. 只有叶节点需要

✅ 参考答案

答案：A。 树比较单个特征与阈值，单调尺度变换通常不改变样本顺序和切分，但缺失值、异常与编码仍需处理。

9. max_depth 太大最可能导致什么？

A. 过拟合
B. 必然欠拟合
C. 样本自动平衡
D. 特征消失

✅ 参考答案

答案：A。 深树可形成非常细碎的叶节点，记忆训练噪声，方差较高。

10. min_samples_leaf 增大通常会怎样？

A. 限制小叶节点，使模型更平滑
B. 树一定更深
C. 训练样本变多
D. 信息熵恒为 1

✅ 参考答案

答案：A。 每个叶子必须包含更多样本，能抑制只服务少数样本的切分。

11. CART 回归树叶节点通常输出什么？

A. 叶节点目标值均值
B. 多数类别名称
C. Sigmoid 值
D. 最近邻标签

✅ 参考答案

答案：A。 在平方误差准则下，均值是使叶节点内平方误差最小的常数预测。

12. 成本复杂度剪枝参数 ccp_alpha 增大通常会怎样？

A. 树被剪得更简单
B. 树无限生长
C. 特征自动缩放
D. 测试集参与训练

✅ 参考答案

答案：A。 更大的 alpha 对叶节点数量惩罚更强，通常得到更少节点、更浅的子树。

二、简答题（11 题）

13. 解释信息增益的直观含义

✅ 参考答案

信息增益等于划分前的不确定性减去划分后的加权不确定性。值越大，说明该特征把不同类别分得越开。

14. 为什么“用户唯一编号”容易被 ID3 错选？

✅ 参考答案

编号几乎可把每个样本单独分开，使子节点纯度极高、信息增益很大，但它没有可泛化规律。应删除标识字段、用增益率，或通过验证和剪枝限制。

15. 比较预剪枝和后剪枝

✅ 参考答案

预剪枝在生长过程中通过深度、最小样本数、最小增益等条件提前停止，计算省但可能过早；后剪枝先长出较大树再根据验证误差或复杂度代价删除分支，通常更稳但开销更大。

16. 为什么单棵决策树属于高方差模型？

✅ 参考答案

树的切分是层层贪心的，训练数据轻微变化可能改变靠前切分并引起整棵树结构变化。限制复杂度或使用随机森林等集成方法可降低方差。

17. 决策树如何处理非线性关系和特征交互？

✅ 参考答案

树用多次轴对齐切分组合出分段区域；某特征是否继续切分可依赖上层另一个特征的取值，因此自然表达条件式非线性和交互。

18. 为什么基于不纯度的特征重要性可能有偏？

✅ 参考答案

连续或取值多的特征拥有更多候选切分，更容易获得较大不纯度下降；相关特征还会分摊或互相替代重要性。可辅以置换重要性、SHAP 和业务验证。

19. 决策树遇到类别型特征时需要注意什么？

✅ 参考答案

sklearn 树要求数值输入，通常需 One-Hot 或有依据的序数编码。把无序类别随意编码成整数会引入虚假大小关系；高基数类别还会造成维度或过拟合问题。

20. CART 分类树与回归树的主要差别是什么？

✅ 参考答案

分类树用类别不纯度选择切分，叶节点输出类别概率/类别；回归树用 MSE、绝对误差等减少目标离散度，叶节点输出连续值。二者都使用二叉递归切分。

21. 为什么树模型不应只看训练准确率？

✅ 参考答案

未剪枝树很容易把训练集分到极纯甚至完全记忆。应通过交叉验证选择深度、叶节点大小或剪枝参数，并在独立测试集检查泛化。

22. 如何判断一条树规则是否可信？

✅ 参考答案

除纯度外还要看覆盖样本量、交叉验证稳定性、不同时间/群体表现、是否使用泄漏特征以及业务逻辑。只覆盖一两个样本的高纯规则通常不可靠。

23. 缺失值处理为什么必须放进 Pipeline？

✅ 参考答案

插补统计量应只由训练折计算。放进 Pipeline 可避免验证/测试信息泄漏，也能保证训练、评估和上线使用完全一致的处理规则。

三、代码题（12 题）

24. 用 Python 计算二分类节点的信息熵

✅ 参考答案
Python
import math

def binary_entropy(positive, negative):
    total = positive + negative
    entropy = 0.0
    for count in (positive, negative):
        if count:
            p = count / total
            entropy -= p * math.log2(p)
    return entropy

print(binary_entropy(5, 5))  # 1.0

25. 手工计算一个二分类节点的基尼不纯度

✅ 参考答案
Python
def gini(counts):
    total = sum(counts)
    if total == 0:
        return 0.0
    return 1.0 - sum((count / total) ** 2 for count in counts)

print(gini([7, 3]))  # 0.42

32. 使用成本复杂度路径选择候选 ccp_alpha

✅ 参考答案
Python
from sklearn.model_selection import cross_val_score
from sklearn.tree import DecisionTreeClassifier

base = DecisionTreeClassifier(random_state=42)
path = base.cost_complexity_pruning_path(X_train, y_train)
scores = []
for alpha in path.ccp_alphas[:-1]:
    tree = DecisionTreeClassifier(ccp_alpha=alpha, random_state=42)
    score = cross_val_score(tree, X_train, y_train, cv=5).mean()
    scores.append((score, alpha))

best_score, best_alpha = max(scores)
print(best_alpha, best_score)

34. 为泰坦尼克号的数值/类别特征建立决策树 Pipeline

✅ 参考答案
Python
from sklearn.compose import ColumnTransformer
from sklearn.impute import SimpleImputer
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import OneHotEncoder
from sklearn.tree import DecisionTreeClassifier

preprocess = ColumnTransformer([
    ("num", SimpleImputer(strategy="median"), ["Age", "Fare", "Pclass"]),
    ("cat", Pipeline([
        ("imputer", SimpleImputer(strategy="most_frequent")),
        ("onehot", OneHotEncoder(handle_unknown="ignore")),
    ]), ["Sex", "Embarked"]),
])
model = Pipeline([
    ("preprocess", preprocess),
    ("tree", DecisionTreeClassifier(max_depth=5, random_state=42)),
])
model.fit(X_train, y_train)
