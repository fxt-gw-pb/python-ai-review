🧬 继承、封装与多态

🏷️ 简答 + 实操🏷️ 8 题📖 相关讲义：面向对象进阶与学生管理系统

这一套集中练习继承、方法重写、封装、多继承和多态。写代码时重点观察：哪些能力来自父类，哪些行为由子类重新定义。

一、继承与方法重写

1. 写出单继承和多继承的语法格式

✅ 参考答案
Python
# 单继承：一个子类继承一个父类
class Parent:
    pass


class Child(Parent):
    pass


# 多继承：一个子类同时继承多个父类
class ParentA:
    pass


class ParentB:
    pass


class MultiChild(ParentA, ParentB):
    pass

多继承存在同名方法时，Python 按 MRO（方法解析顺序）查找。可用 类名.__mro__ 或 类名.mro() 查看顺序。

2. 什么是方法重写？为什么需要方法重写？

✅ 参考答案

子类定义一个与父类同名的方法，就是方法重写。调用子类对象的方法时，会优先执行子类版本。

需要重写是因为子类虽然属于父类的一种，但某些行为具有自己的实现。例如所有动物都能发声，但狗和猫的发声方式不同。重写分为两种常见形式：

  • 完全重写：直接用子类实现替代父类实现。
  • 扩展性重写：先通过 super() 调用父类逻辑，再增加子类逻辑。

3. 单继承：Person 与 Student

创建 Person 基类，提供 say_hi()；创建继承它的 Student 类，并扩展 study() 方法。

✅ 参考答案
Python
class Person:
    def say_hi(self):
        print("大家好……")


class Student(Person):
    def study(self):
        print("我是学生，我会学习……")


student = Student()
student.say_hi()  # 继承自 Person
student.study()   # Student 新增的方法

Student 没有重写 say_hi()，因此直接复用父类实现。

4. 完全重写：Animal 与 Dog

创建 Animal 类，包含 name、age 属性和 speak() 方法；创建 Dog 子类，增加 color 属性和 jump() 方法，并完全重写 speak()。

✅ 参考答案
Python
class Animal:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def speak(self):
        print("动物正在发出声音")


class Dog(Animal):
    def __init__(self, name, age, color):
        super().__init__(name, age)
        self.color = color

    def speak(self):
        print(f"{self.name}：汪汪！")

    def jump(self):
        print(f"{self.color}的{self.name}跳了起来")


dog = Dog("小白", 2, "白色")
dog.speak()
dog.jump()

子类初始化时用 super().__init__() 复用父类对 name、age 的赋值，再单独保存 color。

5. 扩展性重写：Hero 与 Fighter

创建 Hero 类，包含体力值、移动和攻击方法；创建 Fighter 子类，增加怒气值。每次攻击保留父类攻击效果并增加 50 点怒气；怒气达到 100 时额外释放“致命一击”。

✅ 参考答案
Python
class Hero:
    def __init__(self, health):
        self.health = health

    def move(self):
        print("英雄向前移动")

    def attack(self):
        print("英雄发动普通攻击")


class Fighter(Hero):
    def __init__(self, health):
        super().__init__(health)
        self.anger = 0

    def attack(self):
        super().attack()
        self.anger += 50
        print(f"当前怒气：{self.anger}")
        if self.anger >= 100:
            print("致命一击！")
            self.anger -= 100


fighter = Fighter(1000)
fighter.attack()
fighter.attack()

super().attack() 保留父类普通攻击，这正是扩展性重写的关键。释放技能后扣除 100 怒气，使对象可以继续积攒下一次技能。

二、封装与多继承

1. 封装银行账户 BankAccount

设计 BankAccount 类，将投保人、累计保费、最低持有年限设为私有属性；提供查询投保人、查询与增加保费、查询与修改最低持有年限的方法。

✅ 参考答案
Python
class BankAccount:
    def __init__(self, holder, balance=0, min_years=1):
        self.__holder = holder
        self.__balance = balance
        self.__min_years = min_years

    def get_holder(self):
        return self.__holder

    def get_balance(self):
        return self.__balance

    def add_balance(self, amount):
        if amount <= 0:
            raise ValueError("增加的保费必须大于 0")
        self.__balance += amount

    def get_min_years(self):
        return self.__min_years

    def set_min_years(self, years):
        if not isinstance(years, int) or years <= 0:
            raise ValueError("最低持有年限必须是正整数")
        self.__min_years = years


account = BankAccount("小周", 10000, 3)
account.add_balance(2000)
account.set_min_years(5)
print(account.get_holder(), account.get_balance(), account.get_min_years())

双下划线属性不能按普通属性名从类外直接访问；对外方法还能顺便完成数据校验，这就是封装带来的好处。

2. 多继承：综合跑步、游泳和烹饪能力

创建跑步专家 Runner、游泳专家 Swimmer、烹饪专家 Cooker，再创建学徒类，同时继承三种能力并保存姓名、年龄。

✅ 参考答案
Python
class Runner:
    def run(self):
        print("百米跑步用时 9.5 秒")


class Swimmer:
    def swim(self):
        print("百米游泳用时 50 秒")


class Cooker:
    def cook(self):
        print("烹饪厨艺大成")


class Apprentice(Runner, Swimmer, Cooker):
    def __init__(self, name, age):
        self.name = name
        self.age = age


student = Apprentice("小林", 20)
student.run()
student.swim()
student.cook()
print(Apprentice.mro())

这里三个父类没有同名方法，因此调用关系很清楚。如果父类存在同名方法，应查看 MRO，避免只凭书写直觉判断。

三、多态

1. 用多态实现“汽车启动”

定义汽车父类、燃油车子类和电动车子类，分别实现 start()；再定义统一的 press() 函数，使传入任意一种汽车都能启动。

✅ 参考答案
Python
class Car:
    def start(self):
        raise NotImplementedError("子类必须实现 start 方法")


class FuelCar(Car):
    def start(self):
        print("点火，燃油发动机启动")


class ElectricCar(Car):
    def start(self):
        print("通电，电机启动")


def press(car):
    car.start()


cars = [FuelCar(), ElectricCar()]
for car in cars:
    press(car)

press() 只依赖对象具有 start() 方法，而不判断具体车型。同一调用作用于不同对象并表现出不同行为，就是多态。
