⌨️ 基础语法编程实操

🏷️ 编程实操🏷️ 9 题📖 相关讲义：Python 基础语法与条件判断

九道动手编程题，全部围绕「输入 → 类型转换 → 计算 → 格式化输出」这条主线。强烈建议每道题先自己敲完、跑通，再看参考代码。

一、课堂拓展实操

1. 个人信息介绍器

编写一个程序，要求用户依次输入姓名、年龄和爱好，然后使用格式化输出显示：

Text
我的名字是[姓名]，今年[年龄]岁，我的爱好是[爱好]。

要求：① 使用 input() 获取输入；② 使用 f-string 格式化输出；③ 变量命名符合规范、见名知意。（输出时不用带中括号）

✅ 参考答案
Python
name = input("请输入姓名：")
age = int(input("请输入年龄："))
hobby = input("请输入爱好：")

print(f"我的名字是{name}，今年{age}岁，我的爱好是{hobby}。")

运行示例：输入 小明、18、打篮球，输出：

Text
我的名字是小明，今年18岁，我的爱好是打篮球。

2. 简易计算器

编写一个程序，让用户输入两个数字，然后计算并显示：两数的和、差、乘积，以及第一个数除以第二个数的商（保留 2 位小数）。

要求：① 注意把输入的字符串转换为数值类型；② 使用格式化输出；③ 每个结果用不同的变量存储。

✅ 参考答案
Python
num1 = float(input("请输入第一个数字: "))
num2 = float(input("请输入第二个数字: "))

he = num1 + num2
cha = num1 - num2
ji = num1 * num2
shang = num1 / num2

print(f"两数之和: {he}")
print(f"两数之差: {cha}")
print(f"两数乘积: {ji}")
print(f"两数之商: {shang:.2f}")

⚠️ 易错点：除法前应保证第二个数不为 0；商保留两位小数用 :.2f。

3. 温度转换器

让用户输入一个摄氏温度值，将其转换为华氏温度并输出，结果保留 1 位小数。转换公式：华氏温度 = 摄氏温度 × 1.8 + 32。

Text
输出格式：[摄氏温度]摄氏度等于[华氏温度]华氏度
✅ 参考答案
Python
celsius = float(input("请输入摄氏温度: "))
fahrenheit = celsius * 1.8 + 32

print(f"{celsius}摄氏度等于{fahrenheit:.1f}华氏度")

运行示例：输入 37.5，输出 37.5摄氏度等于99.5华氏度。

4. 学生平均分计算

让用户输入三门课程的成绩（语文、数学、英语），计算并输出平均分（保留 2 位小数）。

Text
语文成绩：[成绩1]
数学成绩：[成绩2]
英语成绩：[成绩3]
三门课程的平均分为：[平均分]
✅ 参考答案
Python
chinese = float(input("请输入语文成绩: "))
math = float(input("请输入数学成绩: "))
english = float(input("请输入英语成绩: "))

average = (chinese + math + english) / 3

print(f"语文成绩：{chinese}")
print(f"数学成绩：{math}")
print(f"英语成绩：{english}")
print(f"三门课程的平均分为：{average:.2f}")

5. 长方形面积计算器

让用户输入长方形的长和宽，计算并输出周长和面积。周长 = 2 × (长 + 宽)，面积 = 长 × 宽。

Text
长方形的长为：[长]
长方形的宽为：[宽]
长方形的周长为：[周长]
长方形的面积为：[面积]
✅ 参考答案
Python
length = float(input("请输入长方形的长: "))
width = float(input("请输入长方形的宽: "))

perimeter = 2 * (length + width)
area = length * width

print(f"长方形的长为：{length}")
print(f"长方形的宽为：{width}")
print(f"长方形的周长为：{perimeter}")
print(f"长方形的面积为：{area}")

二、自主练习

1. 购物清单计算器

让用户输入商品名称、单价和购买数量，计算总金额并输出（总金额保留 2 位小数）：

Text
您购买了[商品名称]，单价[单价]元，数量[数量]件，总金额为[总金额]元。
✅ 参考答案
Python
product_name = input("请输入商品名称: ")
price = float(input("请输入单价: "))
quantity = int(input("请输入购买数量: "))

total_amount = price * quantity

print(f"您购买了{product_name}，单价{price}元，数量{quantity}件，"
      f"总金额为{total_amount:.2f}元。")

解析：单价可能带小数用 float，数量是整件数用 int——类型的选择要贴合业务含义。

2. 简单储蓄计算器

让用户输入每月储蓄金额和储蓄月数，计算总储蓄金额并按下面三行输出（总金额保留 2 位小数）：

Text
每月储蓄金额：[金额]元
储蓄月数：[月数]个月
总储蓄金额：[总金额]元
✅ 参考答案
Python
monthly_savings = float(input("请输入每月储蓄金额: "))
months = int(input("请输入储蓄月数: "))

total_savings = monthly_savings * months

print(f"每月储蓄金额：{monthly_savings}元")
print(f"储蓄月数：{months}个月")
print(f"总储蓄金额：{total_savings:.2f}元")

3. 旅行时间计算器

让用户输入出发时间和到达时间（24 小时制的小时数，如 8 表示 8:00），计算并输出旅行所用小时数：

Text
出发时间：[出发时间]:00
到达时间：[到达时间]:00
旅行用时：[用时]小时
✅ 参考答案
Python
departure = int(input("请输入出发时间(小时): "))
arrival = int(input("请输入到达时间(小时): "))

travel_hours = arrival - departure

print(f"出发时间：{departure}:00")
print(f"到达时间：{arrival}:00")
print(f"旅行用时：{travel_hours}小时")

💡 思考题：如果是晚上 22 点出发、第二天早上 6 点到达呢？可以用 (arrival - departure) % 24 来处理跨天的情况。

4. 水果折扣计算器

让用户输入水果名称、原价和折扣率（如 0.9 表示 9 折），计算折后价格并输出（价格保留 1 位小数）：

Text
[水果名称]原价[原价]元，打[折扣率×10]折后价格为[折后价]元。
✅ 参考答案
Python
fruit_name = input("请输入水果名称: ")
original_price = float(input("请输入原价: "))
discount_rate = float(input("请输入折扣率(如0.9表示9折): "))

discounted_price = original_price * discount_rate

print(f"{fruit_name}原价{original_price}元，"
      f"打{discount_rate * 10:.0f}折后价格为{discounted_price:.1f}元。")

解析：0.9 × 10 = 9.0，用 :.0f 把折数显示成整数 9 更自然。
