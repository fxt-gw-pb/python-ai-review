🧠 神经网络基础综合训练

🏷️ 选择 + 简答 + 代码🏷️ 55 题📖 相关讲义：神经网络基础：结构、损失与优化

55 道练习，覆盖神经元与激活函数、初始化、网络参数、损失函数、反向传播、优化器、学习率调度、正则化、BatchNorm 和分类训练闭环。

一、选择题（30 题）

1. 一个标准人工神经元的核心计算顺序是什么？

A. 先保存模型，再读取 CSV
B. 先反向传播，再产生预测
C. 只计算输入均值
D. 加权求和与偏置，再经过激活函数

✅ 参考答案

答案：D。 神经元先计算 $z=\sum_i w_ix_i+b$，再计算激活值 $a=f(z)$。

2. 为什么多层全连接网络不能全部省略激活函数？

A. 没有激活函数就不能使用 CPU
B. 线性层不包含参数
C. 多个线性变换仍可合并为一个线性变换
D. 激活函数负责读取数据

✅ 参考答案

答案：C。 线性层叠加仍是线性映射，无法表示弯曲决策边界；非线性激活赋予网络更强表达能力。

3. 在全连接网络中，“全连接”通常表示什么？

A. 每个参数都等于 1
B. 当前层每个神经元连接上一层全部神经元
C. 只有输出层有连接
D. 同一层所有神经元互相循环连接

✅ 参考答案

答案：B。 全连接层的每个输出单元都接收全部输入特征；同层神经元通常不直接连接。

4. 批量输入形状为 [32, 20]，经过 nn.Linear(20, 128) 后形状是什么？

A. [20, 128]
B. [32, 128]
C. [128, 20]
D. [32]

✅ 参考答案

答案：B。 线性层只改变最后一个特征维，批量维 32 保持不变。

5. Sigmoid 导数的最大值是多少？

A. 1
B. 没有上限
C. 2
D. 0.25

✅ 参考答案

答案：D。 $\sigma'(x)=\sigma(x)(1-\sigma(x))$，在 $x=0$ 时取得最大值 0.25。

6. Sigmoid 作为深层隐藏层激活的主要风险是什么？

A. 饱和区导数接近 0，易产生梯度消失
B. 输出一定大于 10
C. 不可微
D. 只能处理整数

✅ 参考答案

答案：A。 输入绝对值较大时 Sigmoid 接近 0 或 1，梯度很小；输出还不是以 0 为中心。

7. Tanh 相比 Sigmoid 的一个优点是什么？

A. 输出以 0 为中心
B. 永不饱和
C. 导数恒为 1
D. 不需要浮点计算

✅ 参考答案

答案：A。 Tanh 值域为 (-1, 1)，以 0 为中心；但大绝对值输入仍会饱和。

8. 关于 ReLU 的导数，下列说法正确的是？

A. 所有位置恒为 0
B. 所有位置恒为 1
C. $x>0$ 时为 1，$x<0$ 时为 0，0 点由框架约定次梯度
D. 只在负半轴为 1

✅ 参考答案

答案：C。 “ReLU 梯度恒为 1”是错误简化；负半轴梯度为 0。

9. 死亡 ReLU 指什么？

A. Dropout 在一次训练中随机屏蔽神经元
B. 神经元长期处于负半轴，输出和梯度持续为 0
C. 模型被保存到磁盘
D. 权重使用 Xavier 初始化

✅ 参考答案

答案：B。 可尝试更合适的学习率、初始化，或使用 LeakyReLU/PReLU 等变体。

10. 对形状 [batch, classes] 的多分类 logits，Softmax 通常沿哪个维度计算？

A. 所有维同时展开
B. 批量维 dim=0
C. 类别维 dim=1 或 dim=-1
D. 不需要指定任何语义

✅ 参考答案

答案：C。 每个样本的各类别概率应加和为 1，因此沿类别轴归一化。

11. Softmax 温度 $\tau$ 变小到 0.5 时，分布通常如何变化？

A. 概率和不再为 1
B. 更均匀
C. 所有概率变成负数
D. 更尖锐，更偏向最大 logit

✅ 参考答案

答案：D。 使用 softmax(logits / τ)，较小温度会放大 logit 差异。

12. 使用 nn.CrossEntropyLoss 训练互斥多分类时，模型最后一层应直接返回什么？

A. 手动 Softmax 后的概率
B. raw logits
C. 类别字符串
D. 只有最大类别下标

✅ 参考答案

答案：B。 CrossEntropyLoss 内部融合了稳定的 log_softmax + NLLLoss，训练前不要手动 Softmax。

13. 二分类模型输出一个 logit 时，推荐使用哪个损失？

A. BCEWithLogitsLoss
B. 先 Sigmoid 再 CrossEntropyLoss
C. L1Loss
D. CosineAnnealingLR

✅ 参考答案

答案：A。 它把 Sigmoid 与 BCE 融合，数值稳定；目标应为同形状浮点张量。

14. 一个样本可同时属于“体育”“新闻”“视频”多个标签，应采用什么输出与损失？

A. 单个回归值 + MSE
B. 单个 Softmax + CrossEntropyLoss
C. 每类独立 logit + BCEWithLogitsLoss
D. K-Means

✅ 参考答案

答案：C。 多标签类别互不排斥，不能用一个 Softmax 强制总概率为 1。

15. 为什么隐藏层权重不能全部初始化为 0？

A. 偏置必须等于 1
B. 0 不能存入 Tensor
C. 会自动变成字符串
D. 神经元保持对称，得到相同输出和梯度，无法分工

✅ 参考答案

答案：D。 零权重不一定让所有输出永久为 0，但会使同层神经元无法打破对称性。偏置初始化为 0 通常没有这个问题。

16. Xavier 初始化通常更适合哪类激活？

A. 只适合 Softmax
B. Tanh/Sigmoid 等近似对称激活
C. 只适合无参数层
D. 只适合整数网络

✅ 参考答案

答案：B。 Xavier 同时考虑 fan_in 和 fan_out，目标是稳定前向和反向方差。

17. ReLU 隐藏层优先考虑哪种初始化？

A. Kaiming/He
B. 所有权重恒为 1
C. 所有权重恒为 0
D. 只初始化偏置

✅ 参考答案

答案：A。 Kaiming 针对 ReLU 截断负半轴的特性补偿方差，常用方差约为 $2/fan_{in}$。

18. nn.Linear(3, 5) 的 weight 形状是什么？

A. [5, 3]
B. [3, 5]
C. [8]
D. [1, 3, 5]

✅ 参考答案

答案：A。 PyTorch 线性层计算 $Y=XW^T+b$，权重按 [out_features, in_features] 存储。

19. nn.Linear(20, 128) 含偏置时共有多少个参数？

A. 20
B. 2560
C. 148
D. 2688

✅ 参考答案

答案：D。 参数量是 $20\times128+128=(20+1)\times128=2688$。

20. 多分类硬标签传给 CrossEntropyLoss 时，推荐的类型和形状是什么？

A. 字符串列表
B. long 类型、形状 [N] 的类别下标
C. bool 类型、形状 [N, C, 1]
D. 必须是 float64 标量

✅ 参考答案

答案：B。 硬标签用类别索引最直接。软标签则可使用与 logits 同形状的浮点概率分布。

21. 标签平滑的主要作用是什么？

A. 降低模型过度自信，改善泛化的可能性
B. 自动纠正所有错误标签
C. 把分类变成回归
D. 删除小类别

✅ 参考答案

答案：A。 它把尖锐 one-hot 标签分配少量概率给其他类别，但不能代替数据清洗。

22. 与 MAE 相比，MSE 的典型特点是什么？

A. 梯度始终为 ±1
B. 对大误差惩罚更强，也更易受离群值影响
C. 在 0 点不可导
D. 只能用于分类

✅ 参考答案

答案：B。 MSE 平滑，梯度随残差增大；MAE 对离群值更稳健，但 0 点不可导且近最优时梯度幅度不自然减小。

23. Smooth L1 / Huber 损失如何结合 L1 与 L2？

A. 小误差区二次，远离 0 后线性
B. 小误差区线性，大误差区指数
C. 全部区间恒定
D. 只对权重计算

✅ 参考答案

答案：A。 它在原点附近保持平滑，远处减弱离群值的影响。

24. 1000 个样本、batch size 128、drop_last=False 时，每个 epoch 有多少步？

A. 128
B. 7
C. 8
D. 1000

✅ 参考答案

答案：C。 $\lceil1000/128\rceil=8$，最后一个 batch 含剩余样本。

25. 反向传播对一条计算链求导依赖什么法则？

A. 大数定律
B. K-Means 质心更新
C. 全概率公式
D. 链式法则

✅ 参考答案

答案：D。 一条路径上的局部导数相乘；同一参数通过多条路径影响损失时，各路径贡献相加。

26. 关于 Adam、Momentum 等优化器，下列说法正确的是？

A. 可改善训练轨迹，但不保证非凸网络达到全局最优
B. Adam 一定得到全局最优
C. Momentum 一定消除所有鞍点
D. 只要换优化器就不需要调学习率

✅ 参考答案

答案：A。 优化器是搜索策略，不是全局最优证明；模型、数据、学习率和正则化仍然重要。

27. Momentum 的主要思想是什么？

A. 把损失变成概率
B. 永久冻结全部参数
C. 累积历史梯度方向，减少震荡并保留惯性
D. 每步随机删除权重

✅ 参考答案

答案：C。 经典形式维护速度 $v_t=\mu v_{t-1}+g_t$，再按速度更新参数。

28. AdaGrad 的典型缺点是什么？

A. 不支持任何参数
B. 平方梯度累积单调增大，学习率可能过早衰减
C. 只能处理一个样本
D. 不使用梯度

✅ 参考答案

答案：B。 RMSProp 用平方梯度的 EMA 替代无限累加，缓解了这一问题。

29. Adam 为什么要做一阶矩和二阶矩的偏差校正？

A. 为了把标签转成长整型
B. 为了减少类别数量
C. 为了关闭 Dropout
D. 状态从 0 初始化，训练早期的移动平均偏向 0

✅ 参考答案

答案：D。 使用 $1-\beta_1^t$、$1-\beta_2^t$ 修正早期低估。

30. 标准模型推理最完整的组合是什么？

A. 只调用 model.train()
B. 只调用 optimizer.zero_grad()
C. model.eval() + torch.inference_mode()
D. 只调用 scheduler.step()

✅ 参考答案

答案：C。 eval() 切换 Dropout/BatchNorm 行为；inference_mode() 关闭训练期图跟踪。两者职责不同，不能互相替代。

二、简答题（15 题）

31. 人工神经元的权重、偏置和激活函数分别起什么作用？

✅ 参考答案

权重衡量各输入对当前神经元的影响并决定线性组合方向；偏置平移触发阈值，使模型不必强制经过原点；激活函数注入非线性，让多层网络不能被合并成一个线性变换。

32. 如何为隐藏层选择 Sigmoid、Tanh、ReLU 或 LeakyReLU？

✅ 参考答案

普通全连接/卷积隐藏层可先尝试 ReLU；若出现大量死亡神经元可尝试 LeakyReLU/PReLU。Tanh 适合希望状态有界且正负对称的情形。Sigmoid 主要用于二分类/多标签输出，深层隐藏层中因饱和和非零中心通常不是首选。最终选择需结合梯度和验证表现。

33. 分别说明二分类、多分类、多标签和回归的输出—损失配对

✅ 参考答案

二分类：1 个 raw logit + BCEWithLogitsLoss；互斥多分类：C 个 raw logits + CrossEntropyLoss；多标签：每类独立 raw logit + BCEWithLogitsLoss；回归：连续输出 + MSE/MAE/SmoothL1。概率变换放在推理解释阶段。

34. 为什么融合版交叉熵比“Softmax 后再取 log”更稳定？

✅ 参考答案

直接指数可能溢出，小概率再取 log 可能下溢。融合实现使用减最大值、log-sum-exp 等稳定技巧，并避免中间概率精度损失。CrossEntropyLoss 因此直接接收 logits。

35. Xavier 与 Kaiming 初始化背后的共同目标和差异是什么？

✅ 参考答案

共同目标是让前向激活与反向梯度的方差在层间保持稳定，避免消失或爆炸。Xavier 同时考虑 fan_in/fan_out，常配 Tanh/Sigmoid；Kaiming 针对 ReLU 截断负半轴，主要依据 fan_in 并补偿方差。

36. 怎样计算全连接网络参数量？state_dict 中又保存什么？

✅ 参考答案

每个 Linear(in,out) 参数量为 in*out + out，即权重加偏置，逐层相加。state_dict 保存模块的参数和持久 buffer，例如权重、偏置、BatchNorm 的运行均值/方差；它不保存 Python 类结构，因此加载前要先实例化同构模型。

37. 区分 L1/L2 预测损失与 L1/L2 权重正则化

✅ 参考答案

预测损失衡量预测与标签的残差，例如 MAE 和 MSE；权重正则化把参数绝对值或平方和加入总目标，约束模型复杂度。两者数学形式相似但作用对象和目的不同，不能混称。

38. 为什么隐藏层前方某个权重的梯度可能包含多条路径贡献？

✅ 参考答案

该权重影响一个隐藏激活，而这个激活可能连接多个后续神经元并共同影响总损失。链式法则对每条路径计算局部导数乘积；当计算图在该节点分叉时，各下游路径梯度在反向传播中相加。

39. 比较 Batch GD、单样本 SGD 与 mini-batch GD

✅ 参考答案

Batch GD 每步用全部样本，梯度稳定但单步成本高；单样本 SGD 噪声大、更新频繁；mini-batch 利用硬件并行并保留适度梯度噪声，是训练神经网络的常用折中。batch size 还会影响 BatchNorm 统计和学习率选择。

40. 用一句核心机制比较 Momentum、AdaGrad、RMSProp 与 Adam

✅ 参考答案

Momentum 平滑一阶梯度方向；AdaGrad 累积平方梯度，为不同参数缩放步长但可能过早衰减；RMSProp 对平方梯度做 EMA，避免累积无界增长；Adam 同时维护一阶、二阶矩并做偏差校正。

41. 学习率调度器与 Adam 的自适应步长是否重复？调用顺序是什么？

✅ 参考答案

不重复：Adam 按参数历史梯度调整相对有效步长，调度器改变全局基础学习率。按 epoch 调度时，通常先完成该 epoch 的所有 optimizer.step()，再调用 scheduler.step()；恢复训练要同时恢复优化器和调度器状态。

42. Dropout 在训练和推理时如何工作？为什么要缩放保留的激活？

✅ 参考答案

训练时每个激活以概率 p 置 0，保留值乘 1/(1-p)，使训练期输出期望与推理期大致一致；每次随机子网络减少共同适应。推理时 Dropout 关闭，不再随机屏蔽，也无需额外缩放。

43. BatchNorm 在训练和推理时分别使用什么统计量？

✅ 参考答案

训练时使用当前 mini-batch 的均值和方差进行标准化，并更新运行均值/方差，同时学习 gamma、beta。推理时使用训练期间积累的运行统计量，避免单样本或小批量统计不可靠；gamma、beta 仍参与变换。

44. model.train()、model.eval() 与 torch.no_grad() 分别控制什么？

✅ 参考答案

train() 让 Dropout、BatchNorm 等处于训练行为；eval() 切换为评估行为，但仍会记录计算图；no_grad() 关闭梯度图记录，却不会切换层模式。推理一般需要 eval() 与 no_grad()/inference_mode() 同时使用。

45. 手机价格分类案例中，为什么要先划分数据再拟合 StandardScaler？

✅ 参考答案

若在全数据上拟合标准化器，测试集的均值和方差信息会泄漏到训练过程，评估偏乐观。应只对训练特征 fit_transform，再用同一 scaler 对验证/测试特征 transform。超参数选择还应使用独立验证集，测试集只用于最终评估。

47. 实现数值稳定的带温度 Softmax

不直接调用 torch.softmax，实现沿最后一维的稳定 Softmax，并验证概率和为 1。

✅ 参考答案
Python
import torch


def stable_softmax(logits, temperature=1.0):
    if temperature <= 0:
        raise ValueError("temperature must be positive")
    scaled = logits / temperature
    shifted = scaled - scaled.max(dim=-1, keepdim=True).values
    exp_values = shifted.exp()
    return exp_values / exp_values.sum(dim=-1, keepdim=True)


logits = torch.tensor([[1000.0, 1001.0, 999.0]])
for temperature in (0.5, 1.0, 2.0):
    probabilities = stable_softmax(logits, temperature)
    print(temperature, probabilities, probabilities.sum(dim=-1))

import torch.nn as nn


class InitializedNet(nn.Module):
    def __init__(self):
        super().__init__()
        self.layer1 = nn.Linear(10, 32)
        self.layer2 = nn.Linear(32, 16)
        self.output = nn.Linear(16, 4)

        nn.init.xavier_uniform_(self.layer1.weight)
        nn.init.zeros_(self.layer1.bias)

        nn.init.zeros_(self.output.bias)

    def forward(self, x):
        x = torch.tanh(self.layer1(x))
        x = nn.functional.relu(self.layer2(x))
        return self.output(x)

51. 编写规范的多分类训练循环

给定 model、train_loader、device，使用交叉熵和 AdamW 训练 5 个 epoch，并按样本数加权统计损失。

✅ 参考答案
Python
import torch
import torch.nn as nn


def train_classifier(model, train_loader, device, epochs=5):
    model.to(device)
    criterion = nn.CrossEntropyLoss()
    optimizer = torch.optim.AdamW(
        model.parameters(),
        lr=1e-3,
        weight_decay=1e-4,
    )

    for epoch in range(epochs):
        model.train()
        loss_sum = 0.0
        sample_count = 0

        for features, labels in train_loader:
            features = features.to(device)
            labels = labels.to(device)

            optimizer.zero_grad(set_to_none=True)
            logits = model(features)
            loss = criterion(logits, labels)
            loss.backward()
            optimizer.step()

            batch_size = labels.size(0)
            loss_sum += loss.item() * batch_size
            sample_count += batch_size

        print(epoch + 1, loss_sum / sample_count)

52. 配置优化器与余弦学习率调度

✅ 参考答案
Python
import torch


def optimize_with_scheduler(model, train_loader, criterion, epochs=20):
    optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)
    scheduler = torch.optim.lr_scheduler.CosineAnnealingLR(
        optimizer,
        T_max=epochs,
        eta_min=1e-5,
    )

    for epoch in range(epochs):
        model.train()
        for features, labels in train_loader:
            optimizer.zero_grad(set_to_none=True)
            loss = criterion(model(features), labels)
            loss.backward()
            optimizer.step()

        scheduler.step()
        print(epoch + 1, scheduler.get_last_lr()[0])

54. 正确准备手机价格分类数据

读取含 price_range 的 CSV，分层划分训练/测试集，只在训练集上拟合标准化器，并构建 DataLoader。

✅ 参考答案
Python
import numpy as np
import pandas as pd
import torch
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from torch.utils.data import DataLoader, TensorDataset


def make_phone_loaders(csv_path, batch_size=32):
    frame = pd.read_csv(csv_path)
    x = frame.drop(columns="price_range").to_numpy(dtype=np.float32)
    y = frame["price_range"].to_numpy(dtype=np.int64)

    x_train, x_test, y_train, y_test = train_test_split(
        x,
        y,
        test_size=0.2,
        random_state=42,
        stratify=y,
    )

    scaler = StandardScaler()
    x_train = scaler.fit_transform(x_train).astype(np.float32)
    x_test = scaler.transform(x_test).astype(np.float32)

    train_set = TensorDataset(
        torch.from_numpy(x_train),
        torch.from_numpy(y_train),
    )
    test_set = TensorDataset(
        torch.from_numpy(x_test),
        torch.from_numpy(y_test),
    )

    return (
        DataLoader(train_set, batch_size=batch_size, shuffle=True),
        DataLoader(test_set, batch_size=256, shuffle=False),
        scaler,
    )

55. 实现分类模型的评估、保存与加载

编写准确率评估函数，并演示保存与恢复 state_dict。假设 model_factory() 返回同结构的新模型。

✅ 参考答案
Python
from pathlib import Path

import torch


def evaluate_accuracy(model, data_loader, device):
    model.eval()
    correct = 0
    total = 0

    with torch.inference_mode():
        for features, labels in data_loader:
            features = features.to(device)
            labels = labels.to(device)
            predictions = model(features).argmax(dim=1)
            correct += (predictions == labels).sum().item()
            total += labels.numel()

    return correct / total


def save_and_restore(model, model_factory, path, device):
    path = Path(path)
    path.parent.mkdir(parents=True, exist_ok=True)
    torch.save(model.state_dict(), path)

    restored = model_factory().to(device)
    state = torch.load(path, map_location=device, weights_only=True)
    restored.load_state_dict(state)
    restored.eval()
    return restored
