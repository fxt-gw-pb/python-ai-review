🎁 复制、闭包与装饰器

🏷️ 实操🏷️ 8 题📖 相关讲义：闭包与装饰器

从对象复制、函数对象和回调开始，逐步过渡到闭包与装饰器。做题时留意“变量保存的是值还是引用”以及“函数是否也能作为参数和返回值”。

一、浅拷贝与深拷贝

1. 电视机配置修改与重置

默认配置如下。创建一份当前配置，把对比度改为 80、亮度改为 85、音量改为 90；随后将当前配置重置为默认值，并保证默认配置本身不会被修改。

Python
default_config = {
    "contrast": 75,
    "brightness": 75,
    "volume": 30,
}
✅ 参考答案
Python
default_config = {
    "contrast": 75,
    "brightness": 75,
    "volume": 30,
}

# 创建独立字典，不能直接写 current_config = default_config
current_config = default_config.copy()
current_config.update({
    "contrast": 80,
    "brightness": 85,
    "volume": 90,
})
print("自定义配置：", current_config)
print("默认配置：", default_config)

# 重置时重新复制，不与默认字典共用同一个对象
current_config = default_config.copy()
print("重置后：", current_config)

这个字典只有一层，浅拷贝已经足够。如果值中还有列表、字典等可变嵌套对象，就要考虑深拷贝。

2. 创建互不影响的游戏存档备份

复制以下游戏状态，要求当前存档与备份完全独立。复制后在当前存档中新增城市“天津”并修改车辆价格，观察备份是否变化。

Python
current_game_state = {
    "name": "游遍天下大王君",
    "money": 20000000,
    "car": {"brand": "宝马", "price": 200000},
    "cities": ["上海", "北京", "南京"],
    "friends": ["刘阳", "马飞", "吴军"],
}
✅ 参考答案
Python
import copy

current_game_state = {
    "name": "游遍天下大王君",
    "money": 20000000,
    "car": {"brand": "宝马", "price": 200000},
    "cities": ["上海", "北京", "南京"],
    "friends": ["刘阳", "马飞", "吴军"],
}

backup_game_state = copy.deepcopy(current_game_state)

current_game_state["cities"].append("天津")
current_game_state["car"]["price"] = 180000

print("当前存档：", current_game_state)
print("备份存档：", backup_game_state)

必须使用 deepcopy()：浅拷贝只复制外层字典，内部的 car 字典和 cities 列表仍会与原对象共享。

二、函数对象与回调

1. 用函数对象实现简易计算器

补全 calculator()，让第三个参数接收一个运算函数。测试代码应分别输出 8 和 15。

Python
def calculator(a, b, operation):
    pass


def add(x, y):
    return x + y


def multiply(x, y):
    return x * y


print(calculator(3, 5, add))
print(calculator(3, 5, multiply))
✅ 参考答案
Python
def calculator(a, b, operation):
    return operation(a, b)


def add(x, y):
    return x + y


def multiply(x, y):
    return x * y


print(calculator(3, 5, add))       # 8
print(calculator(3, 5, multiply))  # 15

传参时写 add 而不是 add()：前者传递函数对象，后者会立即执行函数并传递它的返回值。

2. 用回调函数自定义下载进度和完成提示

改造下载函数：下载过程中调用外部传入的进度回调，显示 0%、20%……100%；结束后调用完成回调，显示“恭喜下载完成，正在为您自动安装……”。

✅ 参考答案
Python
import time


def download_file(url, on_progress, on_complete):
    print(f"开始下载：{url}")
    for progress in range(0, 101, 20):
        time.sleep(0.1)
        on_progress(progress)
    on_complete()


def show_progress(progress):
    print(f"拼命加载中……当前进度 {progress}%")


def install_after_download():
    print("恭喜下载完成，正在为您自动安装……")


download_file(
    "https://example.com/guide.pdf",
    show_progress,
    install_after_download,
)

下载函数负责“何时通知”，回调函数负责“通知时做什么”，从而让同一套下载流程可以自由替换界面提示。

三、闭包

1. 用闭包统计函数执行次数

实现 func_count()，使下面的调用依次显示“执行了 1 次”“执行了 2 次”“执行了 3 次”。

Python
f = func_count()
f()
f()
f()
✅ 参考答案
Python
def func_count():
    count = 0

    def say_hello():
        nonlocal count
        count += 1
        print("hello world")
        print(f"执行了 {count} 次")

    return say_hello


f = func_count()
f()
f()
f()

外层函数执行结束后，内层函数仍引用着 count。由于需要修改外层局部变量，必须声明 nonlocal count。

2. 用闭包管理银行余额

创建一个闭包保存余额，支持存钱、取钱和查询；取款金额不能超过当前余额。

✅ 参考答案
Python
def create_account(initial_balance=0):
    balance = initial_balance

    def operate(action, amount=0):
        nonlocal balance

        if action == "存钱":
            if amount <= 0:
                raise ValueError("存款金额必须大于 0")
            balance += amount
        elif action == "取钱":
            if amount <= 0:
                raise ValueError("取款金额必须大于 0")
            if amount > balance:
                print("余额不足")
                return balance
            balance -= amount
        elif action != "查询":
            raise ValueError("不支持的操作")

        print(f"当前余额：{balance:.2f} 元")
        return balance

    return operate


account = create_account(1000)
account("存钱", 500)
account("取钱", 300)
account("查询")

余额保存在闭包环境中，外部只能通过 operate() 规定的操作修改它。

四、装饰器

1. 用装饰器统计函数执行时间

定义一个函数，循环输出 100000 次 YYDS；在不修改该函数主体的前提下，用装饰器统计执行时间。

✅ 参考答案
Python
import time
from functools import wraps


def timer(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        start = time.perf_counter()
        result = func(*args, **kwargs)
        elapsed = time.perf_counter() - start
        print(f"{func.__name__} 执行耗时：{elapsed:.4f} 秒")
        return result

    return wrapper


@timer
def print_yyds(count=100000):
    for _ in range(count):
        print("YYDS")


# 正式测试可调用 print_yyds()；练习时可先传较小数字避免刷屏
print_yyds(10)

计时使用 time.perf_counter() 更适合测量短时间间隔；@wraps 用来保留原函数的名称和文档信息。

2. 用装饰器为返回值添加 .txt 后缀

定义一个返回字符串的函数，再用装饰器为该字符串追加 .txt。如果原字符串已经以 .txt 结尾，则不要重复添加。

✅ 参考答案
Python
from functools import wraps


def add_txt_suffix(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        result = func(*args, **kwargs)
        if not isinstance(result, str):
            raise TypeError("被装饰函数必须返回字符串")
        return result if result.endswith(".txt") else result + ".txt"

    return wrapper


@add_txt_suffix
def make_filename(name):
    return name


print(make_filename("学习笔记"))      # 学习笔记.txt
print(make_filename("复习清单.txt"))  # 复习清单.txt

装饰器既可以在调用函数前增加行为，也可以处理函数的返回值。
