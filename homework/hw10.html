🧱 面向对象基础

🏷️ 简答 + 实操🏷️ 9 题📖 相关讲义：面向对象基础与继承

这套练习覆盖面向对象与面向过程、类和对象、封装，以及 __init__、__str__ 等常用魔法方法。先独立完成类设计，再展开参考答案对照。

一、概念理解

1. 面向对象和面向过程有什么区别？请结合生活案例说明

✅ 参考答案
  • 面向过程关注“完成任务要依次执行哪些步骤”，通常把问题拆成一个个函数。例如自己做饭时，依次执行买菜、洗菜、切菜、炒菜。
  • 面向对象关注“由哪些对象协作完成任务”，对象同时保存数据和行为。例如餐厅场景中有顾客、服务员、厨师等对象，各对象负责自己的职责。

两者不是互相排斥的：小而线性的任务用面向过程更直接；对象多、状态复杂、需要扩展和复用的系统更适合面向对象。

2. 如何理解面向对象的封装、继承和多态？

✅ 参考答案
  • 封装：把属性和相关方法放进类中，并隐藏不希望外部直接操作的细节，只开放稳定接口。
  • 继承：子类复用父类已有的属性和方法，也可以增加或改写功能，减少重复代码。
  • 多态：不同类型的对象对同一方法调用给出各自实现。调用者只依赖统一接口，不必关心对象的具体类型。

例如 Dog 和 Cat 都继承 Animal，各自重写 speak()；遍历动物并调用 speak() 时，就会表现出多态。

3. 什么是类？什么是对象？二者有什么关系？

✅ 参考答案

类是对一类事物共同属性和行为的抽象，相当于创建对象的模板；对象是根据类创建出的具体实例。

例如 Student 类规定学生具有姓名以及学习行为，Student("小明") 和 Student("小红") 是两个彼此独立的对象。一个类可以创建多个对象，每个对象拥有自己的实例属性。

二、类与对象实操

1. 定义一个手机类：能开机、关机和拍照

✅ 参考答案
Python
class Phone:
    def __init__(self, brand):
        self.brand = brand
        self.is_on = False

    def power_on(self):
        self.is_on = True
        print(f"{self.brand} 手机已开机")

    def power_off(self):
        self.is_on = False
        print(f"{self.brand} 手机已关机")

    def take_photo(self):
        if self.is_on:
            print("咔嚓，照片拍摄成功")
        else:
            print("请先开机")


phone = Phone("星云")
phone.power_on()
phone.take_photo()
phone.power_off()

这里用 is_on 保存开关机状态，拍照前先检查状态，体现“对象属性影响对象行为”。

2. 定义一个电脑类：保存品牌和价格，并能用于编程、看视频

✅ 参考答案
Python
class Computer:
    def __init__(self, brand, price):
        self.brand = brand
        self.price = price

    def program(self, language):
        print(f"正在用 {self.brand} 电脑编写 {language} 程序")

    def watch_video(self, title):
        print(f"正在观看《{title}》")

    def __str__(self):
        return f"电脑品牌：{self.brand}，价格：{self.price:.2f} 元"


computer = Computer("未来", 6999)
print(computer)
computer.program("Python")
computer.watch_video("面向对象入门")

3. 定义算法工程师类，并用魔法方法初始化和展示信息

要求使用 __init__() 初始化岗位名称、薪资等属性；工作内容是每天摸鱼打代码；使用 __str__() 展示对象的全部信息。

✅ 参考答案
Python
class AlgorithmEngineer:
    def __init__(self, name, position, salary):
        self.name = name
        self.position = position
        self.salary = salary

    def work(self):
        print(f"{self.name} 今天继续摸鱼打代码")

    def __str__(self):
        return (
            f"姓名：{self.name}，岗位：{self.position}，"
            f"月薪：{self.salary:.2f} 元"
        )


engineer = AlgorithmEngineer("小林", "算法工程师", 18000)
print(engineer)
engineer.work()

__str__() 必须返回字符串；调用 print(对象) 时，Python 会自动使用这个返回值。

4. 设计 Book 类

创建一个 Book 类：必需参数为 title、author；price 默认为 0，is_available 默认为 True；实现 display_info() 显示图书信息。

✅ 参考答案
Python
class Book:
    def __init__(self, title, author, price=0, is_available=True):
        self.title = title
        self.author = author
        self.price = price
        self.is_available = is_available

    def display_info(self):
        status = "可借" if self.is_available else "已借出"
        print(
            f"书名：《{self.title}》｜作者：{self.author}｜"
            f"价格：{self.price:.2f} 元｜状态：{status}"
        )


book1 = Book("Python 入门", "小周", 59.9)
book2 = Book("算法手册", "小李", is_available=False)
book1.display_info()
book2.display_info()

调用时可以使用关键字参数跳过 price，直接给 is_available 传值。

三、综合类设计

1. 完成“烤地瓜”案例

设计一个地瓜类，记录烘烤时间、成熟状态和添加的调料；能够继续烘烤、添加调料并显示完整信息。

✅ 参考答案
Python
class SweetPotato:
    def __init__(self):
        self.cook_time = 0
        self.cook_state = "生的"
        self.condiments = []

    def cook(self, minutes):
        if minutes < 0:
            raise ValueError("烘烤时间不能为负数")
        self.cook_time += minutes

        if self.cook_time < 3:
            self.cook_state = "生的"
        elif self.cook_time < 5:
            self.cook_state = "半生不熟"
        elif self.cook_time < 8:
            self.cook_state = "熟了"
        else:
            self.cook_state = "烤焦了"

    def add_condiment(self, condiment):
        self.condiments.append(condiment)

    def __str__(self):
        flavors = "、".join(self.condiments) if self.condiments else "无"
        return (
            f"烘烤时间：{self.cook_time} 分钟，状态：{self.cook_state}，"
            f"调料：{flavors}"
        )


potato = SweetPotato()
potato.cook(4)
potato.add_condiment("蜂蜜")
potato.cook(2)
print(potato)

2. 植物生长跟踪系统 ⭐

实现一个植物类，保存生长天数、生长状态和养料列表。它能增加生长天数、添加养料，并按以下规则更新状态：0～6 天为种子，7～13 天为发芽，14～29 天为幼苗，30～59 天为开花，60 天及以上为结果。

✅ 参考答案
Python
class Plant:
    def __init__(self, name):
        self.name = name
        self.growth_days = 0
        self.growth_state = "种子"
        self.care_methods = []

    def grow(self, days):
        if days < 0:
            raise ValueError("生长天数不能为负数")

        self.growth_days += days
        if self.growth_days < 7:
            self.growth_state = "种子"
        elif self.growth_days < 14:
            self.growth_state = "发芽"
        elif self.growth_days < 30:
            self.growth_state = "幼苗"
        elif self.growth_days < 60:
            self.growth_state = "开花"
        else:
            self.growth_state = "结果"

    def add_care(self, care):
        self.care_methods.append(care)

    def __str__(self):
        cares = "、".join(self.care_methods) if self.care_methods else "暂无"
        return (
            f"植物：{self.name}｜生长：{self.growth_days} 天｜"
            f"状态：{self.growth_state}｜养料：{cares}"
        )


plant = Plant("番茄")
plant.add_care("浇水")
plant.add_care("有机肥")
plant.grow(35)
print(plant)

原题的区间端点存在重叠，这里统一改为左闭右开区间，例如第 7 天开始进入“发芽”状态。
