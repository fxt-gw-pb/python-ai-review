🧵 循环、字符串与列表

🏷️ 实操🏷️ 22 题📖 相关讲义：数据容器：列表、元组、字典与集合

这套练习是「量最大」的一套：for 循环热身、字符串六连、列表八连，最后三道综合题压轴。字符串和列表的方法非常多，做题是把它们记牢的最好方式。

一、for 循环

1. 遍历字符串

使用 for 循环遍历字符串，逐个打印每个字符。

Python
s = 'python is easy~'
✅ 参考答案
Python
s = 'python is easy~'
for ch in s:
    print(ch)

解析：字符串是可迭代对象，for 循环会逐字符取出。注意不要用 str 作变量名——它是内置类型名，会被覆盖。

2. 统计字符出现次数

统计字符串中 k 字符出现的次数。

Python
s = 'ok, i say ok! ok?'
✅ 参考答案
Python
s = 'ok, i say ok! ok?'
count = 0
for ch in s:
    if ch == 'k':
        count += 1
print(count)   # 3

解析：遍历 + 计数器是最基础的统计套路。也可以直接 s.count('k') 一行搞定。

3. 求 31–55 的数字和

✅ 参考答案
Python
total = 0
for i in range(31, 56):
    total += i
print(total)   # 1075

4. 求 100–200 之间能被 7 整除的和

✅ 参考答案
Python
total = 0
for i in range(100, 201):
    if i % 7 == 0:
        total += i
print(total)   # 2107

解析：100–200 之间 7 的倍数是 105, 112, …, 196 共 14 个，和为 2107。

5. 能被 7 整除的和，且跳过第 3 个被 7 整除的数

✅ 参考答案
Python
count = 0
total = 0
for i in range(100, 201):
    if i % 7 == 0:
        count += 1
        if count == 3:
            continue
        total += i
print(total)   # 1988

解析：第 3 个 7 的倍数是 119，用 continue 跳过它，2107 − 119 = 1988。

二、字符串

1. 切片：截取域名

把 "http://www.jd.com" 截取出 www.jd.com。

✅ 参考答案
Python
url = "http://www.jd.com"

# 方法1：切片（'http://' 占 7 个字符）
print(url[7:])            # www.jd.com

# 方法2：find 定位 // 再切片，更通用
idx = url.find('//')
print(url[idx + 2:])      # www.jd.com

2. 切片：提取后缀名

从 "music.avi" 中提取后缀名 avi。

✅ 参考答案
Python
filename = "music.avi"

# 方法1：rfind 找到最后一个点，再切片
idx = filename.rfind('.')
print(filename[idx + 1:])   # avi

# 方法2：split 分割取最后一段
print(filename.split('.')[-1])   # avi

解析：用 rfind（从右往左找）比 find 更稳妥，可以正确处理 my.music.avi 这种带多个点的文件名。

3. 用户邮箱处理

提取邮箱 "zhangsan123@gmail.com" 中 @ 前面的用户名部分。提示：用 find() 找到 @ 的位置，再用切片提取。

✅ 参考答案
Python
email = "zhangsan123@gmail.com"

idx = email.find('@')
username = email[:idx]
print(username)   # zhangsan123

解析：email[:idx] 表示从开头切到 @ 之前（不含 @）。

4. 文章关键词统计

Python
article = "人工智能正在改变世界。人工智能技术发展迅速，人工智能将带来新的革命。"

任务 1：统计"人工智能"在文章中出现的次数；任务 2：将所有"人工智能"替换为 "AI"。

✅ 参考答案
Python
article = "人工智能正在改变世界。人工智能技术发展迅速，人工智能将带来新的革命。"

# 任务1：count 统计
print(article.count("人工智能"))          # 3

# 任务2：replace 替换
new_article = article.replace("人工智能", "AI")
print(new_article)   # AI正在改变世界。AI技术发展迅速，AI将带来新的革命。

解析：字符串是不可变类型，replace 不会修改原字符串，而是返回一个新字符串，需要用变量接住。

5. 通讯录格式整理

Python
contacts = "张三-13800138000;李四-13900139000;王五-13700137000"

任务：分号分隔改逗号、横杠分隔改冒号，最终得到 张三:13800138000,李四:13900139000,王五:13700137000。

✅ 参考答案
Python
contacts = "张三-13800138000;李四-13900139000;王五-13700137000"

result = contacts.replace(';', ',').replace('-', ':')
print(result)
# 张三:13800138000,李四:13900139000,王五:13700137000

解析：replace 返回新字符串，所以可以像糖葫芦一样链式调用，一行完成两次替换。

6. 字符串向右旋转 n 位

将字符串向右旋转 n 位，例如 "abcdef" 向右旋转 2 位得到 "efabcd"。

✅ 参考答案
Python
s = "abcdef"
n = 2

result = s[-n:] + s[:-n]
print(result)   # efabcd

解析：向右旋转 n 位 = 把末尾 n 个字符搬到前面。s[-n:] 取末尾 n 个（ef），s[:-n] 取剩余部分（abcd），拼起来即可。

三、列表

1. 基础查询

Python
fruits = ['apple', 'banana', 'orange', 'grape', 'banana']

① 查找第一个 'banana' 的位置；② 统计 'banana' 出现的次数；③ 判断 'mango' 是否在列表中。

✅ 参考答案

print(fruits.count('banana'))     # 2

2. 学生名单管理

Python
students = ['张三', '李四', '王五']

① 在列表末尾添加 '赵六'；② 在 '李四' 后面插入 '钱七'；③ 批量添加 ['孙八', '周九']。

✅ 参考答案
Python
students = ['张三', '李四', '王五']

students.append('赵六')                              # 末尾追加
students.insert(students.index('李四') + 1, '钱七')  # 在李四后面插入
students.extend(['孙八', '周九'])                    # 批量追加

print(students)
# ['张三', '李四', '钱七', '王五', '赵六', '孙八', '周九']

解析：三个添加方法各司其职——append 加单个到末尾、insert 指定位置插入、extend 合并整个列表。注意 append(['孙八','周九']) 会把整个列表当成一个元素塞进去，这是常见错误。

scores = [85, 92, 78, 95, 88]

4. 购物车操作

Python
cart = ['牛奶', '面包', '鸡蛋', '水果', '饼干']

① 删除索引为 2 的商品；② 删除 '面包' 并返回被删除的商品名；③ 删除第一个 '水果'。

✅ 参考答案
Python
cart = ['牛奶', '面包', '鸡蛋', '水果', '饼干']

# 1- del 按索引删除
del cart[2]                              # 删掉 '鸡蛋'

# 2- pop 按索引删除并返回该元素
removed = cart.pop(cart.index('面包'))
print(f"被删除的商品: {removed}")          # 面包

# 3- remove 按值删除第一个匹配项
cart.remove('水果')

print(cart)    # ['牛奶', '饼干']

解析：三种删除方式对比——del 按下标删、pop 按下标删且有返回值、remove 按值删第一个匹配。

5. 游戏角色管理

Python
characters = ['战士', '法师', '射手']

① 在指定位置（如下标 1）添加 '刺客'；② 批量添加 ['辅助', '坦克']；③ 删除最后一个角色并显示。

8. 价格调整

Python
prices = [100, 200, 300, 400, 500]

① 将所有价格打 8 折；② 将列表反转；③ 排序后找出最贵的价格。

✅ 参考答案
Python
prices = [100, 200, 300, 400, 500]

# 1- 所有价格打 8 折（按下标逐个修改）
for i in range(len(prices)):
    prices[i] = prices[i] * 0.8
print(prices)          # [80.0, 160.0, 240.0, 320.0, 400.0]

# 2- 反转
prices.reverse()
print(prices)          # [400.0, 320.0, 240.0, 160.0, 80.0]

# 3- 排序后最后一个就是最贵的
prices.sort()
print(prices[-1])      # 400.0（也可以直接 max(prices)）

解析：修改列表元素必须通过下标 prices[i] = ...；for p in prices: p = p*0.8 改的只是临时变量，列表不会变。

四、综合编程

1. 数字累加器

让用户输入一个正整数 n，用 while 循环计算 1 到 n 之间所有整数的和与积，并格式化输出。

✅ 参考答案
Python
n = int(input("请输入一个正整数:"))

i = 1
total = 0    # 和的初始值：0（加 0 不影响结果）
product = 1  # 积的初始值：1（乘 1 不影响结果）

while i <= n:
    total += i
    product *= i
    i += 1

print(f"和:{total}")
print(f"积:{product}")

解析：核心在初始值的选择——累加从 0 开始、累乘从 1 开始（如果从 0 开始，乘出来永远是 0）。输入 5 时输出 和:15、积:120。

2. 学生成绩管理

Python
students = ['张三', '李四', '王五', '赵六']
scores = [85, 92, 78, 88]

① 用 for 循环计算平均分；② 找出高于平均分的学生名单；③ 添加新学生 '钱七'，成绩 95；④ 删除成绩最低的学生；⑤ 将所有成绩提高 5 分（最高不超过 100）。

✅ 参考答案
Python
students = ['张三', '李四', '王五', '赵六']
scores = [85, 92, 78, 88]

# 1- 计算平均分
total = 0
for s in scores:
    total += s
avg = total / len(scores)
print(f"平均分: {avg}")            # 85.75

# 2- 高于平均分的学生（两个列表按下标一一对应）
above = []
for i in range(len(scores)):
    if scores[i] > avg:
        above.append(students[i])
print(f"高于平均分: {above}")       # ['李四', '赵六']

# 3- 添加新学生（姓名、成绩两个列表都要加）
students.append('钱七')
scores.append(95)

# 4- 删除成绩最低的学生
min_idx = scores.index(min(scores))
students.pop(min_idx)
scores.pop(min_idx)

# 5- 所有成绩 +5，封顶 100
for i in range(len(scores)):
    scores[i] = min(scores[i] + 5, 100)

print(students)   # ['张三', '李四', '赵六', '钱七']
print(scores)     # [90, 97, 93, 100]

解析：两个平行列表必须同步增删，否则姓名和成绩就错位了。min(x + 5, 100) 是「封顶」的惯用写法。

3. 幸运数字（约瑟夫环）⭐ 挑战题

你穿越到了古代，赶上皇帝清点囚犯，有个好消息和坏消息——坏消息：你也在其中；好消息：你可以自选站位。规则：所有人从 1 开始报数，每次加 1，报到 3 的倍数的人被拖走，直到只剩最后一人获得赦免，他站的位置就是幸运数字。

要求：输入参与人数 n，输出幸运数字。（参考：10 个人时，幸运数字是 4）

✅ 参考答案
Python
# 1- 输入人数
n = int(input("请输入参与游戏的人数: "))

# 2- 生成位置列表 [1, 2, ..., n]
num_list = [i for i in range(1, n + 1)]

# 3- 循环报数，报到 3 的倍数就移除该位置的人
current_num = 0   # 当前报的数
i = 0             # 当前报数人的下标
while len(num_list) != 1:
    current_num += 1
    if current_num % 3 == 0:
        num_list.pop(i)
        i -= 1            # 删除后，后面的人补位到下标 i，回退一步抵消 i += 1
    i += 1
    if i >= len(num_list):
        i = 0             # 报到队尾就绕回队头

print(f"幸运数字: {num_list[0]}")   # n=10 时输出 4

解析：两个关键细节——① pop(i) 之后队伍缩短，下一个人正好顶到下标 i，所以要 i -= 1 抵消后面的 i += 1；② 下标越界时归零，实现"围成一圈"的效果。
