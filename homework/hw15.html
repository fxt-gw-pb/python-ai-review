2. 判断变量名是否合法

判断 123_python、python@2024、_holiday、class 是否能作为 Python 变量名，并说明原因。

✅ 参考答案
  • 123_python：不合法，变量名不能以数字开头。
  • python@2024：不合法，@ 不是变量名允许使用的字符。
  • _holiday：合法，可以以下划线开头。
  • class：不合法，它是 Python 关键字。

可使用 str.isidentifier() 检查命名形式，再用 keyword.iskeyword() 排除关键字：

Python
import keyword

names = ["123_python", "python@2024", "_holiday", "class"]
for name in names:
    valid = name.isidentifier() and not keyword.iskeyword(name)
    print(name, valid)

✅ 参考答案

def find_common_elements(list1, list2):
    # 保留 list1 中的出现顺序，同时去重
    return list(dict.fromkeys(item for item in list1 if item in list2))

class Book:
    def __init__(self, title, author, price=0, is_available=True):
        self.title = title
        self.author = author
        self.price = price
        self.is_available = is_available

    def display_info(self):
        status = "是" if self.is_available else "否"

        )

十八、面向对象三大特性

1. 用继承与多态实现动物发声

定义 Animal 父类；Dog 子类增加颜色和叼飞盘行为并重写发声；Cat 子类重写发声。定义统一函数 animal_speak(animal) 调用传入对象的 speak()。

✅ 参考答案
Python
class Animal:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def speak(self):
        print(f"{self.name} 发出叫声……")


class Dog(Animal):
    def __init__(self, name, age, color):
        super().__init__(name, age)
        self.color = color

    def speak(self):
        print(f"{self.name}（{self.color}）：汪汪汪！")

    def fetch(self):
        print(f"{self.name} 正在叼飞盘……")


class Cat(Animal):
    def speak(self):
        print(f"{self.name}：喵喵喵！")


def animal_speak(animal):
    animal.speak()


my_dog = Dog("旺财", 2, "黄色")
my_cat = Cat("咪咪", 1)
animal_speak(my_dog)
animal_speak(my_cat)
my_dog.fetch()

Animal 提供统一接口，子类重写 speak()；animal_speak() 不判断具体类型，体现多态。
