📨 朴素贝叶斯与文本分类

🏷️ 选择 + 简答 + 代码🏷️ 30 题📖 相关讲义：朴素贝叶斯与文本情感分析

30 道练习，覆盖条件概率、贝叶斯公式、条件独立假设、拉普拉斯平滑、三类常用朴素贝叶斯模型、词袋表示和中文情感分类。

一、选择题（10 题）

1. 条件概率 P(A|B) 表示什么？

A. 在 B 已发生条件下 A 发生的概率
B. A 与 B 都不发生的概率
C. A 的样本数量
D. A 与 B 的距离

✅ 参考答案

答案：A。 P(A|B)=P(A∩B)/P(B)，前提是 P(B)>0。

2. 贝叶斯公式的主要用途是什么？

A. 用先验和似然更新后验概率
B. 计算欧氏距离
C. 生成聚类中心
D. 求回归斜率

✅ 参考答案

答案：A。 观察证据后，贝叶斯公式把原先的类别信念更新为后验概率。

3. 朴素贝叶斯“朴素”在哪里？

A. 假设给定类别后各特征条件独立
B. 不需要训练数据
C. 所有概率都相等
D. 只支持一个特征

✅ 参考答案

答案：A。 该假设通常不完全真实，但能大幅简化联合概率估计，文本分类中常很有效。

4. 文本词频特征最常搭配哪种朴素贝叶斯？

A. MultinomialNB
B. KMeans
C. LinearRegression
D. PCA

✅ 参考答案

答案：A。 多项式朴素贝叶斯适合非负计数或 TF-IDF 特征。

5. 每个特征近似服从类条件高斯分布时，应优先尝试什么？

A. GaussianNB
B. MultinomialNB
C. KNN 回归
D. Apriori

✅ 参考答案

答案：A。 GaussianNB 对每个类别、每个连续特征估计均值和方差。

6. 0/1 的词是否出现特征最适合哪种模型？

A. BernoulliNB
B. Ridge
C. K-Means
D. 决策树回归

✅ 参考答案

答案：A。 BernoulliNB 建模二元特征是否出现，也适用于布尔事件数据。

7. 拉普拉斯平滑主要解决什么问题？

A. 未见词导致条件概率为 0
B. 特征尺度不同
C. 梯度爆炸
D. 样本顺序混乱

✅ 参考答案

答案：A。 若某类别从未出现某词，直接频率估计为 0，会让整篇文档的联合概率乘积归零。

8. 为什么实际计算常使用对数概率？

A. 把连乘变连加并避免数值下溢
B. 让概率超过 1
C. 删除先验概率
D. 强制特征独立

✅ 参考答案

答案：A。 大量很小概率相乘会接近浮点数下限，取对数后更稳定。

9. CountVectorizer.fit 应在哪部分数据上进行？

A. 只在训练集
B. 训练集与测试集一起
C. 只在测试集
D. 任意外部语料且无需说明

✅ 参考答案

答案：A。 词表也属于从数据学习的参数，使用测试集构建词表会造成数据泄漏。

10. 文本分类准确率很高，但少数负面评论几乎识别不到，应优先查看什么？

A. 少数类 Recall、F1 和混淆矩阵
B. 只看总词数
C. 回归 R²
D. 聚类 SSE

✅ 参考答案

答案：A。 类别不平衡会掩盖少数类失败，必须按类别评估。

二、简答题（10 题）

11. 用疾病检测举例解释先验、似然和后验

✅ 参考答案

先验是检测前人群患病概率；似然是患病/未患病时出现阳性结果的概率；后验是看到阳性结果后真正患病的概率。后验同时受检测性能和基础患病率影响。

12. 为什么“检测准确率高”不等于“阳性就大概率患病”？

✅ 参考答案

当疾病非常罕见时，即使假阳性率很低，大量健康人也可能产生不少假阳性。阳性后的患病概率必须用先验患病率、灵敏度和特异度一起通过贝叶斯公式计算。

13. 条件独立假设明显不成立时，朴素贝叶斯为什么仍可能有效？

✅ 参考答案

分类只需正确比较各类别的相对后验得分，不一定要求概率完全真实。即使相关特征被重复计权，决策边界仍可能足够好；模型简单、方差低，在小数据高维文本中尤其有优势。

14. 词袋模型丢失了哪些信息？

✅ 参考答案

它主要保留词是否出现或出现次数，忽略词序、长距离依赖、否定范围和上下文语义。例如“非常不好”和“不是非常好”可能有相似词计数。

15. 停用词应不应该全部删除？

✅ 参考答案

不应机械删除。任务无关的高频虚词可减少噪声，但“不是”“没有”等否定词对情感至关重要。停用词表应结合任务、数据和验证结果定制。

16. 比较词频与 TF-IDF

✅ 参考答案

词频直接统计出现次数，简单且适合 MultinomialNB；TF-IDF 下调跨文档常见词、突出区分性词。两者都是非负特征，可通过交叉验证比较，TF-IDF 不一定总优。

17. alpha 太小或太大分别可能怎样？

✅ 参考答案

太小会让稀有/未见事件概率极端，对小样本敏感；太大会把各词条件概率过度拉向均匀分布，削弱有区分力的证据。应在训练集内调参。

18. 为什么分词器也属于建模流程的一部分？

✅ 参考答案

分词决定特征边界，同一文本的切分方式会改变词表与计数。训练和上线必须使用相同分词器、词典和清洗规则，并与向量器、模型一起版本化。

19. 如何处理训练时从未出现的新词？

✅ 参考答案

固定词表的向量器会忽略词表外词，不会报错；模型仍根据已知词和先验预测。若新词大量出现，说明分布漂移，应更新语料、词表和模型，而不是临时在测试时重新 fit。

20. 朴素贝叶斯概率可以直接当成可靠业务概率吗？

✅ 参考答案

不一定。特征独立假设常使概率过于极端，排序/分类可好但校准较差。若业务依赖概率值，应在独立数据上检查校准曲线，并考虑校准方法。

三、代码题（10 题）

21. 用贝叶斯公式计算阳性后的患病概率

患病率 1%，灵敏度 95%，假阳性率 5%。

✅ 参考答案
Python
prevalence = 0.01
sensitivity = 0.95
false_positive_rate = 0.05

numerator = sensitivity * prevalence
denominator = numerator + false_positive_rate * (1 - prevalence)
posterior = numerator / denominator
print(posterior)  # 约 0.161

虽然检测性能不错，阳性后患病概率仍只有约 16.1%，原因是先验患病率很低。

25. 用字符 n-gram 构建无需分词器的中文文本分类器

✅ 参考答案
Python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.pipeline import make_pipeline

model = make_pipeline(
    TfidfVectorizer(analyzer="char", ngram_range=(2, 4), min_df=2),
    MultinomialNB(alpha=0.5),
)
model.fit(text_train, y_train)
print(model.score(text_test, y_test))

26. 使用 jieba 分词后训练中文情感分类器

✅ 参考答案
Python
# 首次运行前安装：pip install jieba
import jieba
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.pipeline import make_pipeline

def tokenize(text):
    return [word for word in jieba.lcut(text) if word.strip()]

model = make_pipeline(
    CountVectorizer(tokenizer=tokenize, token_pattern=None),
    MultinomialNB(alpha=1.0),
)
model.fit(text_train, y_train)

27. 输出每个类别最有代表性的 10 个词

假设 model 是 CountVectorizer + MultinomialNB Pipeline。

✅ 参考答案
Python
import numpy as np

vectorizer = model.named_steps["countvectorizer"]
classifier = model.named_steps["multinomialnb"]
words = np.asarray(vectorizer.get_feature_names_out())

for class_name, log_probability in zip(classifier.classes_, classifier.feature_log_prob_):
    top_words = words[np.argsort(log_probability)[-10:][::-1]]
    print(class_name, top_words.tolist())
