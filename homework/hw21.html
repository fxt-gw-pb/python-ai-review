🧠 机器学习概述与建模流程

🏷️ 选择 + 简答 + 代码🏷️ 30 题📖 相关讲义：机器学习概述与建模流程

30 道综合练习，覆盖机器学习基本概念、监督/无监督学习、特征工程、数据集划分、评估规范与拟合诊断。建议先独立作答，再展开参考答案。

一、选择题（10 题）

1. 下列哪项最符合监督学习的定义？

A. 训练数据只有特征，没有目标值
B. 训练数据同时包含特征和目标值
C. 模型完全不需要历史数据
D. 只用于把样本分成若干簇

✅ 参考答案

答案：B。 监督学习从带标签样本中学习特征到目标的映射；分类和回归都属于监督学习。

2. 预测明天的最高气温属于哪类任务？

A. 分类
B. 回归
C. 聚类
D. 降维

✅ 参考答案

答案：B。 最高气温是连续数值，目标是预测具体大小，因此属于回归。

3. 在没有客户标签的情况下自动划分消费群体，最适合使用哪类方法？

A. 无监督聚类
B. 监督分类
C. 线性回归
D. 时间序列插值

✅ 参考答案

答案：A。 数据没有预先定义的类别，目标是发现样本内部结构，属于无监督聚类。

4. 验证集的主要用途是什么？

A. 训练模型参数
B. 调整超参数和选择模型
C. 代替全部训练数据
D. 只用于保存原始数据

✅ 参考答案

答案：B。 训练集用于拟合参数，验证集用于模型选择和调参，测试集留到最后做一次无偏评估。

5. 下列哪种做法会造成数据泄漏？

A. 在训练集上拟合标准化器，再变换测试集
B. 先用全部数据拟合标准化器，再划分训练集和测试集
C. 固定随机种子
D. 在训练集上做交叉验证

✅ 参考答案

答案：B。 标准化器提前看到了测试集的均值和标准差，测试信息泄漏进了训练过程。

6. 模型在训练集和测试集上的表现都很差，最可能是什么问题？

A. 过拟合
B. 欠拟合
C. 数据泄漏
D. 测试集过小必然导致

✅ 参考答案

答案：B。 训练误差也很高，说明模型连训练数据中的主要规律都没有学好，通常是欠拟合。

7. 模型训练准确率 99%，测试准确率 68%，首先应怀疑什么？

A. 欠拟合
B. 过拟合
C. 标签一定无误
D. 模型一定可上线

✅ 参考答案

答案：B。 训练与测试表现差距很大是典型过拟合信号，也应同时排查划分方式和数据分布差异。

8. 类别型特征“城市={北京、上海、广州}”通常应如何处理？

A. 随意映射成 1、2、3 并认为有大小关系
B. One-Hot 编码
C. 删除所有样本
D. 取字符串长度

✅ 参考答案

答案：B。 名义类别没有自然顺序，One-Hot 能避免人为引入“广州大于上海”的错误序关系。

9. random_state 的主要作用是什么？

A. 自动提高模型精度
B. 让涉及随机性的实验可复现
C. 删除异常值
D. 自动平衡类别

✅ 参考答案

答案：B。 固定随机种子可复现数据划分、初始化和采样结果，但并不会天然提高性能。

10. 面对正类仅占 1% 的欺诈检测，哪项判断最合理？

A. 只看准确率就足够
B. 应结合 Precision、Recall、PR-AUC 等指标
C. 永远预测负类是优秀模型
D. 测试集不必保留类别比例

✅ 参考答案

答案：B。 极度不平衡时，全部预测为负也有 99% 准确率，必须关注少数类能否被识别及误报情况。

二、简答题（10 题）

11. 用自己的话说明“特征”和“标签”的区别，并各举一个例子

✅ 参考答案

特征是模型用来做判断的输入变量，标签是希望模型预测的目标。例如房价预测中，面积、楼层和房龄是特征，成交价是标签。

12. 为什么测试集不能反复用于调参？

✅ 参考答案

反复根据测试结果修改模型，相当于间接让模型“看见”测试集，最终测试分数会对这份数据过拟合。正确做法是用训练集/验证集或交叉验证调参，测试集只在方案确定后使用。

13. 说明分类与回归的核心区别

✅ 参考答案

分类预测离散类别或类别概率，如“是否违约”；回归预测连续数值，如“预计损失金额”。两者的损失函数和评估指标也不同。

14. 什么是泛化能力？

✅ 参考答案

泛化能力是模型对训练时未见过、但来自相同业务分布的新样本保持良好表现的能力。训练分数高不等于泛化好，必须用独立数据评估。

15. 列举三种常见的数据质量问题及处理思路

✅ 参考答案

可包括：缺失值（删除或合理插补）、重复记录（按业务主键去重）、异常值（核查来源后截断/变换/保留）、错误类型（统一格式）、标签错误（抽样复核或重标注）。处理方式要结合业务含义，不能只机械删除。

16. 为什么数据划分应尽量模拟真实上线场景？

✅ 参考答案

离线评估只有在数据生成方式接近上线时才可信。例如预测未来应按时间划分，识别新用户应按用户分组划分；随机拆行可能让同一用户或未来信息同时出现在训练和测试中，造成虚高结果。

17. 解释“模型参数”和“超参数”的区别

✅ 参考答案

模型参数由训练算法从数据中学习，例如线性回归系数；超参数由训练前设置或通过验证选择，例如 KNN 的 n_neighbors、树的 max_depth。

18. 为什么建立简单基线模型很重要？

✅ 参考答案

基线提供最低可接受参照，能判断复杂模型是否真的带来收益，也便于发现数据或评估流程错误。分类可用多数类，回归可用均值/中位数，之后再与正式模型比较。

19. 交叉验证解决了什么问题？

✅ 参考答案

一次训练/验证划分可能因抽样偶然性导致分数不稳定。K 折交叉验证轮流使用不同折验证，汇总均值和波动，能更稳健地比较方案并充分利用有限训练数据。

20. 从数据、模型和评估三个层面各给出一种缓解过拟合的方法

✅ 参考答案
  • 数据：增加有代表性的训练样本或做合理数据增强；
  • 模型：降低复杂度、剪枝或加入正则化；
  • 评估：使用严格交叉验证、按组/按时间划分，并排查数据泄漏。

22. 为数值列和类别列建立无泄漏预处理器

数值列为 age、income，类别列为 city。

✅ 参考答案
Python
from sklearn.compose import ColumnTransformer
from sklearn.impute import SimpleImputer
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import OneHotEncoder, StandardScaler

numeric = Pipeline([
    ("imputer", SimpleImputer(strategy="median")),
    ("scaler", StandardScaler()),
])
category = Pipeline([
    ("imputer", SimpleImputer(strategy="most_frequent")),
    ("onehot", OneHotEncoder(handle_unknown="ignore")),
])
preprocess = ColumnTransformer([
    ("num", numeric, ["age", "income"]),
    ("cat", category, ["city"]),
])

23. 将预处理和逻辑回归组合成一个 Pipeline 并训练

✅ 参考答案
Python
from sklearn.linear_model import LogisticRegression
from sklearn.pipeline import Pipeline

model = Pipeline([
    ("preprocess", preprocess),
    ("classifier", LogisticRegression(max_iter=1000)),
])
model.fit(X_train, y_train)
y_pred = model.predict(X_test)

Pipeline 能保证预处理只在每次训练折上拟合，并方便整体保存和部署。

24. 编写代码比较多数类基线与正式分类器的准确率

✅ 参考答案
Python
from sklearn.dummy import DummyClassifier
from sklearn.metrics import accuracy_score

baseline = DummyClassifier(strategy="most_frequent")
baseline.fit(X_train, y_train)

baseline_acc = accuracy_score(y_test, baseline.predict(X_test))
model_acc = accuracy_score(y_test, model.predict(X_test))
print({"baseline": baseline_acc, "model": model_acc})

正式模型至少应显著优于合理基线，且还需结合业务指标判断。

25. 对一个分类器做 5 折分层交叉验证，并输出均值和标准差

27. 用学习曲线判断模型是否过拟合

写出计算训练分数和验证分数的代码，不要求绘图。

✅ 参考答案
Python
import numpy as np
from sklearn.model_selection import learning_curve

sizes, train_scores, valid_scores = learning_curve(
    model,
    X,
    y,
    cv=5,
    scoring="accuracy",
    train_sizes=np.linspace(0.2, 1.0, 5),
    n_jobs=-1,
)
print("train:", train_scores.mean(axis=1))
print("valid:", valid_scores.mean(axis=1))

若训练分数持续很高、验证分数明显较低，且两者差距大，通常表示过拟合。
