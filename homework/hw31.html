🧮 NumPy 综合训练

🏷️ 选择 + 简答 + 代码🏷️ 40 题📖 相关讲义：数据分析环境与 NumPy

40 道练习，覆盖 ndarray 的创建与属性、dtype、索引和视图、广播、聚合、缺失值、随机数以及向量化数据分析。

一、选择题（20 题）

1. NumPy 数组相较于普通 Python 列表，最核心的计算优势是什么？

A. 可以存放任意不同类型且没有额外开销
B. 连续、同类型存储便于向量化和底层高效计算
C. 会自动把所有缺失值删除
D. 不需要占用内存

✅ 参考答案

答案：B。 ndarray 通常以同一 dtype 连续或规则地组织数据，可把循环交给底层 C 实现。列表更灵活，但数值批量计算通常更慢。

2. 对形状为 (3, 4) 的数组 a，下列说法正确的是？

A. a.ndim 为 12
B. a.size 为 7
C. a.ndim 为 2 且 a.size 为 12
D. a.shape 为 12

✅ 参考答案

答案：C。 ndim 是轴的数量，shape 是各轴长度组成的元组，size 是元素总数。

3. np.array([1, 2, 3.5]) 默认最可能得到哪种类型？

A. 字符串
B. 布尔值
C. 浮点数
D. 对象类型且无法计算

✅ 参考答案

答案：C。 为容纳 3.5，整数会被提升到兼容的浮点类型。实际位宽还与平台和输入有关，应通过 arr.dtype 确认。

4. np.arange(1, 8, 2) 的结果是？

A. [1, 3, 5, 7]
B. [1, 3, 5]
C. [2, 4, 6, 8]
D. [1, 2, 3, 4]

✅ 参考答案

答案：A。 arange 使用左闭右开区间；从 1 开始每次增加 2，小于 8 的值依次为 1、3、5、7。

5. np.linspace(0, 1, 5) 中的 5 表示什么？

A. 步长为 5
B. 生成 5 个包含两端点的等距值
C. 生成 5 行数组
D. 最大值为 5

✅ 参考答案

答案：B。 结果为 [0, 0.25, 0.5, 0.75, 1]。linspace 更适合明确指定点数的连续区间。

6. 一个含 12 个元素的数组，下面哪个形状不能直接用于 reshape？

A. (3, 4)
B. (2, 2, 3)
C. (6, -1)
D. (5, 3)

✅ 参考答案

答案：D。 变形前后元素总数必须一致。-1 可以让 NumPy 自动推断该维度，因此 (6, -1) 会得到 (6, 2)。

7. flatten() 与 ravel() 的主要区别是？

A. flatten() 通常返回副本，ravel() 尽量返回视图
B. ravel() 只能处理一维数组
C. flatten() 会删除原数组
D. 二者一定共享内存

✅ 参考答案

答案：A。 flatten() 保证得到独立副本；ravel() 在内存布局允许时返回视图，否则也可能复制。

8. 对一维数组执行 view = a[1:4]，再修改 view[0]，通常会怎样？

A. 原数组对应元素也被修改
B. 原数组一定不变
C. 数组自动变成二维
D. 必然报错

✅ 参考答案

答案：A。 基础切片通常返回共享底层内存的视图。若需要隔离修改，应显式调用 .copy()。

9. a[[0, 2, 4]] 这种整数数组索引通常返回什么？

A. 与原数组共享内存的基础视图
B. 高级索引产生的副本
C. 只能返回标量
D. 原数组的转置

✅ 参考答案

答案：B。 整数数组和布尔数组属于高级索引，通常返回副本；它与基础切片的视图语义不同。

10. 对二维数组 a 使用同形状布尔掩码 a[a > 0]，结果通常是？

A. 一维数组，包含所有大于 0 的元素
B. 保持原二维形状并用 0 填充
C. 只返回第一行
D. 一个布尔标量

✅ 参考答案

答案：A。 布尔索引会抽取满足条件的元素并展平为一维结果。若要保持形状，可使用 np.where。

11. 形状 (3, 1) 与 (1, 4) 的数组相加，广播后的结果形状是？

A. (3, 4)
B. (1, 1)
C. (4, 3)
D. 无法广播

✅ 参考答案

答案：A。 从末尾维度比较，1 可以扩展为对应维度，最终组合成 3 行 4 列。

12. 下列哪一对形状不能直接广播？

A. (5, 1) 与 (1, 3)
B. (2, 3, 4) 与 (4,)
C. (2, 3) 与 (3,)
D. (2, 3) 与 (2,)

✅ 参考答案

答案：D。 从末尾比较时 3 与 2 既不相等，也都不是 1。其余形状均满足从右向左相等或一方为 1 的规则。

13. 对形状 (3, 4) 的数组执行 a.sum(axis=0)，结果表示什么？

A. 每行求和，形状 (3,)
B. 每列沿行方向求和，形状 (4,)
C. 全部元素求和，得到标量
D. 保留原形状

✅ 参考答案

答案：B。 axis=0 表示压缩第 0 轴，即把各行叠加，留下列这一维。

14. a.mean(axis=1, keepdims=True) 中 keepdims=True 的作用是？

A. 保留被聚合轴为长度 1，便于后续广播
B. 保留所有原始数据不计算
C. 强制结果为整数
D. 删除全部单例维

✅ 参考答案

答案：A。 例如 (3, 4) 按行求均值会得到 (3, 1)，可以直接与原数组做广播运算。

15. 若数组包含 np.nan，希望计算时忽略缺失值，应优先使用？

A. np.mean
B. np.nanmean
C. np.argmax
D. np.unique

✅ 参考答案

答案：B。 普通 mean 遇到 NaN 通常返回 NaN；nanmean 会忽略 NaN，但若某个切片全为 NaN 仍需额外处理。

16. np.concatenate([a, b], axis=0) 与 np.stack([a, b], axis=0) 的区别是？

A. stack 新增一个轴，concatenate 在已有轴上拼接
B. concatenate 总会新增轴
C. 二者只能拼接标量
D. 二者完全等价

✅ 参考答案

答案：A。 stack 要求输入形状一致并创建新轴；concatenate 要求除拼接轴外其他维度兼容。

17. 想创建互不干扰且可复现的现代随机数生成器，推荐使用？

A. np.random.default_rng(42)
B. 每次都调用系统时间
C. np.empty(42)
D. np.arange(42)

✅ 参考答案

答案：A。 default_rng 返回独立 Generator，比依赖全局随机状态更易测试和复现。

18. np.where(condition, x, y) 的作用是？

A. 条件为真取 x，否则取 y，并支持广播
B. 永远只返回索引
C. 对数组排序
D. 删除重复值

✅ 参考答案

答案：A。 三参数形式用于向量化条件选择；单参数 np.where(condition) 才主要返回满足条件的位置索引。

19. 若 a.shape 为 (2, 3)、b.shape 为 (3, 4)，则 (a @ b).shape 是？

A. (2, 4)
B. (3, 3)
C. (2, 3, 4)
D. (4, 2)

✅ 参考答案

答案：A。 矩阵乘法要求内部维度 3 一致，结果保留两个外部维度 2 与 4。

20. 为什么不应使用 if a == b: 判断两个 NumPy 数组是否完全相等？

A. a == b 产生逐元素布尔数组，其真值通常不明确
B. NumPy 不支持比较
C. 只能比较浮点数
D. 会自动修改数组

✅ 参考答案

答案：A。 完全相等可用 np.array_equal(a, b)；浮点近似相等应使用 np.allclose(a, b)。

二、简答题（8 题）

21. 什么是向量化？为什么它通常比 Python 层 for 循环更快？

✅ 参考答案

向量化是用数组算子一次描述整批数据的计算，把循环交给 NumPy 底层实现。这样可减少 Python 解释器逐元素调度开销，并更好利用连续内存、SIMD 和优化过的数值库。向量化并非绝对更省内存，大型中间数组仍需关注。

22. 请用自己的话说明广播规则，并判断 (2, 1, 4) 与 (3, 4) 能否广播。

✅ 参考答案

从两个形状的最右侧开始逐维比较：维度相等，或其中一个为 1，或某一方不存在该维度时可以广播。(2, 1, 4) 与 (3, 4) 对齐为 (2, 1, 4) 与 (1, 3, 4)，结果为 (2, 3, 4)，因此可以广播。

23. 视图与副本有什么区别？分析时为什么要明确二者？

✅ 参考答案

视图与原数组共享底层内存，修改一方可能影响另一方；副本拥有独立数据。基础切片常返回视图，高级索引和 .copy() 通常返回副本。若不清楚共享关系，清洗步骤可能意外改写原数据；反过来，无意义复制又会增加内存消耗。

24. 如何理解 axis=0 和 axis=1？请以二维“行是样本、列是特征”的数组为例。

✅ 参考答案

聚合时，axis 指被压缩掉的轴。axis=0 把多行压缩，得到每个特征在所有样本上的统计；axis=1 把多列压缩，得到每个样本跨特征的统计。先写出输入与期望输出形状，比死记“行/列”更可靠。

25. 整数数组为什么不能直接用 np.nan 表示缺失值？有哪些处理办法？

✅ 参考答案

传统 NumPy 整数 dtype 没有 NaN 这一特殊浮点编码。常见办法是转为浮点数组后使用 NaN；同时维护布尔掩码；使用 np.ma.MaskedArray；或在进入 Pandas 后采用可空整数类型 Int64。

26. 为什么列标准化时常写成 (x - mean) / std，并给 mean、std 保留维度？

✅ 参考答案

减均值用于中心化，除以标准差把不同量纲缩放到可比较范围。使用 mean(axis=0, keepdims=True) 和同形状的 std 会得到 (1, 特征数)，可以清晰地广播到全部样本。标准差为 0 的常量列必须单独处理。

27. 固定随机种子能保证任何机器、任何 NumPy 版本都产生逐位一致的结果吗？

✅ 参考答案

不能绝对保证。固定生成器和种子可让同一算法、版本与环境中的随机序列复现，但不同 NumPy 版本、位生成器、硬件或并行算法仍可能有差异。严谨实验应记录代码、依赖版本、种子和关键输入。

28. 当一个向量化表达式产生多个超大中间数组时，可以怎样降低内存压力？

✅ 参考答案

可分块处理数据；使用原地算子但先确认不会破坏后续依赖；通过 out= 复用输出缓冲区；选择足够而不过宽的 dtype；避免重复广播物化；或使用能融合操作、延迟执行及处理磁盘数据的工具。优化前先用形状和内存占用定位瓶颈。

三、代码题（12 题）

29. 创建一个包含 1～24 的数组并变形为 (2, 3, 4)，输出其形状、维数、元素总数和类型。

✅ 参考答案
Python
import numpy as np

a = np.arange(1, 25).reshape(2, 3, 4)
print(a)
print("shape:", a.shape)
print("ndim:", a.ndim)
print("size:", a.size)
print("dtype:", a.dtype)

30. 给定 a = np.arange(1, 21).reshape(4, 5)，取出第 2～3 行、第 2～4 列，并创建独立副本。

✅ 参考答案
Python
import numpy as np

a = np.arange(1, 21).reshape(4, 5)
result = a[1:3, 1:4].copy()
print(result)
print("与原数组共享内存：", np.shares_memory(a, result))

切片采用左闭右开区间；.copy() 防止后续修改 result 时影响 a。

31. 将数组中小于 0 的值替换为 0，大于 100 的值替换为 100，要求不修改原数组。

✅ 参考答案
Python
import numpy as np

a = np.array([-8, 20, 115, 60, -1, 100])
clipped = np.clip(a, 0, 100)

33. 对每一列做 Z-score 标准化；若某列标准差为 0，则该列结果设为 0。

✅ 参考答案

    [10.0, 2.0, 5.0],
    [12.0, 2.0, 9.0],
    [14.0, 2.0, 7.0],
])

mean = x.mean(axis=0, keepdims=True)
std = x.std(axis=0, keepdims=True)

39. 给定二维坐标点，计算所有点两两之间的欧氏距离矩阵，不能写 Python 双重循环。

✅ 参考答案
Python
import numpy as np

40. 使用独立随机数生成器，通过 100 万个随机点估计圆周率，并保证同环境下可复现。

✅ 参考答案
Python
import numpy as np

rng = np.random.default_rng(42)
n = 1_000_000
points = rng.uniform(-1, 1, size=(n, 2))
inside = (points[:, 0] ** 2 + points[:, 1] ** 2) <= 1
pi_estimate = 4 * inside.mean()
print("π 估计值：", pi_estimate)

单位正方形中落入内切圆的比例约为 π / 4。
