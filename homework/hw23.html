📈 线性回归与正则化

🏷️ 选择 + 简答 + 代码🏷️ 40 题📖 相关讲义：线性回归、梯度下降与正则化

40 道练习，覆盖线性模型、最小二乘、正规方程、梯度下降、回归指标、多项式回归、过拟合与 L1/L2 正则化。

一、选择题（14 题）

1. 一元线性回归模型通常写成什么形式？

A. y = wx + b
B. y = x / 0
C. y = max(x)
D. y = class(x)

✅ 参考答案

答案：A。 w 是斜率，b 是截距；多元线性回归可写成向量形式 y = Xw + b。

2. 最小二乘法希望最小化什么？

A. 预测误差绝对值的最大值
B. 残差平方和
C. 样本数量
D. 特征数量

✅ 参考答案

答案：B。 残差平方和对大误差惩罚更重，也便于求导优化。

3. MSE 与 RMSE 的主要区别是什么？

A. RMSE 是 MSE 开平方，与目标同量纲
B. 二者永远数值相同
C. MSE 只能用于分类
D. RMSE 不受异常值影响

✅ 参考答案

答案：A。 两者都对大误差敏感；RMSE 还原到目标变量的单位，更容易解释。

4. MAE 相比 MSE 的典型特点是什么？

A. 对极端误差相对更稳健
B. 会把误差平方
C. 只能为负数
D. 必须先标准化标签

✅ 参考答案

答案：A。 MAE 线性惩罚误差，不会像平方损失那样极度放大离群点影响。

5. R²=1 表示什么？

A. 模型预测完全匹配真实值
B. 模型一定过拟合
C. 模型和均值预测相同
D. 所有系数为 1

✅ 参考答案

答案：A。 在当前评估数据上残差平方和为 0。是否泛化仍需看独立测试集。

6. 测试集上的 R²<0 通常说明什么？

A. 模型比直接预测测试集均值还差
B. sklearn 出错
C. 目标值必须为负
D. 模型完美

✅ 参考答案

答案：A。 R² 可以为负，表示残差平方和大于以均值为基准的总平方和。

7. 梯度下降中学习率过大可能造成什么？

A. 损失震荡或发散
B. 一定更快收敛
C. 自动消除过拟合
D. 特征数减少

✅ 参考答案

答案：A。 步长过大会越过最优点甚至不断放大；过小则收敛缓慢。

8. 多元线性回归中严重多重共线性可能导致什么？

A. 系数不稳定、解释困难
B. 样本自动增多
C. 目标变成分类
D. MSE 必为零

✅ 参考答案

答案：A。 高度相关特征使同一解释可由多个系数组合承担，轻微数据变化就可能导致系数大幅波动。

9. 多项式回归为什么仍可视为线性模型？

A. 它对变换后的特征系数仍是线性的
B. 它没有参数
C. 它只能拟合直线
D. 它不计算损失

✅ 参考答案

答案：A。 例如 y = w1*x + w2*x² + b 对参数 w1,w2,b 仍是线性的。

10. L1 正则化的典型效果是什么？

A. 促使部分系数精确变为 0
B. 让所有系数完全相等
C. 增加特征数量
D. 禁止截距

✅ 参考答案

答案：A。 Lasso 可产生稀疏解，因此也具有一定特征选择作用。

11. L2 正则化的典型效果是什么？

A. 平滑收缩系数，通常不强制变为 0
B. 删除全部样本
C. 只保留一个特征
D. 把回归变成聚类

✅ 参考答案

答案：A。 Ridge 惩罚系数平方，能缓解共线性并降低模型方差。

12. 正则化强度 alpha 越大，通常会怎样？

A. 系数被压缩得更强
B. 模型必然更复杂
C. 训练损失必然更小
D. 特征自动恢复原尺度

✅ 参考答案

答案：A。 过大的 alpha 可能把模型压得过于简单而欠拟合，应通过交叉验证选择。

13. 为什么 Lasso/Ridge 前通常要缩放特征？

A. 让正则项公平作用于不同量纲的系数
B. 让目标变成整数
C. 让样本排序
D. 防止读取 CSV

✅ 参考答案

答案：A。 不同特征尺度会导致同等业务影响对应不同系数大小，从而受到不公平的正则惩罚。

14. 下列哪种现象最像高阶多项式过拟合？

A. 训练误差极低但验证误差很高，曲线剧烈波动
B. 训练和验证误差都高
C. 训练数据为空
D. 所有预测等于均值且最优

✅ 参考答案

答案：A。 高阶模型可能把噪声也拟合进去，训练点附近曲线异常摆动，泛化性能下降。

二、简答题（12 题）

15. 解释“残差”的含义

✅ 参考答案

残差是样本真实值与模型预测值之差，通常写作 e_i = y_i - ŷ_i。分析残差可帮助发现非线性、异方差、异常点和遗漏变量。

16. 正规方程与梯度下降各有什么优缺点？

✅ 参考答案

正规方程可一次得到闭式解，超参数少，但特征很多时矩阵运算和求逆代价高；梯度下降适合大规模数据，可在线/分批优化，但需要学习率、迭代次数和收敛诊断。

17. 为什么特征缩放能帮助梯度下降？

✅ 参考答案

尺度差异大会使损失面的等高线狭长，梯度在不同方向步幅失衡、呈之字形收敛。缩放后曲面更接近圆形，通常能更稳定快速地优化。

18. MAE、RMSE 应如何根据业务选择？

✅ 参考答案

若希望每单位误差线性计价且需降低极端值影响，可看 MAE；若大误差代价显著更高，RMSE 更合适。最好同时报告并用业务单位解释。

19. 为什么不能只报告 R²？

✅ 参考答案

R² 是相对基线指标，不直接展示误差的业务量纲，也可能被数据范围影响。应结合 MAE/RMSE、残差分布、基线和不同子群表现综合判断。

20. 什么是异方差？它会带来什么问题？

✅ 参考答案

异方差指残差方差随预测值或特征而变化，例如房价越高误差波动越大。它会使经典线性回归的标准误和显著性推断不可靠，也提示可能需要目标变换、加权回归或更合适模型。

21. 如何从残差图判断模型遗漏了非线性关系？

✅ 参考答案

理想残差应围绕 0 随机分布。若残差随预测值呈明显弧线、波浪或系统趋势，说明线性形式没有捕捉到结构，可增加合理变换/交互项或换非线性模型。

22. 解释偏差—方差权衡在线性回归中的体现

✅ 参考答案

简单线性模型限制强，偏差可能高但方差低；加入大量高阶项后偏差下降、方差上升。正则化通过允许少量训练误差来降低方差，常能改善测试表现。

23. Lasso 和 Ridge 应如何选择？

✅ 参考答案

希望得到稀疏模型、特征较多且部分确实无关时可尝试 Lasso；相关特征较多、希望稳定保留它们时 Ridge 往往更稳。Elastic Net 可折中，最终通过交叉验证和业务可解释性选择。

24. 为什么在训练集上 MSE 最小的模型不一定最好？

✅ 参考答案

复杂模型可通过记忆噪声不断降低训练误差，但新数据误差可能升高。模型选择应看验证/交叉验证表现，并考虑稳定性、复杂度和业务成本。

25. 如果房价目标分布严重右偏，可尝试什么处理？

✅ 参考答案

可对目标使用 log1p 变换，使相对误差更均衡、缓解异方差；预测后用 expm1 还原。评估时要明确是在对数尺度还是原始价格尺度计算指标。

26. 解释交互项 面积 × 地段等级 的含义

✅ 参考答案

交互项表示面积对价格的影响会随地段等级变化，而不是两个特征效果简单相加。例如核心地段每增加 1 平方米带来的价格增量可能更高。

27. 用 numpy 手工计算一元线性回归的斜率和截距

✅ 参考答案
Python
import numpy as np

x = np.array([1.0, 2.0, 3.0, 4.0])
y = np.array([2.0, 3.0, 5.0, 6.0])
w = np.sum((x - x.mean()) * (y - y.mean())) / np.sum((x - x.mean()) ** 2)
b = y.mean() - w * x.mean()
print(w, b)

29. 写一个函数计算 MSE，不能调用 sklearn

✅ 参考答案
Python
import numpy as np

def mse(y_true, y_pred):
    y_true = np.asarray(y_true)
    y_pred = np.asarray(y_pred)
    return np.mean((y_true - y_pred) ** 2)

31. 用正规方程求线性回归系数，并兼容不可逆矩阵

✅ 参考答案
Python
import numpy as np

X_bias = np.c_[np.ones(len(X)), X]
theta = np.linalg.pinv(X_bias.T @ X_bias) @ X_bias.T @ y
print(theta)  # 第一个元素是截距，其余是系数

使用伪逆 pinv 比直接 inv 更能处理奇异或近似奇异矩阵。

32. 手写批量梯度下降拟合一元线性回归

✅ 参考答案
Python
import numpy as np

x = np.asarray(x, dtype=float)
y = np.asarray(y, dtype=float)
w, b = 0.0, 0.0
learning_rate = 0.01

for _ in range(5000):
    prediction = w * x + b
    error = prediction - y
    w -= learning_rate * 2 * np.mean(error * x)
    b -= learning_rate * 2 * np.mean(error)

print(w, b)

from sklearn.dummy import DummyRegressor
