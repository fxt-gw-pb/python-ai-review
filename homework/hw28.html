🪐 K-Means 聚类与评估

🏷️ 选择 + 简答 + 代码🏷️ 35 题📖 相关讲义：K-Means 聚类与聚类评估

35 道练习，覆盖 K-Means 迭代过程、初始化、尺度与异常值、SSE、肘部法、轮廓系数、Calinski–Harabasz 指标和客户分群。

一、选择题（12 题）

1. K-Means 属于哪类学习方法？

A. 无监督聚类
B. 监督回归
C. 强化学习
D. 关联规则

✅ 参考答案

答案：A。 K-Means 在没有标签的情况下根据样本距离发现 K 个簇。

2. K-Means 的目标函数是什么？

A. 最小化样本到所属质心的平方距离和
B. 最大化分类准确率
C. 最小化标签交叉熵
D. 最大化回归 R²

✅ 参考答案

答案：A。 该值在 sklearn 中可通过 inertia_ 查看，也称簇内平方和 SSE。

3. 标准 K-Means 中每个簇的中心如何更新？

A. 取簇内样本均值
B. 取全数据最大值
C. 随机保持不变
D. 取类别标签

✅ 参考答案

答案：A。 对平方欧氏距离而言，均值是使簇内平方误差最小的代表点。

4. K-Means 迭代中的两个核心步骤是什么？

A. 分配样本到最近中心；重新计算中心
B. 训练/测试划分；监督打标签
C. 求梯度；反向传播
D. 编码；解码

✅ 参考答案

答案：A。 两步交替执行，直到标签/中心基本不变或达到迭代上限。

5. K-Means 为什么对特征尺度敏感？

A. 距离会被量纲大的特征主导
B. 它只支持字符串
C. 它不使用距离
D. 缩放会改变 K

✅ 参考答案

答案：A。 年收入的数值范围可能远大于评分，若不缩放，收入几乎决定聚类。

6. K-Means++ 的主要作用是什么？

A. 更合理地分散初始化中心
B. 自动给出真实标签
C. 删除异常值
D. 把 K 固定为 2

✅ 参考答案

答案：A。 K-Means++ 让新中心更倾向远离已有中心，通常比纯随机初始化更稳定、收敛更快。

7. n_init 表示什么？

A. 使用不同初始化重复运行的次数/策略
B. 特征数量
C. 样本数量
D. 最大簇大小

✅ 参考答案

答案：A。 多次初始化后选择目标函数最小的结果，能缓解局部最优。

8. SSE 随 K 增大通常如何变化？

A. 单调不增
B. 单调增加
C. 固定不变
D. 必然先负后正

✅ 参考答案

答案：A。 更多中心可让样本离所属中心更近；极端地 K 等于样本数时 SSE 可为 0，因此不能只选最小 SSE。

9. 轮廓系数越接近 1 通常表示什么？

A. 簇内紧密且簇间分离较好
B. 聚类一定错误
C. 所有样本都在一个簇
D. SSE 最大

✅ 参考答案

答案：A。 接近 0 表示样本在边界附近，负值说明可能更接近其他簇。

10. CH 指标通常如何解释？

A. 同一数据上值越大通常表示簇间更分离、簇内更紧密
B. 越小越好且可为负
C. 等于分类准确率
D. 只用于回归

✅ 参考答案

答案：A。 Calinski–Harabasz 是簇间离散与簇内离散的比值，并对自由度做调整。

11. K-Means 最不适合哪种簇形状？

A. 两个弯月形、非凸簇
B. 大小相近的球状簇
C. 方差相近的圆团
D. 中心分离明显的簇

✅ 参考答案

答案：A。 基于质心和欧氏距离的 Voronoi 划分偏好凸、近似球状且尺度相近的簇。

12. 对含大量极端异常值的数据直接运行 K-Means，可能发生什么？

A. 均值质心被异常值拉偏
B. 异常值自动消失
C. 一定得到真实 K
D. 距离不受影响

✅ 参考答案

答案：A。 平方距离会放大极端点影响，应先核查异常、使用稳健缩放/截断，或考虑 K-Medoids、DBSCAN 等方法。

二、简答题（11 题）

13. 用文字描述 K-Means 的完整迭代流程

✅ 参考答案

选择 K 个初始中心；将每个样本分配给最近中心；对每簇重新计算样本均值作为新中心；重复分配和更新，直到中心变化很小、标签不变或达到最大迭代次数。

14. 为什么 K-Means 可能得到局部最优？

✅ 参考答案

目标函数非凸，不同初始中心会走向不同解。使用 K-Means++、多次初始化并比较 inertia_，能降低差初始化风险，但不能保证全局最优。

15. 肘部法如何选择 K？它有什么主观性？

✅ 参考答案

绘制 K 与 SSE，寻找从快速下降转为平缓的“肘点”。有些数据曲线平滑无明显拐点，判断较主观，应结合轮廓系数、CH、稳定性和业务可解释性。

16. 聚类没有真实标签时如何评估？

✅ 参考答案

可用内部指标（SSE、轮廓系数、CH）、重采样稳定性、不同初始化一致性、降维可视化，以及簇画像是否可解释、能否支持具体业务行动。单一分数不足以决定方案。

17. 为什么聚类前通常应做标准化？

✅ 参考答案

K-Means 完全由距离驱动。标准化让各特征以相近尺度参与距离，避免大单位特征支配；但若业务上某特征确应更重要，可在缩放后显式加权。

18. K-Means 的簇编号 0、1、2 是否有大小或固定含义？

✅ 参考答案

没有。编号只是本次运行的任意标签，初始化或数据变化后可能置换。业务系统应根据簇画像命名，例如“高收入高消费”，不能依赖编号本身。

19. 如何判断客户分群是否具有业务价值？

✅ 参考答案

检查每簇规模、特征画像、稳定性和可触达性；不同簇应能对应差异化策略，并通过 A/B 测试验证转化或留存收益。漂亮的二维图不等于业务有效。

20. K-Means 对类别型特征有什么限制？

✅ 参考答案

均值和欧氏距离不适合无序类别。简单 One-Hot 后虽可运行，但距离含义可能失真、高基数维度膨胀。可考虑 K-Modes/K-Prototypes、Gower 距离或针对业务设计表示。

21. 为什么轮廓系数不一定能选出业务最合适的 K？

✅ 参考答案

它偏好几何上紧密、分离的簇，未考虑营销成本、最小可运营人群、合规和策略差异。业务上可能选择分数略低但更可解释、可行动的 K。

22. 比较 K-Means 与 DBSCAN 的适用场景

✅ 参考答案

K-Means 需预设 K，擅长近球状簇、可预测新点且速度快；DBSCAN 按密度发现任意形状簇、能标记噪声且无需 K，但对密度差异、高维和参数尺度敏感。

23. 新客户到来后如何分配到已有簇？

✅ 参考答案

必须使用训练时保存的同一预处理器缩放新客户，再由已训练 K-Means 的 predict 分配到最近中心。不能为单个新客户重新 fit，否则中心与标签语义会改变。

plt.ylabel("SSE")
plt.show()

35. 保存聚类 Pipeline，并为新客户分簇

✅ 参考答案
Python
import joblib
import pandas as pd
from sklearn.cluster import KMeans
from sklearn.pipeline import make_pipeline
from sklearn.preprocessing import StandardScaler

df = pd.read_csv("data/customers.csv")
features = ["Annual Income (k$)", "Spending Score (1-100)"]
pipeline = make_pipeline(
    StandardScaler(),
    KMeans(n_clusters=5, n_init="auto", random_state=42),
)
pipeline.fit(df[features])
joblib.dump(pipeline, "customer_cluster_pipeline.joblib")
loaded = joblib.load("customer_cluster_pipeline.joblib")

new_customers = pd.DataFrame({
    "Annual Income (k$)": [35, 90],
    "Spending Score (1-100)": [80, 20],
})
new_labels = loaded.predict(new_customers[features])
print(new_labels)
