🐧 Linux 基础测评

🏷️ 选择 + 简答🏷️ 26 题📖 相关讲义：Linux 基础：从终端到服务器运维

先自己作答再展开答案。选择题建议限时完成（两组各约 15 分钟），简答题动手在虚拟机里敲一遍效果最好。配套讲义：《Linux 基础：从终端到服务器运维》。

一、Linux 认知与环境（作业一 · 选择题）

1.【单选】以下哪个不属于计算机软件（　）

  • A. QQ
  • B. CPU
  • C. 微信
  • D. 操作系统
✅ 参考答案

B。CPU 是硬件（中央处理器）；QQ、微信是应用软件，操作系统是系统软件。

2.【单选】以下哪个不属于计算机操作系统（　）

  • A. MacOS
  • B. Linux
  • C. Windows
  • D. Symbian
✅ 参考答案

D。Symbian（塞班）是早年诺基亚手机上的移动操作系统，不属于计算机（桌面/服务器）操作系统；MacOS、Linux、Windows 都是。

3.【单选】Linux 的创始人为（　）

  • A. Elon Reeve Musk
  • B. Larry Ellison
  • C. Linus Torvalds
  • D. James Gosling
✅ 参考答案

C。Linus Torvalds 于 1991 年发布了 Linux 内核。Larry Ellison 是 Oracle 创始人，James Gosling 是 Java 之父。

4.【单选】以下哪项不是 VMware 支持的网络模式（　）

  • A. NAT 网络
  • B. 桥接模式
  • C. 仅主机模式
  • D. 云连接模式
✅ 参考答案

D。VMware 的三种网络模式：NAT（借宿主机上网）、桥接（虚拟机像局域网独立设备）、仅主机（只与宿主机通信）。没有"云连接模式"。

5.【单选】课程中使用的操作系统属于（　）

  • A. Ubuntu
  • B. CentOS
  • C. RedHat
  • D. RedFlag
✅ 参考答案

B。课程环境使用 CentOS（Red Hat 系的社区发行版）。注意区分：CentOS 基于 RedHat（RHEL）重构，但属于不同发行版。

6.【多选】在 Windows 端使用远程连接软件连接 Linux 需要哪些？（　）

  • A. 网络支持
  • B. SSH 服务
  • C. FTP 服务
  • D. Linux 账号密码
✅ 参考答案

ABD。远程登录 Linux 的三要素：网络可达、Linux 端开启 SSH 服务、有效的账号密码。FTP 是文件传输服务，不是远程登录必需的。

7.【单选】SSH 服务的端口为（　）

  • A. 21
  • B. 22
  • C. 23
  • D. 24
✅ 参考答案

B。SSH 默认端口 22。顺带记忆：21 是 FTP，23 是 Telnet。

8.【多选】FTP 服务支持以下哪些操作（　）

  • A. 上传
  • B. 下载
  • C. 远程连接
  • D. 缓存管理
✅ 参考答案

AB。FTP（File Transfer Protocol）是文件传输协议，核心功能是文件的上传和下载；远程命令行连接是 SSH 的职责。

9.【多选】Linux 是一个（　）操作系统？

  • A. 免费开源
  • B. 收费闭源
  • C. 单用户多任务
  • D. 多用户多任务
✅ 参考答案

AD。Linux 免费开源，并且是多用户、多任务操作系统——多个用户可以同时登录同一台机器，各自运行多个程序。

二、Linux 认知与环境（作业一 · 简答题）

10. 与 Windows Server 相比，为什么更多企业选择 Linux 服务器操作系统？

✅ 参考答案

主要原因：

  1. 免费开源：没有授权费用，源代码可审计、可定制；
  2. 稳定可靠：可长时间不重启运行，适合 7×24 小时服务；
  3. 安全性好：权限模型严格，病毒木马生态远小于 Windows；
  4. 资源占用低：无图形界面的服务器版本对硬件要求低，同样的机器能承载更多服务；
  5. 生态与运维：主流服务端软件（Nginx、MySQL、Redis、Docker、K8s 等）原生支持，社区资料丰富，SSH 远程管理和自动化运维工具链成熟。

11. 请列举 2-3 个 Linux 的发行版本

✅ 参考答案

常见发行版：Ubuntu、CentOS、Debian、Red Hat Enterprise Linux（RHEL）、Fedora、openSUSE、Arch Linux 等。它们的区别主要在软件组合、发布节奏和包管理器（apt / dnf / yum 等），核心都是 Linux 内核。

三、Linux 常用命令（作业二 · 选择题）

12.【单选】用什么命令可以查看 ls 帮助（　）

  • A. man ls
  • B. ls man
  • C. --help ls
  • D. help(ls)
✅ 参考答案

A。man 命令名 查看手册页；另一种常用写法是 ls --help（注意 --help 跟在命令后面）。

13.【单选】查看当前所在的目录位置，应使用的命令是（　）

  • A. ls -l
  • B. ls -a
  • C. pwd
  • D. cat
✅ 参考答案

C。pwd（print working directory）打印当前工作目录的绝对路径。

14.【多选】列出当前目录下的文件和目录（　）

  • A. ls -a
  • B. ls -l
  • C. ls -h
  • D. ls -x
✅ 参考答案

AB。ls -a 列出包括隐藏文件在内的所有条目；ls -l 以详细列表形式列出。-h 需配合 -l 使用（人性化显示文件大小），单独使用无意义。

15.【单选】在当前目录下，创建测试目录 test（　）

  • A. touch test
  • B. mkdir test
  • C. rmdir test
  • D. rm test
✅ 参考答案

B。mkdir 创建目录；touch 创建空文件；rmdir/rm 是删除。

16.【单选】在当前目录创建一个新的空文件 newfile（　）

  • A. mkdir newfile
  • B. touch newfile
  • C. rmdir newfile
  • D. rm newfile
✅ 参考答案

B。touch 创建空文件（文件已存在时更新其时间戳）。

17.【单选】利用 cp 命令复制系统文件 /etc/profile 到当前目录下（　）

  • A. cp /etc/profile /
  • B. cp /etc/profile ./
  • C. cp / /etc/profile
  • D. cp ./ /etc/profile
✅ 参考答案

B。cp 源文件 目标位置，./ 表示当前目录。A 是复制到根目录，C、D 源和目标写反了。

18.【单选】复制文件 profile 到一个新文件 profile.bak，作为备份（　）

  • A. cp profile profile.bak
  • B. cp profile.bak profile
  • C. mv profile profile.bak
  • D. mv profile.bak profile
✅ 参考答案

A。备份要用复制 cp（原文件保留）；mv 是移动/改名，原文件就没了。

19.【单选】删除 test 子目录下的所有文件（　）

  • A. rmdir test
  • B. rm -rf test/*
  • C. rmdir -p test
  • D. del test/*
✅ 参考答案

B。rm -rf test/* 递归强制删除 test 目录下的所有内容（保留 test 目录本身）。rmdir 只能删空目录；del 是 Windows 命令。rm -rf 极其危险，路径务必反复确认。

20.【单选】删除文件命令为（　）

  • A. mkdir
  • B. rmdir
  • C. mv
  • D. rm
✅ 参考答案

D。rm 删除文件（配 -r 可删目录）；rmdir 只能删除空目录。

21.【单选】建立一个新文件可以使用的命令为（　）

  • A. chmod
  • B. more
  • C. cp
  • D. touch
✅ 参考答案

D。touch 新建空文件；chmod 改权限，more 分页查看，cp 复制。

22.【单选】在 Linux 中，系统默认的（　）用户对整个系统拥有完全的控制权

  • A. root
  • B. guest
  • C. admin
  • D. super
✅ 参考答案

A。root 是超级管理员，拥有系统的完全控制权。日常操作建议用普通用户 + sudo，避免误操作损坏系统。

23.【单选】Linux 中自动补全按键为（　）

  • A. Esc
  • B. Ctrl
  • C. Tab
  • D. Home
✅ 参考答案

C。Tab 补全命令名和路径，连按两次可列出所有候选项——是命令行效率第一快捷键。

24.【单选】在终端窗口中强制停止正在执行的操作，可以使用（　）键

  • A. Shift
  • B. Ctrl + C
  • C. Ctrl + Z
  • D. Ctrl + Alt
✅ 参考答案

B。Ctrl + C 向前台进程发送中断信号（SIGINT）终止它。注意区分 Ctrl + Z 是挂起（暂停放入后台），进程并没有被终止。

四、Linux 常用命令（作业二 · 简答题）

25. 使用 Linux 命令完成以下操作

① 在 / 目录下创建一个 export 目录，然后在其内部创建一个 software 目录，最终路径 /export/software ② 复制 /home/student 目录下的 hadoop.tar.gz 文件到①的目录中 ③ 切换到 /export/software 目录

✅ 参考答案
Shell
# ① 一条命令递归创建多级目录（-p：父目录不存在时一并创建）
mkdir -p /export/software
# 等价的分步写法: mkdir /export && mkdir /export/software

# ② 复制文件到目标目录
cp /home/student/hadoop.tar.gz /export/software

# ③ 切换目录
cd /export/software

提示：在 / 下建目录通常需要管理员权限，普通用户请在命令前加 sudo。
