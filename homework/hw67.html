⛓️ LangChain：模型、Prompt、链与 Agent 综合训练

🏷️ 选择 + 简答 + 代码🏷️ 40 题📖 相关章节：LangChain：模型、Prompt、链与 Agent🚀 所属项目：EduRAG：企业私有知识智能问答

40 道练习，覆盖 LangChain 定位与六大组件、三类 Models、Prompt（zero/few-shot）、Chains、Agents、Memory 与 Indexes（加载 / 切分 / 向量库 / 检索）。重点练「组件如何链接成应用」以及「LLM vs Chat vs Embedding」输入输出差异。

建议先独立作答，再展开参考答案。代码题用纯 Python 模拟模板填充、链式拼接、记忆与检索，不调用真实大模型或向量库。

一、选择题（20 题）

1. LangChain 更准确的定位是？

A. 自己训练并发布全部闭源基座模型
B. 围绕 LLM 的应用框架：提供统一接口，把相关组件「链接」起来
C. 仅做操作系统内核调度
D. 仅做 CSS 布局引擎

✅ 参考答案

答案：B。 自身不开发 LLM，降低复杂应用开发难度。

A. 匿名 1990 年
B. Harrison Chase，约 2022 年 10 月

D. 仅浏览器厂商 2005 年

✅ 参考答案

答案：C。 六大：Models、Prompts、Memory、Indexes、Chains、Agents。

二、简答题（12 题）

21. 用一句话说明：LangChain「不开发 LLM」却仍重要的原因。

✅ 参考答案

它把模型调用、提示、记忆、索引、链与代理等工程拼装标准化，让开发者用统一接口快速组合复杂应用，而不是从零对接每个模型与工具。

22. 对比 LLMs 与 Chat Models：各举一个合适场景。

✅ 参考答案

LLMs：单次文本生成/补全（讲故事、起名）。Chat Models：带系统人设的多轮对话（「你是歌手，写歌词」），消息角色更清晰。

23. embed_query 与 embed_documents 差在哪里？

✅ 参考答案

前者输入单个字符串，返回一条向量；后者输入字符串列表，批量返回多条向量。检索入库常用 documents；单条查询向量化常用 query。

24. 解释 PromptTemplate 中 input_variables 与 format 的关系。

✅ 参考答案

input_variables 声明模板占位符（如 lastname）；format(**kwargs) 填入具体值，得到最终 prompt 文本再交给模型。

25. Few-shot 示例里 prefix / examples / suffix 各自角色？

✅ 参考答案

prefix：总任务说明（「给出反义词」）；examples：示范输入输出对；suffix：当前待答题目槽位。三者拼成完整提示。

26. LLMChain 相比「手动 format 再 model()」带来什么？

✅ 参考答案

把 prompt 与 llm 绑成可复用单元，用 run/invoke 统一执行；便于嵌入更大链、减少样板代码，接口更稳定。

27. Agent 与普通 Chain 的本质差别？

✅ 参考答案

Chain 多是预定好的固定步骤；Agent 由模型动态决定下一步行动（选工具、观察结果、再决策），直到任务完成。

28. 短期记忆与长期记忆（课件口径）如何区分？

✅ 参考答案

短期：单一会话内传递上下文。长期：跨多个会话获取/更新信息（常需序列化落盘或外存）。ConversationChain 体现会话内记忆；messages_to_dict 利于长期保存。

29. 为何长文档要切分？切坏代码文本会怎样？

✅ 参考答案

模型有上下文长度限制。按纯长度乱切可能把函数/语义单元切断，片段失去意义；应优先保持语义完整（或用面向代码/Markdown 的 splitter）。

30. 简述「加载 → 切分 → 嵌入 → 向量库 → 检索」流水线。

✅ 参考答案

Loader 读入 Document；Splitter 切 chunk；Embedding 转向量；写入 VectorStore；Retriever/similarity_search 按问题取回相关片段，供后续问答链使用。

31. as_retriever(search_kwargs={'k': 1}) 中 k 的含义？

✅ 参考答案

每次检索返回的文档条数（Top-k）。k=1 表示只取最相关的一条，适合演示或强过滤场景。

32. 为何 EduRAG 一类项目会选 LangChain 做编排层？

✅ 参考答案

需要把本地模型、提示路由、检索、记忆与工具调用串成可维护流水线；LangChain 提供组件化抽象，减少重复胶水代码，便于迭代策略。

三、代码题（8 题）

33. 实现迷你 PromptTemplate：按变量 format

✅ 参考答案
Python
class MiniPromptTemplate:
    def __init__(self, template: str, input_variables: list[str]):
        self.template = template
        self.input_variables = input_variables

    def format(self, **kwargs) -> str:
        missing = [k for k in self.input_variables if k not in kwargs]
        if missing:
            raise KeyError(f"missing: {missing}")
        return self.template.format(**kwargs)


prompt = MiniPromptTemplate(
    "我的邻居姓{lastname}，他生了个儿子，给他儿子起个名字",
    ["lastname"],
)
text = prompt.format(lastname="王")
assert "姓王" in text

def split_text(text: str, separator: str = " ", chunk_size: int = 5, chunk_overlap: int = 0) -> list[str]:
    parts = text.split(separator)
    chunks, buf = [], []
    for p in parts:
        candidate = separator.join(buf + [p]) if buf else p
        if len(candidate) <= chunk_size:
            buf.append(p)
        else:
            if buf:
                chunks.append(separator.join(buf))
            buf = [p]
    if buf:
        chunks.append(separator.join(buf))
    # 教学忽略 overlap 细节；与课件示例对齐主要看 size
    return chunks


assert split_text("a b c d e f", chunk_size=5) == ["a b c", "d e f"]

def similarity_search(docs: list[str], query: str, k: int = 1) -> list[str]:
    scored = []
    for d in docs:
        score = sum(1 for w in query if w in d)  # 字符级教学打分
        scored.append((score, d))
    scored.sort(key=lambda x: x[0], reverse=True)
    return [d for _, d in scored[:k]]


corpus = [
    "北京大学创办于1898年，初名京师大学堂。",
    "1937年卢沟桥事变后学校南迁。",
    "今天天气晴朗适合散步。",
]
hits = similarity_search(corpus, "北京大学什么时候成立", k=1)
assert "1898" in hits[0]
print(hits[0])

def retrieve(query: str, corpus: list[str], k: int = 1) -> list[str]:
    return sorted(corpus, key=lambda d: sum(c in d for c in query), reverse=True)[:k]


def answer_with_context(query: str, corpus: list[str]) -> str:
    ctx = "\n".join(retrieve(query, corpus, k=1))
    prompt = f"根据资料回答。\n资料：{ctx}\n问题：{query}\n答案："
    # 教学假 LLM：从资料截取前 12 字
    return prompt + ctx[:12]


corpus = ["北京大学创办于1898年，是中国近现代第一所国立综合性大学。"]
out = answer_with_context("北大何时创办？", corpus)
assert "1898" in out and "资料" in out
print(out)
